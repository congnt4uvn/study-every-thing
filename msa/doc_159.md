
- Từ phiên bản Grafana Loki 3.0 trở đi, Promtail (thành phần dùng để thu thập log) đã được thay thế bằng một sản phẩm mới tên là **Alloy**.  
- Mặc dù các bài giảng tiếp theo vẫn nói về Promtail, nhưng **Alloy hoạt động tương tự**, chỉ khác là dùng file cấu hình của Alloy thay cho Promtail.  
- Vì đây là thành phần nội bộ của Loki nên **sự thay đổi này không ảnh hưởng lớn** đến cách sử dụng tổng thể.  
- Tất cả file **Docker Compose** trong repo GitHub của khóa học đã được cập nhật sang Alloy.  
- Tài liệu về Alloy kèm Loki có tại:  
  https://grafana.com/docs/loki/latest/get-started/quick-start/  
- Nếu dự án thực tế của bạn dùng **Loki phiên bản cũ**, bạn vẫn cần dùng **Promtail**; cấu hình Promtail nằm ở các **branch cũ** trong repo GitHub của khóa học.
