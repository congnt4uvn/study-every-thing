Dưới đây là một ví dụ về mã Java cho thuật toán Heap Sort, bao gồm tất cả các hàm chính, cấu trúc dữ liệu và thuật toán, cùng với phần nhập và xuất dữ liệu. Mã này được chú thích bằng tiếng Việt để bạn có thể dễ dàng hiểu cách hoạt động của nó.

```java
import java.util.Scanner;

public class HeapSort {

    // Hàm để sắp xếp một mảng sử dụng Heap Sort
    public void sort(int arr[]) {
        int n = arr.length;

        // Xây dựng một heap (sắp xếp lại mảng)
        for (int i = n / 2 - 1; i >= 0; i--) {
            heapify(arr, n, i);
        }

        // Trích xuất từng phần tử từ heap
        for (int i = n - 1; i > 0; i--) {
            // Di chuyển gốc hiện tại đến cuối
            int temp = arr[0];
            arr[0] = arr[i];
            arr[i] = temp;

            // Gọi heapify trên heap giảm
            heapify(arr, i, 0);
        }
    }

    // Hàm để heapify một cây con có gốc tại node i
    void heapify(int arr[], int n, int i) {
        int largest = i; // Khởi tạo largest như gốc
        int left = 2 * i + 1; // left = 2*i + 1
        int right = 2 * i + 2; // right = 2*i + 2

        // Nếu nút con bên trái lớn hơn gốc
        if (left < n && arr[left] > arr[largest]) {
            largest = left;
        }

        // Nếu nút con bên phải lớn hơn largest
        if (right < n && arr[right] > arr[largest]) {
            largest = right;
        }

        // Nếu largest không phải là gốc
        if (largest != i) {
            int swap = arr[i];
            arr[i] = arr[largest];
            arr[largest] = swap;

            // Đệ quy heapify cây con bị ảnh hưởng
            heapify(arr, n, largest);
        }
    }

    // Hàm để in mảng
    static void printArray(int arr[]) {
        int n = arr.length;
        for (int i = 0; i < n; ++i) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
    }

    // Hàm main
    public static void main(String args[]) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Nhập số lượng phần tử của mảng:");
        int n = scanner.nextInt();
        int arr[] = new int[n];

        System.out.println("Nhập các phần tử của mảng:");
        for (int i = 0; i < n; i++) {
            arr[i] = scanner.nextInt();
        }

        HeapSort ob = new HeapSort();
        ob.sort(arr);

        System.out.println("Mảng đã được sắp xếp:");
        printArray(arr);

        scanner.close();
    }
}
```

### Giải thích:
- **sort**: Hàm này thực hiện sắp xếp mảng bằng cách xây dựng một heap và sau đó trích xuất từng phần tử từ heap.
- **heapify**: Hàm này đảm bảo rằng cây con với gốc tại `i` là một heap hợp lệ.
- **printArray**: Hàm này in ra mảng đã được sắp xếp.
- **main**: Hàm này đọc dữ liệu đầu vào từ người dùng, gọi hàm sắp xếp và in ra mảng đã sắp xếp.
