Dưới đây là một ví dụ về mã Java cho thuật toán Sắp xếp Chọn (Selection Sort). Mã này bao gồm hàm chính, cấu trúc dữ liệu và thuật toán, cũng như đầu vào và đầu ra.

```java
import java.util.Scanner;

public class SelectionSortExample {

    // Hàm để thực hiện thuật toán Sắp xếp Chọn
    public static void selectionSort(int[] arr) {
        int n = arr.length;
        
        // Duyệt qua từng phần tử của mảng
        for (int i = 0; i < n - 1; i++) {
            // Tìm phần tử nhỏ nhất trong mảng chưa được sắp xếp
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            
            // Hoán đổi phần tử nhỏ nhất với phần tử đầu tiên
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }

    // Hàm chính
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Nhập số lượng phần tử
        System.out.print("Nhập số lượng phần tử: ");
        int n = scanner.nextInt();
        
        // Khởi tạo mảng
        int[] arr = new int[n];
        
        // Nhập các phần tử của mảng
        System.out.println("Nhập các phần tử của mảng:");
        for (int i = 0; i < n; i++) {
            System.out.print("Phần tử " + (i + 1) + ": ");
            arr[i] = scanner.nextInt();
        }
        
        // Gọi hàm sắp xếp
        selectionSort(arr);
        
        // In mảng đã sắp xếp
        System.out.println("Mảng sau khi sắp xếp:");
        for (int i = 0; i < n; i++) {
            System.out.print(arr[i] + " ");
        }
    }
}
```

### Giải thích:
- **selectionSort**: Hàm này thực hiện thuật toán Sắp xếp Chọn. Nó tìm phần tử nhỏ nhất trong mảng chưa được sắp xếp và hoán đổi nó với phần tử đầu tiên của mảng chưa được sắp xếp.
- **main**: Hàm chính để chạy chương trình. Nó yêu cầu người dùng nhập số lượng phần tử và các giá trị của mảng, sau đó gọi hàm `selectionSort` để sắp xếp mảng và in ra mảng đã sắp xếp.

### Đầu vào và đầu ra:
- **Đầu vào**: Người dùng nhập số lượng phần tử và các giá trị của mảng.
- **Đầu ra**: Mảng đã được sắp xếp theo thứ tự tăng dần.
