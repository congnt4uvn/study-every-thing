Dưới đây là một ví dụ về cách triển khai thuật toán tham lam (greedy algorithm) để giải quyết bài toán chọn hoạt động (activity selection) trong Java. Chương trình này sẽ chọn số lượng hoạt động tối đa mà không bị chồng chéo.

```java
import java.util.Arrays;
import java.util.Comparator;

class Activity {
    int start;
    int finish;

    public Activity(int start, int finish) {
        this.start = start;
        this.finish = finish;
    }
}

public class ActivitySelection {

    // Hàm để tìm số lượng hoạt động tối đa có thể thực hiện mà không bị chồng chéo
    public static void selectActivities(Activity[] activities) {
        // Sắp xếp các hoạt động theo thời gian kết thúc
        Arrays.sort(activities, Comparator.comparingInt(a -> a.finish));

        System.out.println("Các hoạt động được chọn:");

        // Chọn hoạt động đầu tiên
        int n = activities.length;
        int i = 0;
        System.out.println("Hoạt động bắt đầu: " + activities[i].start + ", kết thúc: " + activities[i].finish);

        // Xem xét các hoạt động còn lại
        for (int j = 1; j < n; j++) {
            // Nếu hoạt động này bắt đầu sau khi hoạt động trước đó kết thúc, chọn nó
            if (activities[j].start >= activities[i].finish) {
                System.out.println("Hoạt động bắt đầu: " + activities[j].start + ", kết thúc: " + activities[j].finish);
                i = j;
            }
        }
    }

    public static void main(String[] args) {
        // Danh sách các hoạt động với thời gian bắt đầu và kết thúc
        Activity[] activities = {
            new Activity(1, 3),
            new Activity(2, 4),
            new Activity(3, 5),
            new Activity(0, 6),
            new Activity(5, 7),
            new Activity(8, 9),
            new Activity(5, 9)
        };

        // Gọi hàm để chọn các hoạt động
        selectActivities(activities);
    }
}
```

### Giải thích:

1. **Lớp `Activity`:** Đại diện cho một hoạt động với thời gian bắt đầu và kết thúc.

2. **Hàm `selectActivities`:** 
   - Sắp xếp các hoạt động theo thời gian kết thúc.
   - Chọn hoạt động đầu tiên và in ra.
   - Duyệt qua các hoạt động còn lại, nếu hoạt động nào có thời gian bắt đầu không chồng chéo với hoạt động đã chọn trước đó, thì chọn nó.

3. **Hàm `main`:** 
   - Khởi tạo một danh sách các hoạt động.
   - Gọi hàm `selectActivities` để tìm và in ra các hoạt động được chọn.

### Đầu vào và đầu ra:

- **Đầu vào:** Danh sách các hoạt động với thời gian bắt đầu và kết thúc.
- **Đầu ra:** Danh sách các hoạt động được chọn mà không bị chồng chéo.

Chương trình này sẽ in ra các hoạt động được chọn theo thứ tự mà chúng có thể được thực hiện mà không bị chồng chéo.
