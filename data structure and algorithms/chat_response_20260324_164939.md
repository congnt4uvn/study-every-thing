Dưới đây là một ví dụ về cách triển khai một vòng đệm (ring buffer) trong Java. Vòng đệm là một cấu trúc dữ liệu hữu ích cho việc quản lý bộ nhớ đệm với kích thước cố định, nơi mà dữ liệu mới có thể ghi đè lên dữ liệu cũ khi bộ đệm đầy.

```java
public class VongDem {
    private int[] buffer;
    private int size;
    private int start;
    private int end;
    private int count;

    public VongDem(int size) {
        this.size = size;
        this.buffer = new int[size];
        this.start = 0;
        this.end = 0;
        this.count = 0;
    }

    public boolean isFull() {
        return count == size;
    }

    public boolean isEmpty() {
        return count == 0;
    }

    public void them(int value) {
        if (isFull()) {
            System.out.println("Vòng đệm đầy, ghi đè dữ liệu cũ.");
            start = (start + 1) % size; // Di chuyển điểm bắt đầu
        } else {
            count++;
        }
        buffer[end] = value;
        end = (end + 1) % size;
    }

    public int lay() {
        if (isEmpty()) {
            throw new RuntimeException("Vòng đệm rỗng.");
        }
        int value = buffer[start];
        start = (start + 1) % size;
        count--;
        return value;
    }

    public void hienThi() {
        System.out.print("Nội dung vòng đệm: ");
        for (int i = 0; i < count; i++) {
            System.out.print(buffer[(start + i) % size] + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        VongDem vongDem = new VongDem(5);

        vongDem.them(1);
        vongDem.them(2);
        vongDem.them(3);
        vongDem.hienThi();

        vongDem.lay();
        vongDem.hienThi();

        vongDem.them(4);
        vongDem.them(5);
        vongDem.them(6);
        vongDem.hienThi();

        vongDem.them(7);
        vongDem.hienThi();
    }
}
```

### Giải thích:

- **`VongDem`**: Lớp này đại diện cho vòng đệm với các thuộc tính như `buffer` (mảng lưu trữ dữ liệu), `size` (kích thước của vòng đệm), `start` (vị trí bắt đầu), `end` (vị trí kết thúc), và `count` (số lượng phần tử hiện tại trong vòng đệm).

- **`them(int value)`**: Phương thức này thêm một giá trị vào vòng đệm. Nếu vòng đệm đầy, nó sẽ ghi đè lên dữ liệu cũ nhất.

- **`lay()`**: Phương thức này lấy và loại bỏ phần tử ở đầu vòng đệm. Nếu vòng đệm rỗng, nó sẽ ném ra một ngoại lệ.

- **`hienThi()`**: Phương thức này hiển thị nội dung hiện tại của vòng đệm.

- **`main`**: Phương thức này minh họa cách sử dụng vòng đệm bằng cách thêm, lấy và hiển thị các phần tử.
