Dưới đây là một ví dụ mã Java đơn giản bao gồm các cấu trúc dữ liệu, thuật toán, và các chức năng nhập/xuất cơ bản. Mã này sẽ thực hiện một số thao tác cơ bản như thêm phần tử vào danh sách, sắp xếp danh sách, và in ra kết quả.

```java
import java.util.ArrayList;
import java.util.Collections;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        // Tạo một danh sách để lưu trữ các số nguyên
        ArrayList<Integer> numbers = new ArrayList<>();

        // Sử dụng Scanner để nhập dữ liệu từ người dùng
        Scanner scanner = new Scanner(System.in);
        System.out.println("Nhập số lượng phần tử: ");
        int n = scanner.nextInt();

        // Nhập các phần tử vào danh sách
        System.out.println("Nhập các phần tử: ");
        for (int i = 0; i < n; i++) {
            int number = scanner.nextInt();
            numbers.add(number);
        }

        // Sắp xếp danh sách
        Collections.sort(numbers);

        // In ra danh sách đã sắp xếp
        System.out.println("Danh sách sau khi sắp xếp: ");
        for (int number : numbers) {
            System.out.print(number + " ");
        }
    }
}
```

### Giải thích:
- **ArrayList**: Sử dụng `ArrayList` để lưu trữ các số nguyên. Đây là một cấu trúc dữ liệu động cho phép thêm, xóa và truy cập phần tử một cách linh hoạt.
- **Scanner**: Sử dụng `Scanner` để đọc dữ liệu đầu vào từ người dùng.
- **Collections.sort()**: Sử dụng phương thức `sort` của lớp `Collections` để sắp xếp danh sách theo thứ tự tăng dần.
- **Vòng lặp for**: Sử dụng vòng lặp `for` để duyệt qua các phần tử trong danh sách và in ra màn hình.

Mã này minh họa cách sử dụng các cấu trúc dữ liệu và thuật toán cơ bản trong Java, đồng thời thực hiện nhập và xuất dữ liệu.
