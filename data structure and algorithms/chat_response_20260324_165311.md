Dưới đây là ví dụ về cách triển khai thuật toán Merge Sort và Quick Sort bằng Java, sử dụng chiến lược chia để trị (divide and conquer). Mỗi thuật toán sẽ có hàm chính, cấu trúc dữ liệu và thuật toán, cùng với ví dụ về đầu vào và đầu ra.

### Merge Sort

```java
public class MergeSort {

    // Hàm chính để chạy Merge Sort
    public static void main(String[] args) {
        int[] array = {38, 27, 43, 3, 9, 82, 10};
        System.out.println("Mảng ban đầu:");
        printArray(array);

        mergeSort(array, 0, array.length - 1);

        System.out.println("Mảng sau khi sắp xếp bằng Merge Sort:");
        printArray(array);
    }

    // Hàm để thực hiện Merge Sort
    public static void mergeSort(int[] array, int left, int right) {
        if (left < right) {
            int middle = (left + right) / 2;

            // Đệ quy chia đôi mảng
            mergeSort(array, left, middle);
            mergeSort(array, middle + 1, right);

            // Gộp hai nửa đã sắp xếp
            merge(array, left, middle, right);
        }
    }

    // Hàm để gộp hai nửa mảng
    public static void merge(int[] array, int left, int middle, int right) {
        int n1 = middle - left + 1;
        int n2 = right - middle;

        int[] L = new int[n1];
        int[] R = new int[n2];

        for (int i = 0; i < n1; i++)
            L[i] = array[left + i];
        for (int j = 0; j < n2; j++)
            R[j] = array[middle + 1 + j];

        int i = 0, j = 0;
        int k = left;
        while (i < n1 && j < n2) {
            if (L[i] <= R[j]) {
                array[k] = L[i];
                i++;
            } else {
                array[k] = R[j];
                j++;
            }
            k++;
        }

        while (i < n1) {
            array[k] = L[i];
            i++;
            k++;
        }

        while (j < n2) {
            array[k] = R[j];
            j++;
            k++;
        }
    }

    // Hàm để in mảng
    public static void printArray(int[] array) {
        for (int value : array) {
            System.out.print(value + " ");
        }
        System.out.println();
    }
}
```

### Quick Sort

```java
public class QuickSort {

    // Hàm chính để chạy Quick Sort
    public static void main(String[] args) {
        int[] array = {38, 27, 43, 3, 9, 82, 10};
        System.out.println("Mảng ban đầu:");
        printArray(array);

        quickSort(array, 0, array.length - 1);

        System.out.println("Mảng sau khi sắp xếp bằng Quick Sort:");
        printArray(array);
    }

    // Hàm để thực hiện Quick Sort
    public static void quickSort(int[] array, int low, int high) {
        if (low < high) {
            int pi = partition(array, low, high);

            // Đệ quy sắp xếp các phần tử trước và sau điểm phân chia
            quickSort(array, low, pi - 1);
            quickSort(array, pi + 1, high);
        }
    }

    // Hàm để phân chia mảng
    public static int partition(int[] array, int low, int high) {
        int pivot = array[high];
        int i = (low - 1);
        for (int j = low; j < high; j++) {
            if (array[j] <= pivot) {
                i++;

                // Hoán đổi array[i] và array[j]
                int temp = array[i];
                array[i] = array[j];
                array[j] = temp;
            }
        }

        // Hoán đổi array[i+1] và array[high] (hoặc pivot)
        int temp = array[i + 1];
        array[i + 1] = array[high];
        array[high] = temp;

        return i + 1;
    }

    // Hàm để in mảng
    public static void printArray(int[] array) {
        for (int value : array) {
            System.out.print(value + " ");
        }
        System.out.println();
    }
}
```

### Đầu vào và đầu ra

- **Đầu vào:** Mảng số nguyên `{38, 27, 43, 3, 9, 82, 10}`
- **Đầu ra cho cả hai thuật toán:**
  - Mảng sau khi sắp xếp: `3 9 10 27 38 43 82`

Cả hai thuật toán đều sử dụng chiến lược chia để trị để sắp xếp mảng một cách hiệu quả.
