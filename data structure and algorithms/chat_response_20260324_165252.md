Dưới đây là một ví dụ về cách giải quyết bài toán N-Queens bằng cách sử dụng kỹ thuật quay lui (backtracking) trong Java. Chương trình này sẽ đặt các quân hậu trên bàn cờ sao cho không có hai quân hậu nào đe dọa lẫn nhau.

```java
public class NQueens {
    private int size;
    private int[] board;

    public NQueens(int size) {
        this.size = size;
        this.board = new int[size];
    }

    public boolean solve() {
        return placeQueen(0);
    }

    private boolean placeQueen(int row) {
        if (row == size) {
            printSolution();
            return true;
        }

        boolean hasSolution = false;
        for (int col = 0; col < size; col++) {
            if (isSafe(row, col)) {
                board[row] = col;
                hasSolution = placeQueen(row + 1) || hasSolution;
            }
        }
        return hasSolution;
    }

    private boolean isSafe(int row, int col) {
        for (int i = 0; i < row; i++) {
            if (board[i] == col || Math.abs(board[i] - col) == Math.abs(i - row)) {
                return false;
            }
        }
        return true;
    }

    private void printSolution() {
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (board[i] == j) {
                    System.out.print("Q ");
                } else {
                    System.out.print(". ");
                }
            }
            System.out.println();
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int n = 8; // Kích thước bàn cờ
        NQueens nQueens = new NQueens(n);
        if (!nQueens.solve()) {
            System.out.println("Không có giải pháp.");
        }
    }
}
```

### Giải thích:

- **`NQueens` class**: Lớp này chứa các phương thức để giải quyết bài toán N-Queens.
- **`solve` method**: Bắt đầu quá trình đặt quân hậu từ hàng đầu tiên.
- **`placeQueen` method**: Thử đặt quân hậu vào từng cột của hàng hiện tại và kiểm tra xem có an toàn không. Nếu an toàn, tiếp tục đặt quân hậu vào hàng tiếp theo.
- **`isSafe` method**: Kiểm tra xem vị trí đặt quân hậu có an toàn không, tức là không có quân hậu nào khác đe dọa.
- **`printSolution` method**: In ra một giải pháp hợp lệ.
- **`main` method**: Khởi tạo một đối tượng `NQueens` với kích thước bàn cờ là `n` và gọi phương thức `solve` để tìm giải pháp.

Chương trình này sẽ in ra tất cả các giải pháp có thể cho bài toán N-Queens với kích thước bàn cờ đã cho. Nếu không có giải pháp nào, nó sẽ in ra thông báo "Không có giải pháp."
