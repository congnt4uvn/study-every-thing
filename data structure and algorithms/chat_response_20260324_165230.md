Dưới đây là một ví dụ về cách triển khai thuật toán Tìm Dãy Con Chung Dài Nhất (Longest Common Subsequence - LCS) bằng lập trình động trong Java. Chương trình này bao gồm các hàm chính, cấu trúc dữ liệu và thuật toán cần thiết để giải quyết bài toán LCS. Chương trình cũng bao gồm phần nhập và xuất dữ liệu.

```java
import java.util.Scanner;

public class LongestCommonSubsequence {

    // Hàm để tìm dãy con chung dài nhất
    public static int lcs(String X, String Y) {
        int m = X.length();
        int n = Y.length();
        int[][] L = new int[m + 1][n + 1];

        // Xây dựng bảng L[][] theo cách từ dưới lên
        for (int i = 0; i <= m; i++) {
            for (int j = 0; j <= n; j++) {
                if (i == 0 || j == 0) {
                    L[i][j] = 0;
                } else if (X.charAt(i - 1) == Y.charAt(j - 1)) {
                    L[i][j] = L[i - 1][j - 1] + 1;
                } else {
                    L[i][j] = Math.max(L[i - 1][j], L[i][j - 1]);
                }
            }
        }

        // L[m][n] chứa độ dài của LCS cho X[0..m-1], Y[0..n-1]
        return L[m][n];
    }

    // Hàm chính
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Nhập chuỗi thứ nhất
        System.out.print("Nhập chuỗi thứ nhất: ");
        String X = scanner.nextLine();

        // Nhập chuỗi thứ hai
        System.out.print("Nhập chuỗi thứ hai: ");
        String Y = scanner.nextLine();

        // Tìm và in ra độ dài của dãy con chung dài nhất
        int result = lcs(X, Y);
        System.out.println("Độ dài của dãy con chung dài nhất là: " + result);

        scanner.close();
    }
}
```

### Giải thích:

- **Hàm `lcs`**: Hàm này sử dụng lập trình động để tính toán độ dài của dãy con chung dài nhất giữa hai chuỗi `X` và `Y`. Nó tạo ra một bảng hai chiều `L` để lưu trữ độ dài của LCS cho các cặp tiền tố của `X` và `Y`.

- **Vòng lặp**: Hai vòng lặp lồng nhau được sử dụng để điền vào bảng `L` theo cách từ dưới lên. Nếu các ký tự của `X` và `Y` khớp nhau, giá trị của `L[i][j]` được tăng lên 1 so với `L[i-1][j-1]`. Nếu không, nó sẽ là giá trị lớn hơn giữa `L[i-1][j]` và `L[i][j-1]`.

- **Hàm `main`**: Hàm này xử lý việc nhập dữ liệu từ người dùng và gọi hàm `lcs` để tính toán và in ra kết quả.

Chương trình này sẽ yêu cầu người dùng nhập hai chuỗi và sau đó in ra độ dài của dãy con chung dài nhất giữa hai chuỗi đó.
