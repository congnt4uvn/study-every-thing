A ring buffer, also known as a circular buffer, is a fixed-size data structure that uses a single, contiguous block of memory as if it were connected end-to-end. It is particularly useful for buffering data streams.

Below is a simple Java implementation of a ring buffer, including the main functions such as adding elements, removing elements, and checking if the buffer is full or empty. This example also includes a main method to demonstrate input and output.

```java
public class RingBuffer {
    private int[] buffer;
    private int head;
    private int tail;
    private int size;
    private int capacity;

    public RingBuffer(int capacity) {
        this.capacity = capacity;
        this.buffer = new int[capacity];
        this.head = 0;
        this.tail = 0;
        this.size = 0;
    }

    public boolean isFull() {
        return size == capacity;
    }

    public boolean isEmpty() {
        return size == 0;
    }

    public void enqueue(int value) {
        if (isFull()) {
            throw new RuntimeException("Buffer is full");
        }
        buffer[tail] = value;
        tail = (tail + 1) % capacity;
        size++;
    }

    public int dequeue() {
        if (isEmpty()) {
            throw new RuntimeException("Buffer is empty");
        }
        int value = buffer[head];
        head = (head + 1) % capacity;
        size--;
        return value;
    }

    public int peek() {
        if (isEmpty()) {
            throw new RuntimeException("Buffer is empty");
        }
        return buffer[head];
    }

    public static void main(String[] args) {
        RingBuffer ringBuffer = new RingBuffer(5);

        // Enqueue elements
        ringBuffer.enqueue(10);
        ringBuffer.enqueue(20);
        ringBuffer.enqueue(30);
        ringBuffer.enqueue(40);
        ringBuffer.enqueue(50);

        // Try to enqueue another element (should throw an exception)
        try {
            ringBuffer.enqueue(60);
        } catch (RuntimeException e) {
            System.out.println(e.getMessage());
        }

        // Dequeue elements
        System.out.println("Dequeued: " + ringBuffer.dequeue());
        System.out.println("Dequeued: " + ringBuffer.dequeue());

        // Enqueue more elements
        ringBuffer.enqueue(60);
        ringBuffer.enqueue(70);

        // Peek at the front element
        System.out.println("Peek: " + ringBuffer.peek());

        // Dequeue remaining elements
        while (!ringBuffer.isEmpty()) {
            System.out.println("Dequeued: " + ringBuffer.dequeue());
        }

        // Try to dequeue from an empty buffer (should throw an exception)
        try {
            ringBuffer.dequeue();
        } catch (RuntimeException e) {
            System.out.println(e.getMessage());
        }
    }
}
```

### Explanation:

- **enqueue(int value):** Adds an element to the end of the buffer. If the buffer is full, it throws an exception.
- **dequeue():** Removes and returns the element at the front of the buffer. If the buffer is empty, it throws an exception.
- **peek():** Returns the element at the front of the buffer without removing it. If the buffer is empty, it throws an exception.
- **isFull():** Checks if the buffer is full.
- **isEmpty():** Checks if the buffer is empty.

The `main` method demonstrates how to use the ring buffer by enqueuing and dequeuing elements, and handling exceptions when trying to operate on a full or empty buffer.
