Dưới đây là một ví dụ về cách sử dụng lập trình động để tính dãy Fibonacci trong Java. Chương trình này bao gồm tất cả các thành phần chính như cấu trúc dữ liệu, thuật toán, hàm chính, đầu vào và đầu ra.

```java
import java.util.Scanner;

public class FibonacciDP {

    // Hàm để tính số Fibonacci sử dụng lập trình động
    public static long fibonacci(int n) {
        if (n <= 1) {
            return n;
        }

        // Tạo một mảng để lưu trữ các giá trị Fibonacci đã tính toán
        long[] fib = new long[n + 1];
        fib[0] = 0;
        fib[1] = 1;

        // Tính toán các giá trị Fibonacci từ 2 đến n
        for (int i = 2; i <= n; i++) {
            fib[i] = fib[i - 1] + fib[i - 2];
        }

        return fib[n];
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Nhập số n từ người dùng
        System.out.print("Nhập số n để tính số Fibonacci: ");
        int n = scanner.nextInt();

        // Tính và in ra số Fibonacci thứ n
        long result = fibonacci(n);
        System.out.println("Số Fibonacci thứ " + n + " là: " + result);

        scanner.close();
    }
}
```

### Giải thích:

- **Cấu trúc dữ liệu**: Sử dụng một mảng `fib` để lưu trữ các giá trị Fibonacci đã tính toán, giúp tránh việc tính toán lại các giá trị đã biết (giải quyết vấn đề con trùng lặp).

- **Thuật toán**: Sử dụng vòng lặp để tính toán các giá trị Fibonacci từ 2 đến n, dựa trên công thức `fib[i] = fib[i - 1] + fib[i - 2]`.

- **Hàm chính**: `main` là điểm bắt đầu của chương trình, nơi người dùng nhập vào giá trị `n` và chương trình sẽ tính toán và in ra số Fibonacci thứ `n`.

- **Đầu vào/Đầu ra**: Người dùng nhập vào một số nguyên `n`, và chương trình sẽ xuất ra số Fibonacci thứ `n`.
