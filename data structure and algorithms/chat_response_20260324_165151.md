Dưới đây là một ví dụ về cách triển khai thuật toán Floyd-Warshall trong Java để tìm đường đi ngắn nhất giữa tất cả các cặp đỉnh trong một đồ thị. Mã này bao gồm các hàm chính, cấu trúc dữ liệu và thuật toán, cũng như phần nhập và xuất dữ liệu.

```java
import java.util.Scanner;

public class FloydWarshall {
    final static int INF = 99999; // Sử dụng một giá trị lớn để biểu thị vô cực

    // Hàm chính để thực hiện thuật toán Floyd-Warshall
    void floydWarshall(int graph[][], int V) {
        int dist[][] = new int[V][V];

        // Khởi tạo ma trận khoảng cách với giá trị ban đầu từ đồ thị
        for (int i = 0; i < V; i++)
            for (int j = 0; j < V; j++)
                dist[i][j] = graph[i][j];

        // Thuật toán Floyd-Warshall
        for (int k = 0; k < V; k++) {
            for (int i = 0; i < V; i++) {
                for (int j = 0; j < V; j++) {
                    if (dist[i][k] + dist[k][j] < dist[i][j])
                        dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }

        // In kết quả
        printSolution(dist, V);
    }

    // Hàm để in ma trận khoảng cách
    void printSolution(int dist[][], int V) {
        System.out.println("Ma trận khoảng cách ngắn nhất giữa các cặp đỉnh:");
        for (int i = 0; i < V; i++) {
            for (int j = 0; j < V; j++) {
                if (dist[i][j] == INF)
                    System.out.print("INF ");
                else
                    System.out.print(dist[i][j] + "   ");
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Nhập số lượng đỉnh:");
        int V = scanner.nextInt();

        int graph[][] = new int[V][V];

        System.out.println("Nhập ma trận trọng số (sử dụng " + INF + " cho vô cực):");
        for (int i = 0; i < V; i++) {
            for (int j = 0; j < V; j++) {
                graph[i][j] = scanner.nextInt();
            }
        }

        FloydWarshall a = new FloydWarshall();
        a.floydWarshall(graph, V);

        scanner.close();
    }
}
```

### Hướng dẫn sử dụng:
1. **Nhập số lượng đỉnh**: Đầu tiên, bạn sẽ nhập số lượng đỉnh của đồ thị.
2. **Nhập ma trận trọng số**: Tiếp theo, bạn nhập ma trận trọng số của đồ thị. Sử dụng giá trị `99999` để biểu thị vô cực (không có đường đi trực tiếp giữa hai đỉnh).
3. **Kết quả**: Chương trình sẽ in ra ma trận khoảng cách ngắn nhất giữa tất cả các cặp đỉnh.

Lưu ý rằng mã này giả định rằng đồ thị không có chu trình âm.
