Certainly! The Longest Common Subsequence (LCS) problem is a classic example of dynamic programming. Below is a Java implementation that includes the main function, necessary data structures, and algorithms to solve the LCS problem. The program takes two strings as input and outputs their longest common subsequence.

```java
public class LongestCommonSubsequence {

    // Function to find the length of the longest common subsequence
    public static int lcs(String str1, String str2) {
        int m = str1.length();
        int n = str2.length();
        
        // Create a 2D array to store lengths of longest common subsequence
        int[][] dp = new int[m + 1][n + 1];

        // Build the dp array from bottom up
        for (int i = 0; i <= m; i++) {
            for (int j = 0; j <= n; j++) {
                if (i == 0 || j == 0) {
                    dp[i][j] = 0;
                } else if (str1.charAt(i - 1) == str2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        // The length of the longest common subsequence
        return dp[m][n];
    }

    // Function to reconstruct the longest common subsequence
    public static String getLCS(String str1, String str2) {
        int m = str1.length();
        int n = str2.length();
        int[][] dp = new int[m + 1][n + 1];

        // Build the dp array
        for (int i = 0; i <= m; i++) {
            for (int j = 0; j <= n; j++) {
                if (i == 0 || j == 0) {
                    dp[i][j] = 0;
                } else if (str1.charAt(i - 1) == str2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        // Reconstruct the LCS from the dp array
        int index = dp[m][n];
        char[] lcs = new char[index];
        int i = m, j = n;
        while (i > 0 && j > 0) {
            if (str1.charAt(i - 1) == str2.charAt(j - 1)) {
                lcs[index - 1] = str1.charAt(i - 1);
                i--;
                j--;
                index--;
            } else if (dp[i - 1][j] > dp[i][j - 1]) {
                i--;
            } else {
                j--;
            }
        }

        return new String(lcs);
    }

    public static void main(String[] args) {
        String str1 = "AGGTAB";
        String str2 = "GXTXAYB";

        System.out.println("Length of LCS: " + lcs(str1, str2));
        System.out.println("LCS: " + getLCS(str1, str2));
    }
}
```

### Explanation:

1. **`lcs` Function**: This function calculates the length of the longest common subsequence using a dynamic programming table (`dp`). It iterates over each character of the two strings and fills the table based on whether the characters match or not.

2. **`getLCS` Function**: This function reconstructs the actual longest common subsequence from the `dp` table. It traces back from `dp[m][n]` to find the characters that are part of the LCS.

3. **`main` Function**: This is the entry point of the program. It initializes two strings, calls the `lcs` and `getLCS` functions, and prints the results.

### Input/Output:
- **Input**: Two strings, `str1` and `str2`.
- **Output**: The length of the LCS and the LCS itself.
