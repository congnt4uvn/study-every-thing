Creating a comprehensive Java program that includes various data structures, algorithms, and input/output operations can be quite extensive. Below is a sample Java program that demonstrates the use of some basic data structures (ArrayList, HashMap), a simple algorithm (bubble sort), and input/output operations. This example is designed to be educational and covers several fundamental concepts.

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.Scanner;

public class Main {

    // Bubble sort algorithm
    public static void bubbleSort(ArrayList<Integer> list) {
        int n = list.size();
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - i - 1; j++) {
                if (list.get(j) > list.get(j + 1)) {
                    // Swap list[j] and list[j+1]
                    int temp = list.get(j);
                    list.set(j, list.get(j + 1));
                    list.set(j + 1, temp);
                }
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Input: Reading integers into an ArrayList
        System.out.println("Enter numbers (type 'done' to finish):");
        ArrayList<Integer> numbers = new ArrayList<>();
        while (scanner.hasNextInt()) {
            numbers.add(scanner.nextInt());
        }
        scanner.nextLine(); // Consume the newline character

        // Process: Sort the numbers using bubble sort
        bubbleSort(numbers);

        // Output: Display the sorted numbers
        System.out.println("Sorted numbers:");
        for (int number : numbers) {
            System.out.print(number + " ");
        }
        System.out.println();

        // Demonstrating the use of a HashMap
        HashMap<String, Integer> nameToAge = new HashMap<>();
        nameToAge.put("Alice", 30);
        nameToAge.put("Bob", 25);
        nameToAge.put("Charlie", 35);

        // Input: Reading a name to find the age
        System.out.println("Enter a name to find the age:");
        String name = scanner.nextLine();

        // Output: Display the age if the name exists in the map
        if (nameToAge.containsKey(name)) {
            System.out.println(name + " is " + nameToAge.get(name) + " years old.");
        } else {
            System.out.println("Name not found.");
        }

        scanner.close();
    }
}
```

### Explanation:

1. **Data Structures**:
   - `ArrayList<Integer>`: Used to store a list of integers input by the user.
   - `HashMap<String, Integer>`: Used to map names to ages, demonstrating key-value pair storage.

2. **Algorithm**:
   - **Bubble Sort**: A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

3. **Input/Output**:
   - The program reads integers from the user until "done" is typed.
   - It sorts the integers and prints the sorted list.
   - It also reads a name from the user and looks up the corresponding age in a `HashMap`.

This program provides a basic overview of how to use some common data structures and algorithms in Java, along with handling user input and output.
