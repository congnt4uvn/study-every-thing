Dưới đây là một ví dụ về cách giải quyết bài toán Knapsack (bài toán cái túi) bằng lập trình động trong Java. Chương trình này bao gồm các hàm chính, cấu trúc dữ liệu và thuật toán cần thiết để giải quyết bài toán với các ràng buộc về trọng lượng.

```java
import java.util.Scanner;

public class Knapsack {

    // Hàm để tìm giá trị lớn nhất có thể đạt được với các ràng buộc về trọng lượng
    public static int knapsack(int[] weights, int[] values, int capacity) {
        int n = values.length;
        int[][] dp = new int[n + 1][capacity + 1];

        // Xây dựng bảng dp theo cách từ dưới lên
        for (int i = 0; i <= n; i++) {
            for (int w = 0; w <= capacity; w++) {
                if (i == 0 || w == 0) {
                    dp[i][w] = 0;
                } else if (weights[i - 1] <= w) {
                    dp[i][w] = Math.max(values[i - 1] + dp[i - 1][w - weights[i - 1]], dp[i - 1][w]);
                } else {
                    dp[i][w] = dp[i - 1][w];
                }
            }
        }

        return dp[n][capacity];
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Nhập số lượng vật phẩm
        System.out.print("Nhập số lượng vật phẩm: ");
        int n = scanner.nextInt();

        // Nhập trọng lượng và giá trị của từng vật phẩm
        int[] weights = new int[n];
        int[] values = new int[n];
        System.out.println("Nhập trọng lượng và giá trị của từng vật phẩm:");
        for (int i = 0; i < n; i++) {
            System.out.print("Vật phẩm " + (i + 1) + " - Trọng lượng: ");
            weights[i] = scanner.nextInt();
            System.out.print("Vật phẩm " + (i + 1) + " - Giá trị: ");
            values[i] = scanner.nextInt();
        }

        // Nhập sức chứa của cái túi
        System.out.print("Nhập sức chứa của cái túi: ");
        int capacity = scanner.nextInt();

        // Tính toán giá trị lớn nhất có thể đạt được
        int maxValue = knapsack(weights, values, capacity);

        // Xuất kết quả
        System.out.println("Giá trị lớn nhất có thể đạt được là: " + maxValue);

        scanner.close();
    }
}
```

### Giải thích:

- **Cấu trúc dữ liệu**: Sử dụng mảng hai chiều `dp` để lưu trữ giá trị lớn nhất có thể đạt được với từng trọng lượng và số lượng vật phẩm khác nhau.
- **Thuật toán**: Sử dụng phương pháp lập trình động để điền vào bảng `dp` từ dưới lên, dựa trên việc quyết định có chọn một vật phẩm vào túi hay không.
- **Đầu vào**: Người dùng nhập số lượng vật phẩm, trọng lượng và giá trị của từng vật phẩm, và sức chứa của cái túi.
- **Đầu ra**: Giá trị lớn nhất có thể đạt được với các ràng buộc đã cho.
