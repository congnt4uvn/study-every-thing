Dưới đây là một ví dụ về cách triển khai thuật toán Dijkstra trong Java để tìm đường đi ngắn nhất trong đồ thị có trọng số. Mã này bao gồm các cấu trúc dữ liệu chính, hàm và đầu vào/đầu ra.

```java
import java.util.*;

class Graph {
    private int V; // Số đỉnh
    private LinkedList<Edge>[] adj; // Danh sách kề

    class Edge {
        int dest;
        int weight;

        Edge(int dest, int weight) {
            this.dest = dest;
            this.weight = weight;
        }
    }

    Graph(int v) {
        V = v;
        adj = new LinkedList[v];
        for (int i = 0; i < v; ++i)
            adj[i] = new LinkedList<>();
    }

    void addEdge(int src, int dest, int weight) {
        adj[src].add(new Edge(dest, weight));
        adj[dest].add(new Edge(src, weight)); // Nếu đồ thị vô hướng
    }

    void dijkstra(int src) {
        int[] dist = new int[V];
        boolean[] sptSet = new boolean[V];

        PriorityQueue<Node> pq = new PriorityQueue<>(V, new Node());

        for (int i = 0; i < V; i++) {
            dist[i] = Integer.MAX_VALUE;
            sptSet[i] = false;
        }

        pq.add(new Node(src, 0));
        dist[src] = 0;

        while (!pq.isEmpty()) {
            int u = pq.poll().node;

            sptSet[u] = true;

            for (Edge edge : adj[u]) {
                if (!sptSet[edge.dest] && dist[u] != Integer.MAX_VALUE && dist[u] + edge.weight < dist[edge.dest]) {
                    dist[edge.dest] = dist[u] + edge.weight;
                    pq.add(new Node(edge.dest, dist[edge.dest]));
                }
            }
        }

        printSolution(dist);
    }

    void printSolution(int[] dist) {
        System.out.println("Đỉnh \t Khoảng cách từ nguồn");
        for (int i = 0; i < V; i++)
            System.out.println(i + " \t\t " + dist[i]);
    }

    class Node implements Comparator<Node> {
        public int node;
        public int cost;

        public Node() {}

        public Node(int node, int cost) {
            this.node = node;
            this.cost = cost;
        }

        @Override
        public int compare(Node node1, Node node2) {
            return Integer.compare(node1.cost, node2.cost);
        }
    }

    public static void main(String[] args) {
        int V = 5;
        Graph graph = new Graph(V);

        graph.addEdge(0, 1, 9);
        graph.addEdge(0, 2, 6);
        graph.addEdge(0, 3, 5);
        graph.addEdge(0, 4, 3);
        graph.addEdge(2, 1, 2);
        graph.addEdge(2, 3, 4);

        graph.dijkstra(0);
    }
}
```

### Giải thích:

- **Graph**: Lớp này đại diện cho đồ thị với số đỉnh `V` và danh sách kề `adj`.
- **Edge**: Lớp con để lưu trữ đích và trọng số của cạnh.
- **addEdge**: Hàm để thêm cạnh vào đồ thị.
- **dijkstra**: Hàm thực hiện thuật toán Dijkstra để tìm đường đi ngắn nhất từ đỉnh nguồn `src`.
- **printSolution**: Hàm để in ra khoảng cách từ đỉnh nguồn đến các đỉnh khác.
- **Node**: Lớp để lưu trữ thông tin về đỉnh và chi phí, được sử dụng trong hàng đợi ưu tiên.

### Đầu vào/Đầu ra:
- **Đầu vào**: Đồ thị với các cạnh và trọng số.
- **Đầu ra**: Khoảng cách ngắn nhất từ đỉnh nguồn đến các đỉnh khác trong đồ thị.
