```java
import java.util.*;

// Class to represent a graph
class Graph {
    private int vertices; // Number of vertices
    private LinkedList<Integer>[] adjacencyList; // Adjacency list

    // Constructor
    public Graph(int vertices) {
        this.vertices = vertices;
        adjacencyList = new LinkedList[vertices];
        for (int i = 0; i < vertices; i++) {
            adjacencyList[i] = new LinkedList<>();
        }
    }

    // Method to add an edge to the graph
    public void addEdge(int source, int destination) {
        adjacencyList[source].add(destination);
    }

    // DFS utility function
    private void DFSUtil(int vertex, boolean[] visited) {
        // Mark the current node as visited and print it
        visited[vertex] = true;
        System.out.print(vertex + " ");

        // Recur for all the vertices adjacent to this vertex
        for (int adjacent : adjacencyList[vertex]) {
            if (!visited[adjacent]) {
                DFSUtil(adjacent, visited);
            }
        }
    }

    // The DFS function that uses DFSUtil
    public void DFS(int startVertex) {
        // Mark all the vertices as not visited
        boolean[] visited = new boolean[vertices];

        // Call the recursive helper function to print DFS traversal
        DFSUtil(startVertex, visited);
    }
}

// Main class
public class DFSDemo {
    public static void main(String[] args) {
        // Create a graph with 4 vertices
        Graph graph = new Graph(4);

        // Add edges to the graph
        graph.addEdge(0, 1);
        graph.addEdge(0, 2);
        graph.addEdge(1, 2);
        graph.addEdge(2, 0);
        graph.addEdge(2, 3);
        graph.addEdge(3, 3);

        // Print DFS traversal starting from vertex 2
        System.out.println("Depth-First Traversal starting from vertex 2:");
        graph.DFS(2);
    }
}
```

### Explanation:

- **Graph Class**: Represents the graph using an adjacency list. It includes methods to add edges and perform DFS.
- **DFSUtil Method**: A recursive utility function that marks the current node as visited and explores all its adjacent nodes.
- **DFS Method**: Initializes the visited array and calls the DFS utility function.
- **Main Function**: Creates a graph, adds edges, and performs DFS starting from a specified vertex.
