Dưới đây là một ví dụ về mã Java cho cây tìm kiếm nhị phân (Binary Search Tree - BST) bao gồm các chức năng chính như thêm, tìm kiếm và duyệt cây. Mã này cũng bao gồm phần nhập và xuất dữ liệu.

```java
// Lớp Node đại diện cho một nút trong cây
class Node {
    int key;
    Node left, right;

    public Node(int item) {
        key = item;
        left = right = null;
    }
}

// Lớp BinarySearchTree chứa các phương thức chính
class BinarySearchTree {
    Node root;

    // Hàm khởi tạo
    BinarySearchTree() {
        root = null;
    }

    // Phương thức thêm một phần tử vào cây
    void insert(int key) {
        root = insertRec(root, key);
    }

    // Hàm đệ quy để thêm phần tử mới
    Node insertRec(Node root, int key) {
        if (root == null) {
            root = new Node(key);
            return root;
        }
        if (key < root.key)
            root.left = insertRec(root.left, key);
        else if (key > root.key)
            root.right = insertRec(root.right, key);

        return root;
    }

    // Phương thức tìm kiếm một phần tử trong cây
    boolean search(int key) {
        return searchRec(root, key);
    }

    // Hàm đệ quy để tìm kiếm phần tử
    boolean searchRec(Node root, int key) {
        if (root == null)
            return false;
        if (root.key == key)
            return true;
        return key < root.key ? searchRec(root.left, key) : searchRec(root.right, key);
    }

    // Phương thức duyệt cây theo thứ tự trung vị (Inorder)
    void inorder() {
        inorderRec(root);
    }

    // Hàm đệ quy để duyệt cây
    void inorderRec(Node root) {
        if (root != null) {
            inorderRec(root.left);
            System.out.print(root.key + " ");
            inorderRec(root.right);
        }
    }

    // Hàm main để chạy chương trình
    public static void main(String[] args) {
        BinarySearchTree tree = new BinarySearchTree();

        // Thêm các phần tử vào cây
        tree.insert(50);
        tree.insert(30);
        tree.insert(20);
        tree.insert(40);
        tree.insert(70);
        tree.insert(60);
        tree.insert(80);

        // Duyệt cây theo thứ tự trung vị
        System.out.println("Duyệt cây theo thứ tự trung vị:");
        tree.inorder();

        // Tìm kiếm phần tử trong cây
        int keyToSearch = 40;
        System.out.println("\nTìm kiếm phần tử " + keyToSearch + ": " + (tree.search(keyToSearch) ? "Tìm thấy" : "Không tìm thấy"));
    }
}
```

### Giải thích:
- **Node**: Lớp này đại diện cho một nút trong cây với giá trị `key` và hai con trỏ `left` và `right`.
- **BinarySearchTree**: Lớp này chứa các phương thức để thêm (`insert`), tìm kiếm (`search`), và duyệt cây (`inorder`).
- **insertRec**: Hàm đệ quy để thêm một phần tử mới vào cây.
- **searchRec**: Hàm đệ quy để tìm kiếm một phần tử trong cây.
- **inorderRec**: Hàm đệ quy để duyệt cây theo thứ tự trung vị.
- **main**: Phương thức chính để chạy chương trình, thêm các phần tử vào cây, duyệt cây và tìm kiếm phần tử.
