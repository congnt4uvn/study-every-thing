Dưới đây là một ví dụ về cách triển khai thuật toán Kruskal để tìm cây khung nhỏ nhất (Minimum Spanning Tree) bằng ngôn ngữ lập trình Java. Mã này bao gồm các cấu trúc dữ liệu cần thiết, thuật toán chính, và ví dụ về đầu vào và đầu ra.

```java
import java.util.*;

class Edge implements Comparable<Edge> {
    int src, dest, weight;

    public Edge(int src, int dest, int weight) {
        this.src = src;
        this.dest = dest;
        this.weight = weight;
    }

    // So sánh các cạnh dựa trên trọng số
    public int compareTo(Edge compareEdge) {
        return this.weight - compareEdge.weight;
    }
}

class Subset {
    int parent, rank;
}

public class KruskalAlgorithm {
    int V, E; // Số đỉnh và số cạnh
    Edge[] edges; // Mảng chứa tất cả các cạnh

    // Khởi tạo đồ thị với số đỉnh và số cạnh
    KruskalAlgorithm(int v, int e) {
        V = v;
        E = e;
        edges = new Edge[E];
    }

    // Tìm tập hợp của một phần tử i
    int find(Subset[] subsets, int i) {
        if (subsets[i].parent != i) {
            subsets[i].parent = find(subsets, subsets[i].parent);
        }
        return subsets[i].parent;
    }

    // Hợp nhất hai tập hợp x và y
    void union(Subset[] subsets, int x, int y) {
        int xroot = find(subsets, x);
        int yroot = find(subsets, y);

        if (subsets[xroot].rank < subsets[yroot].rank) {
            subsets[xroot].parent = yroot;
        } else if (subsets[xroot].rank > subsets[yroot].rank) {
            subsets[yroot].parent = xroot;
        } else {
            subsets[yroot].parent = xroot;
            subsets[xroot].rank++;
        }
    }

    // Hàm chính để tìm cây khung nhỏ nhất
    void kruskalMST() {
        Edge[] result = new Edge[V]; // Mảng để lưu kết quả
        int e = 0; // Số cạnh trong cây khung nhỏ nhất
        int i = 0; // Chỉ số cho các cạnh đã sắp xếp

        // Bước 1: Sắp xếp tất cả các cạnh theo thứ tự tăng dần của trọng số
        Arrays.sort(edges);

        // Tạo V tập hợp rời rạc
        Subset[] subsets = new Subset[V];
        for (int v = 0; v < V; ++v) {
            subsets[v] = new Subset();
            subsets[v].parent = v;
            subsets[v].rank = 0;
        }

        // Số cạnh cần thêm vào cây khung nhỏ nhất là V-1
        while (e < V - 1) {
            // Bước 2: Lấy cạnh có trọng số nhỏ nhất
            Edge nextEdge = edges[i++];

            int x = find(subsets, nextEdge.src);
            int y = find(subsets, nextEdge.dest);

            // Nếu không tạo thành chu trình, thêm cạnh vào kết quả
            if (x != y) {
                result[e++] = nextEdge;
                union(subsets, x, y);
            }
        }

        // In kết quả
        System.out.println("Các cạnh trong cây khung nhỏ nhất:");
        for (i = 0; i < e; ++i) {
            System.out.println(result[i].src + " -- " + result[i].dest + " == " + result[i].weight);
        }
    }

    public static void main(String[] args) {
        int V = 4; // Số đỉnh
        int E = 5; // Số cạnh
        KruskalAlgorithm graph = new KruskalAlgorithm(V, E);

        // Thêm các cạnh vào đồ thị
        graph.edges[0] = new Edge(0, 1, 10);
        graph.edges[1] = new Edge(0, 2, 6);
        graph.edges[2] = new Edge(0, 3, 5);
        graph.edges[3] = new Edge(1, 3, 15);
        graph.edges[4] = new Edge(2, 3, 4);

        // Tìm cây khung nhỏ nhất
        graph.kruskalMST();
    }
}
```

### Giải thích:
- **Edge**: Lớp đại diện cho một cạnh trong đồ thị, với các thuộc tính `src` (đỉnh nguồn), `dest` (đỉnh đích), và `weight` (trọng số).
- **Subset**: Lớp đại diện cho một tập hợp, dùng để quản lý các tập hợp rời rạc trong thuật toán Kruskal.
- **KruskalAlgorithm**: Lớp chính chứa các phương thức để thực hiện thuật toán Kruskal.
- **find** và **union**: Các hàm hỗ trợ để tìm và hợp nhất các tập hợp.
- **kruskalMST**: Hàm chính để tìm cây khung nhỏ nhất.
- **main**: Hàm chính để chạy chương trình, tạo đồ thị và gọi hàm `kruskalMST`.

### Đầu vào và đầu ra:
- **Đầu vào**: Đồ thị với các đỉnh và cạnh được định nghĩa trong hàm `main`.
- **Đầu ra**: Các cạnh trong cây khung nhỏ nhất được in ra màn hình.
