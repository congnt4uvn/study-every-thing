Dưới đây là một ví dụ về cách triển khai cây AVL trong Java, bao gồm các chức năng chính như chèn, xóa và duyệt cây. Mã này cũng bao gồm phần nhập và xuất dữ liệu.

```java
class Node {
    int key, height;
    Node left, right;

    Node(int d) {
        key = d;
        height = 1;
    }
}

class AVLTree {
    private Node root;

    // Hàm để lấy chiều cao của cây
    int height(Node N) {
        if (N == null)
            return 0;
        return N.height;
    }

    // Hàm để lấy giá trị lớn hơn
    int max(int a, int b) {
        return (a > b) ? a : b;
    }

    // Xoay phải
    Node rightRotate(Node y) {
        Node x = y.left;
        Node T2 = x.right;

        // Thực hiện xoay
        x.right = y;
        y.left = T2;

        // Cập nhật chiều cao
        y.height = max(height(y.left), height(y.right)) + 1;
        x.height = max(height(x.left), height(x.right)) + 1;

        // Trả về nút mới
        return x;
    }

    // Xoay trái
    Node leftRotate(Node x) {
        Node y = x.right;
        Node T2 = y.left;

        // Thực hiện xoay
        y.left = x;
        x.right = T2;

        // Cập nhật chiều cao
        x.height = max(height(x.left), height(x.right)) + 1;
        y.height = max(height(y.left), height(y.right)) + 1;

        // Trả về nút mới
        return y;
    }

    // Lấy hệ số cân bằng của nút N
    int getBalance(Node N) {
        if (N == null)
            return 0;
        return height(N.left) - height(N.right);
    }

    // Hàm chèn một khóa vào cây AVL
    Node insert(Node node, int key) {
        // 1. Thực hiện chèn bình thường
        if (node == null)
            return (new Node(key));

        if (key < node.key)
            node.left = insert(node.left, key);
        else if (key > node.key)
            node.right = insert(node.right, key);
        else // Không cho phép các khóa trùng lặp
            return node;

        // 2. Cập nhật chiều cao của nút tổ tiên này
        node.height = 1 + max(height(node.left), height(node.right));

        // 3. Lấy hệ số cân bằng của nút tổ tiên này để kiểm tra xem nút này có mất cân bằng không
        int balance = getBalance(node);

        // Nếu nút này mất cân bằng, thì có 4 trường hợp

        // Trường hợp Left Left
        if (balance > 1 && key < node.left.key)
            return rightRotate(node);

        // Trường hợp Right Right
        if (balance < -1 && key > node.right.key)
            return leftRotate(node);

        // Trường hợp Left Right
        if (balance > 1 && key > node.left.key) {
            node.left = leftRotate(node.left);
            return rightRotate(node);
        }

        // Trường hợp Right Left
        if (balance < -1 && key < node.right.key) {
            node.right = rightRotate(node.right);
            return leftRotate(node);
        }

        // Trả về con trỏ nút (không thay đổi)
        return node;
    }

    // Hàm để in duyệt cây theo thứ tự
    void preOrder(Node node) {
        if (node != null) {
            System.out.print(node.key + " ");
            preOrder(node.left);
            preOrder(node.right);
        }
    }

    // Hàm chính để kiểm tra
    public static void main(String[] args) {
        AVLTree tree = new AVLTree();

        /* Chèn các giá trị vào cây AVL */
        tree.root = tree.insert(tree.root, 10);
        tree.root = tree.insert(tree.root, 20);
        tree.root = tree.insert(tree.root, 30);
        tree.root = tree.insert(tree.root, 40);
        tree.root = tree.insert(tree.root, 50);
        tree.root = tree.insert(tree.root, 25);

        /* In duyệt cây theo thứ tự */
        System.out.println("Duyệt cây theo thứ tự:");
        tree.preOrder(tree.root);
    }
}
```

### Giải thích:
- **Node**: Lớp này đại diện cho một nút trong cây AVL, chứa khóa, chiều cao và các con trỏ đến nút con trái và phải.
- **AVLTree**: Lớp này chứa các phương thức để thực hiện các thao tác trên cây AVL như chèn, xoay trái, xoay phải và duyệt cây.
- **main**: Phương thức này tạo một cây AVL, chèn các phần tử vào cây và in ra thứ tự duyệt cây.

Bạn có thể chạy mã này trong môi trường Java để thấy cách hoạt động của cây AVL.
