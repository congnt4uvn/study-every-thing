Dưới đây là một ví dụ về danh sách liên kết đơn giản bằng Java, bao gồm các chức năng chính như thêm, xóa, và hiển thị các phần tử. Mã này cũng bao gồm phần nhập và xuất dữ liệu.

```java
class Node {
    int data;
    Node next;

    public Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class SinglyLinkedList {
    private Node head;

    public SinglyLinkedList() {
        this.head = null;
    }

    // Thêm phần tử vào cuối danh sách
    public void add(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = newNode;
        } else {
            Node current = head;
            while (current.next != null) {
                current = current.next;
            }
            current.next = newNode;
        }
    }

    // Xóa phần tử đầu tiên có giá trị cho trước
    public void remove(int data) {
        if (head == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }

        if (head.data == data) {
            head = head.next;
            return;
        }

        Node current = head;
        while (current.next != null && current.next.data != data) {
            current = current.next;
        }

        if (current.next == null) {
            System.out.println("Không tìm thấy phần tử.");
        } else {
            current.next = current.next.next;
        }
    }

    // Hiển thị danh sách
    public void display() {
        if (head == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }

        Node current = head;
        while (current != null) {
            System.out.print(current.data + " ");
            current = current.next;
        }
        System.out.println();
    }
}

public class Main {
    public static void main(String[] args) {
        SinglyLinkedList list = new SinglyLinkedList();

        // Thêm phần tử vào danh sách
        list.add(10);
        list.add(20);
        list.add(30);

        // Hiển thị danh sách
        System.out.println("Danh sách sau khi thêm:");
        list.display();

        // Xóa phần tử
        list.remove(20);
        System.out.println("Danh sách sau khi xóa 20:");
        list.display();

        // Thử xóa phần tử không tồn tại
        list.remove(40);
    }
}
```

### Giải thích:
- **Node**: Lớp này đại diện cho một nút trong danh sách liên kết, chứa dữ liệu và con trỏ đến nút tiếp theo.
- **SinglyLinkedList**: Lớp này quản lý danh sách liên kết, bao gồm các phương thức để thêm, xóa và hiển thị các phần tử.
- **Main**: Lớp này chứa phương thức `main` để chạy chương trình, thực hiện các thao tác thêm, xóa và hiển thị danh sách.

### Nhập/Xuất:
- Chương trình thêm các phần tử 10, 20, 30 vào danh sách.
- Sau đó, nó hiển thị danh sách, xóa phần tử 20, và hiển thị lại danh sách.
- Cuối cùng, nó thử xóa một phần tử không tồn tại (40) để minh họa thông báo lỗi.
