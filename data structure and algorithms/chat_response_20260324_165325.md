Dưới đây là một ví dụ về cách triển khai thuật toán Rabin-Karp trong Java để tìm kiếm chuỗi con. Thuật toán này sử dụng hàm băm để tìm kiếm chuỗi con trong một chuỗi lớn hơn.

```java
public class RabinKarp {
    // Số nguyên tố lớn để tính toán băm
    private final static int PRIME = 101;

    // Hàm để tính giá trị băm của một chuỗi
    private static long createHash(String str, int end) {
        long hash = 0;
        for (int i = 0; i < end; i++) {
            hash += str.charAt(i) * Math.pow(PRIME, i);
        }
        return hash;
    }

    // Hàm để tính lại giá trị băm khi trượt cửa sổ
    private static long recalculateHash(String str, int oldIndex, int newIndex, long oldHash, int patternLen) {
        long newHash = oldHash - str.charAt(oldIndex);
        newHash /= PRIME;
        newHash += str.charAt(newIndex) * Math.pow(PRIME, patternLen - 1);
        return newHash;
    }

    // Hàm để kiểm tra hai chuỗi có khớp nhau không
    private static boolean checkEqual(String str1, int start1, int end1, String str2, int start2, int end2) {
        if (end1 - start1 != end2 - start2) {
            return false;
        }
        while (start1 <= end1 && start2 <= end2) {
            if (str1.charAt(start1) != str2.charAt(start2)) {
                return false;
            }
            start1++;
            start2++;
        }
        return true;
    }

    // Hàm chính để tìm kiếm chuỗi con
    public static int rabinKarp(String text, String pattern) {
        int m = pattern.length();
        int n = text.length();
        long patternHash = createHash(pattern, m);
        long textHash = createHash(text, m);

        for (int i = 0; i <= n - m; i++) {
            if (patternHash == textHash && checkEqual(text, i, i + m - 1, pattern, 0, m - 1)) {
                return i; // Trả về vị trí đầu tiên tìm thấy
            }
            if (i < n - m) {
                textHash = recalculateHash(text, i, i + m, textHash, m);
            }
        }
        return -1; // Không tìm thấy
    }

    public static void main(String[] args) {
        String text = "xin chào, đây là một ví dụ về thuật toán Rabin-Karp";
        String pattern = "thuật toán";
        int result = rabinKarp(text, pattern);
        if (result != -1) {
            System.out.println("Chuỗi con được tìm thấy tại vị trí: " + result);
        } else {
            System.out.println("Không tìm thấy chuỗi con.");
        }
    }
}
```

### Giải thích:
- **createHash**: Tính giá trị băm cho một chuỗi con.
- **recalculateHash**: Tính lại giá trị băm khi cửa sổ trượt qua chuỗi.
- **checkEqual**: So sánh hai chuỗi để kiểm tra xem chúng có giống nhau không.
- **rabinKarp**: Hàm chính để tìm kiếm chuỗi con trong chuỗi lớn hơn.

### Đầu vào và đầu ra:
- **Đầu vào**: Chuỗi lớn `text` và chuỗi con `pattern` cần tìm.
- **Đầu ra**: Vị trí đầu tiên của chuỗi con trong chuỗi lớn, hoặc -1 nếu không tìm thấy.
