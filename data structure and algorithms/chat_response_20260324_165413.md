```java
import java.util.Scanner;
import java.util.Stack;

public class StackExample {

    public static void main(String[] args) {
        // Create a stack of characters
        Stack<Character> stack = new Stack<>();

        // Input: Get a string from the user
        Scanner scanner = new Scanner(System.in);
        System.out.println("Enter a string to reverse:");
        String input = scanner.nextLine();

        // Push all characters of the string onto the stack
        for (char ch : input.toCharArray()) {
            stack.push(ch);
        }

        // Pop all characters from the stack to reverse the string
        StringBuilder reversedString = new StringBuilder();
        while (!stack.isEmpty()) {
            reversedString.append(stack.pop());
        }

        // Output: Display the reversed string
        System.out.println("Reversed string: " + reversedString.toString());

        // Demonstrate basic stack operations
        System.out.println("\nDemonstrating basic stack operations:");
        stack.push('A');
        stack.push('B');
        stack.push('C');
        System.out.println("Stack after pushing A, B, C: " + stack);

        char topElement = stack.peek();
        System.out.println("Top element (peek): " + topElement);

        char poppedElement = stack.pop();
        System.out.println("Popped element: " + poppedElement);
        System.out.println("Stack after popping: " + stack);

        scanner.close();
    }
}
```

### Explanation:

1. **Stack Creation**: We create a `Stack<Character>` to hold characters.

2. **Input**: We use a `Scanner` to read a string from the user.

3. **Push Operation**: We iterate over the input string and push each character onto the stack.

4. **Pop Operation**: We pop characters from the stack to build the reversed string.

5. **Output**: We print the reversed string.

6. **Basic Stack Operations**: We demonstrate pushing, peeking, and popping elements from the stack.

This code provides a simple demonstration of how stacks can be used in Java for both basic operations and a practical algorithmic task like reversing a string.
