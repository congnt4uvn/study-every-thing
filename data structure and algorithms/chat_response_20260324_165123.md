Dưới đây là một ví dụ về cách triển khai thuật toán Tìm kiếm theo chiều sâu (Depth-First Search - DFS) bằng ngôn ngữ Java. Chương trình này bao gồm các cấu trúc dữ liệu và thuật toán cần thiết, cùng với hàm `main` để chạy chương trình. Chương trình sẽ in ra thứ tự các đỉnh được duyệt theo DFS.

```java
import java.util.*;

public class DFSExample {
    // Lớp biểu diễn đồ thị sử dụng danh sách kề
    static class Graph {
        private int V; // Số đỉnh
        private LinkedList<Integer> adj[]; // Danh sách kề

        // Constructor
        Graph(int v) {
            V = v;
            adj = new LinkedList[v];
            for (int i = 0; i < v; ++i)
                adj[i] = new LinkedList();
        }

        // Hàm thêm cạnh vào đồ thị
        void addEdge(int v, int w) {
            adj[v].add(w); // Thêm w vào danh sách kề của v
        }

        // Hàm DFS sử dụng đệ quy
        void DFSUtil(int v, boolean visited[]) {
            // Đánh dấu đỉnh hiện tại là đã thăm
            visited[v] = true;
            System.out.print(v + " ");

            // Duyệt tất cả các đỉnh kề với đỉnh v
            Iterator<Integer> i = adj[v].listIterator();
            while (i.hasNext()) {
                int n = i.next();
                if (!visited[n])
                    DFSUtil(n, visited);
            }
        }

        // Hàm thực hiện DFS
        void DFS(int v) {
            // Đánh dấu tất cả các đỉnh là chưa thăm
            boolean visited[] = new boolean[V];

            // Gọi hàm DFSUtil để duyệt các đỉnh
            DFSUtil(v, visited);
        }
    }

    // Hàm main
    public static void main(String args[]) {
        Graph g = new Graph(4);

        g.addEdge(0, 1);
        g.addEdge(0, 2);
        g.addEdge(1, 2);
        g.addEdge(2, 0);
        g.addEdge(2, 3);
        g.addEdge(3, 3);

        System.out.println("Duyệt đồ thị theo DFS bắt đầu từ đỉnh 2:");

        g.DFS(2);
    }
}
```

### Giải thích:
- **Graph**: Lớp này biểu diễn đồ thị sử dụng danh sách kề.
- **addEdge**: Hàm này thêm một cạnh vào đồ thị.
- **DFSUtil**: Hàm đệ quy để thực hiện DFS từ một đỉnh cụ thể.
- **DFS**: Hàm này khởi tạo mảng `visited` và gọi `DFSUtil` để bắt đầu duyệt từ đỉnh cho trước.
- **main**: Tạo một đồ thị, thêm các cạnh, và thực hiện DFS bắt đầu từ đỉnh 2.

### Đầu vào và đầu ra:
- **Đầu vào**: Đồ thị được định nghĩa trong hàm `main` với các cạnh được thêm vào.
- **Đầu ra**: Thứ tự các đỉnh được duyệt theo DFS, bắt đầu từ đỉnh 2. Ví dụ: `2 0 1 3`.
