Dưới đây là một ví dụ đơn giản về cách triển khai đồ thị trong Java, bao gồm các chức năng chính như thêm đỉnh, thêm cạnh, và hiển thị đồ thị. Mã này sử dụng danh sách kề để biểu diễn đồ thị.

```java
import java.util.*;

class Graph {
    private Map<Integer, List<Integer>> adjList;

    // Khởi tạo đồ thị
    public Graph() {
        adjList = new HashMap<>();
    }

    // Thêm đỉnh vào đồ thị
    public void addVertex(int vertex) {
        adjList.putIfAbsent(vertex, new ArrayList<>());
    }

    // Thêm cạnh vào đồ thị
    public void addEdge(int source, int destination) {
        adjList.putIfAbsent(source, new ArrayList<>());
        adjList.putIfAbsent(destination, new ArrayList<>());
        adjList.get(source).add(destination);
        adjList.get(destination).add(source); // Nếu là đồ thị có hướng, bỏ dòng này
    }

    // Hiển thị đồ thị
    public void printGraph() {
        for (Map.Entry<Integer, List<Integer>> entry : adjList.entrySet()) {
            System.out.print("Đỉnh " + entry.getKey() + ": ");
            for (Integer neighbor : entry.getValue()) {
                System.out.print(neighbor + " ");
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Graph graph = new Graph();

        System.out.println("Nhập số lượng đỉnh:");
        int vertices = scanner.nextInt();

        System.out.println("Nhập các đỉnh:");
        for (int i = 0; i < vertices; i++) {
            int vertex = scanner.nextInt();
            graph.addVertex(vertex);
        }

        System.out.println("Nhập số lượng cạnh:");
        int edges = scanner.nextInt();

        System.out.println("Nhập các cạnh (đỉnh nguồn và đỉnh đích):");
        for (int i = 0; i < edges; i++) {
            int source = scanner.nextInt();
            int destination = scanner.nextInt();
            graph.addEdge(source, destination);
        }

        System.out.println("Đồ thị:");
        graph.printGraph();

        scanner.close();
    }
}
```

### Giải thích:
- **Graph**: Lớp này biểu diễn đồ thị bằng cách sử dụng một `Map` để lưu trữ danh sách kề của mỗi đỉnh.
- **addVertex**: Thêm một đỉnh mới vào đồ thị.
- **addEdge**: Thêm một cạnh giữa hai đỉnh. Nếu đồ thị là có hướng, bạn có thể bỏ dòng `adjList.get(destination).add(source);`.
- **printGraph**: Hiển thị danh sách kề của mỗi đỉnh trong đồ thị.
- **main**: Hàm chính để nhập dữ liệu từ người dùng và hiển thị đồ thị.
