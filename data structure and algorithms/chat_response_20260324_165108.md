Dưới đây là một ví dụ về mã Java cho thuật toán Quick Sort, bao gồm các hàm chính, cấu trúc dữ liệu và thuật toán, cũng như phần nhập và xuất dữ liệu. Mã này được chú thích bằng tiếng Việt để bạn có thể dễ dàng hiểu cách hoạt động của nó.

```java
import java.util.Scanner;

public class QuickSortExample {

    // Hàm để thực hiện phân đoạn mảng
    private static int partition(int[] arr, int low, int high) {
        int pivot = arr[high]; // Chọn phần tử cuối làm chốt
        int i = (low - 1); // Chỉ số của phần tử nhỏ hơn

        for (int j = low; j < high; j++) {
            // Nếu phần tử hiện tại nhỏ hơn hoặc bằng chốt
            if (arr[j] <= pivot) {
                i++;
                // Hoán đổi arr[i] và arr[j]
                int temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }

        // Hoán đổi arr[i+1] và arr[high] (hoặc chốt)
        int temp = arr[i + 1];
        arr[i + 1] = arr[high];
        arr[high] = temp;

        return i + 1;
    }

    // Hàm thực hiện QuickSort
    private static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            // pi là chỉ số phân đoạn, arr[pi] đã ở đúng vị trí
            int pi = partition(arr, low, high);

            // Sắp xếp các phần tử trước và sau phân đoạn
            quickSort(arr, low, pi - 1);
            quickSort(arr, pi + 1, high);
        }
    }

    // Hàm để in mảng
    private static void printArray(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n; ++i) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
    }

    // Hàm chính
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Nhập số lượng phần tử của mảng:");
        int n = scanner.nextInt();
        int[] arr = new int[n];

        System.out.println("Nhập các phần tử của mảng:");
        for (int i = 0; i < n; i++) {
            arr[i] = scanner.nextInt();
        }

        System.out.println("Mảng trước khi sắp xếp:");
        printArray(arr);

        quickSort(arr, 0, n - 1);

        System.out.println("Mảng sau khi sắp xếp:");
        printArray(arr);

        scanner.close();
    }
}
```

### Giải thích:
- **partition**: Hàm này chọn phần tử cuối cùng làm chốt và sắp xếp các phần tử sao cho tất cả các phần tử nhỏ hơn hoặc bằng chốt nằm bên trái và các phần tử lớn hơn nằm bên phải.
- **quickSort**: Hàm này thực hiện sắp xếp nhanh bằng cách gọi đệ quy hàm `partition` và sắp xếp các phần tử trước và sau chốt.
- **printArray**: Hàm này dùng để in mảng ra màn hình.
- **main**: Hàm chính để nhập dữ liệu từ người dùng, thực hiện sắp xếp và in kết quả.
