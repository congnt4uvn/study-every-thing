Dưới đây là một ví dụ về cách biểu diễn đồ thị bằng ma trận kề và danh sách kề sử dụng bảng băm trong Java. Mã này bao gồm các chức năng chính để thêm cạnh, hiển thị đồ thị và một số thao tác cơ bản khác.

```java
import java.util.*;

public class Graph {
    private int numVertices;
    private int[][] adjacencyMatrix;
    private Map<Integer, List<Integer>> adjacencyList;

    public Graph(int numVertices) {
        this.numVertices = numVertices;
        adjacencyMatrix = new int[numVertices][numVertices];
        adjacencyList = new HashMap<>();
        for (int i = 0; i < numVertices; i++) {
            adjacencyList.put(i, new ArrayList<>());
        }
    }

    // Thêm cạnh vào đồ thị
    public void addEdge(int src, int dest) {
        // Ma trận kề
        adjacencyMatrix[src][dest] = 1;
        adjacencyMatrix[dest][src] = 1; // Nếu là đồ thị vô hướng

        // Danh sách kề
        adjacencyList.get(src).add(dest);
        adjacencyList.get(dest).add(src); // Nếu là đồ thị vô hướng
    }

    // Hiển thị ma trận kề
    public void displayAdjacencyMatrix() {
        System.out.println("Ma trận kề:");
        for (int i = 0; i < numVertices; i++) {
            for (int j = 0; j < numVertices; j++) {
                System.out.print(adjacencyMatrix[i][j] + " ");
            }
            System.out.println();
        }
    }

    // Hiển thị danh sách kề
    public void displayAdjacencyList() {
        System.out.println("Danh sách kề:");
        for (int i = 0; i < numVertices; i++) {
            System.out.print(i + ": ");
            for (int j : adjacencyList.get(i)) {
                System.out.print(j + " ");
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Nhập số đỉnh: ");
        int numVertices = scanner.nextInt();

        Graph graph = new Graph(numVertices);

        System.out.print("Nhập số cạnh: ");
        int numEdges = scanner.nextInt();

        System.out.println("Nhập các cạnh (đỉnh nguồn và đỉnh đích):");
        for (int i = 0; i < numEdges; i++) {
            int src = scanner.nextInt();
            int dest = scanner.nextInt();
            graph.addEdge(src, dest);
        }

        graph.displayAdjacencyMatrix();
        graph.displayAdjacencyList();

        scanner.close();
    }
}
```

### Giải thích:
- **Ma trận kề**: Sử dụng mảng hai chiều `adjacencyMatrix` để lưu trữ thông tin về các cạnh giữa các đỉnh.
- **Danh sách kề**: Sử dụng `Map` với khóa là đỉnh và giá trị là danh sách các đỉnh kề.
- **Thêm cạnh**: Hàm `addEdge` thêm cạnh vào cả ma trận kề và danh sách kề.
- **Hiển thị**: Các hàm `displayAdjacencyMatrix` và `displayAdjacencyList` để hiển thị cấu trúc đồ thị.

Chương trình này cho phép người dùng nhập số đỉnh và cạnh, sau đó nhập các cạnh để xây dựng đồ thị và hiển thị nó dưới cả hai dạng biểu diễn.
