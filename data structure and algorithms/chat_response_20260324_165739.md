Certainly! The Knuth-Morris-Pratt (KMP) algorithm is an efficient string matching algorithm that searches for occurrences of a "pattern" string within a "text" string. It preprocesses the pattern to create a partial match table (also known as the "lps" array) that helps in skipping unnecessary comparisons.

Here's a complete Java implementation of the KMP algorithm, including the main function, data structures, and input/output handling:

```java
public class KMPAlgorithm {

    // Function to perform KMP search
    public static void KMPSearch(String pattern, String text) {
        int M = pattern.length();
        int N = text.length();

        // Create lps[] that will hold the longest prefix suffix values for pattern
        int[] lps = new int[M];
        int j = 0; // index for pattern[]

        // Preprocess the pattern to create the lps array
        computeLPSArray(pattern, M, lps);

        int i = 0; // index for text[]
        while (i < N) {
            if (pattern.charAt(j) == text.charAt(i)) {
                j++;
                i++;
            }
            if (j == M) {
                System.out.println("Found pattern at index " + (i - j));
                j = lps[j - 1];
            } else if (i < N && pattern.charAt(j) != text.charAt(i)) {
                // Mismatch after j matches
                if (j != 0) {
                    j = lps[j - 1];
                } else {
                    i++;
                }
            }
        }
    }

    // Function to compute the LPS array
    private static void computeLPSArray(String pattern, int M, int[] lps) {
        int length = 0; // length of the previous longest prefix suffix
        int i = 1;
        lps[0] = 0; // lps[0] is always 0

        // Loop calculates lps[i] for i = 1 to M-1
        while (i < M) {
            if (pattern.charAt(i) == pattern.charAt(length)) {
                length++;
                lps[i] = length;
                i++;
            } else {
                // (pattern[i] != pattern[length])
                if (length != 0) {
                    length = lps[length - 1];
                } else {
                    lps[i] = length;
                    i++;
                }
            }
        }
    }

    // Main function to test the KMP algorithm
    public static void main(String[] args) {
        String text = "ABABDABACDABABCABAB";
        String pattern = "ABABCABAB";
        System.out.println("Text: " + text);
        System.out.println("Pattern: " + pattern);
        System.out.println("Pattern found at indices:");
        KMPSearch(pattern, text);
    }
}
```

### Explanation:

1. **KMPSearch Function**: This function searches for the pattern in the text using the KMP algorithm. It uses the `lps` array to skip unnecessary comparisons.

2. **computeLPSArray Function**: This function preprocesses the pattern to create the `lps` array, which stores the length of the longest proper prefix which is also a suffix for each sub-pattern.

3. **Main Function**: This is the entry point of the program. It initializes the text and pattern strings and calls the `KMPSearch` function to find and print the starting indices of all occurrences of the pattern in the text.

### Input/Output:

- **Input**: The `text` and `pattern` strings are hardcoded in the `main` function.
- **Output**: The program prints the starting indices of each occurrence of the pattern in the text.

You can modify the `text` and `pattern` variables in the `main` function to test with different inputs.
