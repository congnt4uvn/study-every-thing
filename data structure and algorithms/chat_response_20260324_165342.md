```java
import java.util.ArrayList;
import java.util.Collections;
import java.util.Scanner;

public class ArrayListExample {

    public static void main(String[] args) {
        // Create an ArrayList to store integers
        ArrayList<Integer> numbers = new ArrayList<>();

        // Create a Scanner object for input
        Scanner scanner = new Scanner(System.in);

        // Input: Add elements to the ArrayList
        System.out.println("Enter 5 integers:");
        for (int i = 0; i < 5; i++) {
            System.out.print("Enter number " + (i + 1) + ": ");
            int number = scanner.nextInt();
            numbers.add(number);
        }

        // Output: Display the elements of the ArrayList
        System.out.println("\nArrayList elements:");
        for (int number : numbers) {
            System.out.println(number);
        }

        // Access an element at a specific index
        System.out.println("\nElement at index 2: " + numbers.get(2));

        // Remove an element at a specific index
        numbers.remove(2);
        System.out.println("\nArrayList after removing element at index 2:");
        for (int number : numbers) {
            System.out.println(number);
        }

        // Sort the ArrayList
        Collections.sort(numbers);
        System.out.println("\nSorted ArrayList:");
        for (int number : numbers) {
            System.out.println(number);
        }

        // Close the scanner
        scanner.close();
    }
}
```

### Explanation:

1. **ArrayList Creation**: We create an `ArrayList` named `numbers` to store integers.

2. **Input**: We use a `Scanner` to read 5 integers from the user and add them to the `ArrayList`.

3. **Output**: We print the elements of the `ArrayList`.

4. **Accessing Elements**: We demonstrate accessing an element at a specific index using `get()`.

5. **Removing Elements**: We remove an element at a specific index using `remove()`.

6. **Sorting**: We sort the `ArrayList` using `Collections.sort()`.

7. **Closing Scanner**: We close the `Scanner` to prevent resource leaks.

This code provides a basic overview of how to use an `ArrayList` in Java, including input and output operations, and demonstrates some common data structure operations.
