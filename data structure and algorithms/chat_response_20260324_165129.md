Dưới đây là một ví dụ về cách triển khai thuật toán Tìm kiếm theo chiều rộng (Breadth-First Search - BFS) bằng ngôn ngữ Java. Chương trình này bao gồm cấu trúc dữ liệu, thuật toán, và các hàm chính để thực hiện BFS trên một đồ thị đơn giản. 

```java
import java.util.*;

public class BFSExample {
    // Lớp đại diện cho một đồ thị sử dụng danh sách kề
    static class Graph {
        private int numVertices; // Số đỉnh
        private LinkedList<Integer> adjList[]; // Danh sách kề

        // Hàm khởi tạo
        Graph(int numVertices) {
            this.numVertices = numVertices;
            adjList = new LinkedList[numVertices];
            for (int i = 0; i < numVertices; i++) {
                adjList[i] = new LinkedList<>();
            }
        }

        // Hàm thêm cạnh vào đồ thị
        void addEdge(int src, int dest) {
            adjList[src].add(dest);
        }

        // Hàm thực hiện BFS từ một đỉnh cho trước
        void BFS(int startVertex) {
            boolean visited[] = new boolean[numVertices];
            LinkedList<Integer> queue = new LinkedList<>();

            visited[startVertex] = true;
            queue.add(startVertex);

            while (!queue.isEmpty()) {
                int vertex = queue.poll();
                System.out.print(vertex + " ");

                Iterator<Integer> iterator = adjList[vertex].listIterator();
                while (iterator.hasNext()) {
                    int adjVertex = iterator.next();
                    if (!visited[adjVertex]) {
                        visited[adjVertex] = true;
                        queue.add(adjVertex);
                    }
                }
            }
        }
    }

    // Hàm main
    public static void main(String args[]) {
        Graph graph = new Graph(6);

        // Thêm các cạnh vào đồ thị
        graph.addEdge(0, 1);
        graph.addEdge(0, 2);
        graph.addEdge(1, 3);
        graph.addEdge(1, 4);
        graph.addEdge(2, 4);
        graph.addEdge(3, 5);
        graph.addEdge(4, 5);

        System.out.println("Kết quả duyệt BFS bắt đầu từ đỉnh 0:");
        graph.BFS(0);
    }
}
```

### Giải thích:

- **Graph Class**: Lớp `Graph` đại diện cho đồ thị, sử dụng danh sách kề để lưu trữ các cạnh.
- **addEdge Method**: Thêm một cạnh từ đỉnh `src` đến đỉnh `dest`.
- **BFS Method**: Thực hiện duyệt BFS từ đỉnh `startVertex`. Sử dụng một hàng đợi để theo dõi các đỉnh cần duyệt và một mảng `visited` để đánh dấu các đỉnh đã được duyệt.
- **Main Function**: Tạo một đồ thị với 6 đỉnh, thêm các cạnh, và thực hiện BFS bắt đầu từ đỉnh 0.

### Đầu vào và đầu ra:

- **Đầu vào**: Đồ thị được định nghĩa trong hàm `main` với các cạnh được thêm vào.
- **Đầu ra**: Kết quả duyệt BFS được in ra màn hình, bắt đầu từ đỉnh 0. Ví dụ: `0 1 2 3 4 5`.
