Creating a simple graph data structure in Java involves defining classes for the graph, vertices, and edges, and implementing basic operations such as adding vertices and edges. Below is a basic example of a graph implementation in Java, including input and output functionality.

```java
import java.util.*;

class Graph {
    private Map<String, List<String>> adjVertices;

    public Graph() {
        adjVertices = new HashMap<>();
    }

    // Add a vertex to the graph
    public void addVertex(String label) {
        adjVertices.putIfAbsent(label, new ArrayList<>());
    }

    // Add an edge between two vertices
    public void addEdge(String label1, String label2) {
        adjVertices.get(label1).add(label2);
        adjVertices.get(label2).add(label1);
    }

    // Get the list of adjacent vertices
    public List<String> getAdjVertices(String label) {
        return adjVertices.get(label);
    }

    // Print the graph
    public void printGraph() {
        for (String vertex : adjVertices.keySet()) {
            System.out.print(vertex + " -> ");
            for (String adjVertex : adjVertices.get(vertex)) {
                System.out.print(adjVertex + " ");
            }
            System.out.println();
        }
    }
}

public class GraphExample {
    public static void main(String[] args) {
        Graph graph = new Graph();

        // Sample input: adding vertices
        graph.addVertex("A");
        graph.addVertex("B");
        graph.addVertex("C");
        graph.addVertex("D");

        // Sample input: adding edges
        graph.addEdge("A", "B");
        graph.addEdge("A", "C");
        graph.addEdge("B", "D");
        graph.addEdge("C", "D");

        // Output: print the graph
        System.out.println("Graph adjacency list:");
        graph.printGraph();
    }
}
```

### Explanation:

1. **Graph Class**: 
   - Uses a `Map<String, List<String>>` to store adjacency lists for each vertex.
   - `addVertex(String label)`: Adds a vertex to the graph.
   - `addEdge(String label1, String label2)`: Adds an edge between two vertices. This example assumes an undirected graph, so it adds the connection in both directions.
   - `getAdjVertices(String label)`: Returns the list of adjacent vertices for a given vertex.
   - `printGraph()`: Prints the adjacency list representation of the graph.

2. **GraphExample Class**:
   - Demonstrates how to use the `Graph` class by adding vertices and edges.
   - Prints the graph to show its structure.

This code provides a basic framework for a graph data structure in Java. You can expand it by adding more functionalities, such as removing vertices or edges, checking for the existence of a path between two nodes, or implementing graph traversal algorithms like BFS or DFS.
