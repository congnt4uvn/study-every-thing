Dưới đây là một ví dụ về cách giải Sudoku bằng cách sử dụng thuật toán quay lui (backtracking) trong Java. Chương trình này bao gồm tất cả các hàm chính, cấu trúc dữ liệu và thuật toán cần thiết để giải một bảng Sudoku đã được điền một phần.

```java
public class SudokuSolver {
    private static final int SIZE = 9; // Kích thước của bảng Sudoku

    // Hàm để in bảng Sudoku
    private static void printBoard(int[][] board) {
        for (int row = 0; row < SIZE; row++) {
            for (int col = 0; col < SIZE; col++) {
                System.out.print(board[row][col] + " ");
            }
            System.out.println();
        }
    }

    // Kiểm tra xem có thể đặt số vào vị trí (row, col) không
    private static boolean isSafe(int[][] board, int row, int col, int num) {
        // Kiểm tra hàng
        for (int x = 0; x < SIZE; x++) {
            if (board[row][x] == num) {
                return false;
            }
        }

        // Kiểm tra cột
        for (int x = 0; x < SIZE; x++) {
            if (board[x][col] == num) {
                return false;
            }
        }

        // Kiểm tra ô 3x3
        int startRow = row - row % 3;
        int startCol = col - col % 3;
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                if (board[i + startRow][j + startCol] == num) {
                    return false;
                }
            }
        }

        return true;
    }

    // Hàm giải Sudoku bằng cách sử dụng quay lui
    private static boolean solveSudoku(int[][] board) {
        for (int row = 0; row < SIZE; row++) {
            for (int col = 0; col < SIZE; col++) {
                // Tìm ô trống
                if (board[row][col] == 0) {
                    // Thử các số từ 1 đến 9
                    for (int num = 1; num <= SIZE; num++) {
                        if (isSafe(board, row, col, num)) {
                            board[row][col] = num;

                            // Đệ quy để tiếp tục giải
                            if (solveSudoku(board)) {
                                return true;
                            }

                            // Nếu không giải được, quay lui
                            board[row][col] = 0;
                        }
                    }
                    return false; // Không có số nào phù hợp
                }
            }
        }
        return true; // Đã giải xong
    }

    public static void main(String[] args) {
        int[][] board = {
            {5, 3, 0, 0, 7, 0, 0, 0, 0},
            {6, 0, 0, 1, 9, 5, 0, 0, 0},
            {0, 9, 8, 0, 0, 0, 0, 6, 0},
            {8, 0, 0, 0, 6, 0, 0, 0, 3},
            {4, 0, 0, 8, 0, 3, 0, 0, 1},
            {7, 0, 0, 0, 2, 0, 0, 0, 6},
            {0, 6, 0, 0, 0, 0, 2, 8, 0},
            {0, 0, 0, 4, 1, 9, 0, 0, 5},
            {0, 0, 0, 0, 8, 0, 0, 7, 9}
        };

        if (solveSudoku(board)) {
            printBoard(board);
        } else {
            System.out.println("Không thể giải được Sudoku này.");
        }
    }
}
```

### Giải thích:
- **Cấu trúc dữ liệu**: Sử dụng mảng 2 chiều `int[][]` để biểu diễn bảng Sudoku.
- **Hàm `isSafe`**: Kiểm tra xem có thể đặt một số vào vị trí cụ thể mà không vi phạm quy tắc Sudoku.
- **Hàm `solveSudoku`**: Sử dụng thuật toán quay lui để thử các khả năng và giải bảng Sudoku.
- **Hàm `printBoard`**: In bảng Sudoku ra màn hình.
- **Hàm `main`**: Khởi tạo bảng Sudoku và gọi hàm giải.

### Đầu vào và đầu ra:
- **Đầu vào**: Một bảng Sudoku đã điền một phần (các ô trống được biểu diễn bằng số 0).
- **Đầu ra**: Bảng Sudoku đã được giải hoặc thông báo không thể giải được.
