Dưới đây là một ví dụ đơn giản về cách sử dụng `ArrayList` trong Java, bao gồm các chức năng chính như thêm, xóa, tìm kiếm và duyệt qua các phần tử. Mã này cũng bao gồm nhập và xuất dữ liệu. Mã được chú thích bằng tiếng Việt để bạn dễ hiểu hơn.

```java
import java.util.ArrayList;
import java.util.Scanner;

public class ArrayListExample {
    public static void main(String[] args) {
        // Tạo một ArrayList để lưu trữ các số nguyên
        ArrayList<Integer> numbers = new ArrayList<>();
        Scanner scanner = new Scanner(System.in);

        // Nhập dữ liệu từ người dùng
        System.out.println("Nhập số lượng phần tử bạn muốn thêm vào ArrayList:");
        int n = scanner.nextInt();

        System.out.println("Nhập " + n + " số nguyên:");
        for (int i = 0; i < n; i++) {
            int number = scanner.nextInt();
            numbers.add(number); // Thêm phần tử vào ArrayList
        }

        // Hiển thị các phần tử trong ArrayList
        System.out.println("Các phần tử trong ArrayList:");
        for (int number : numbers) {
            System.out.print(number + " ");
        }
        System.out.println();

        // Tìm kiếm một phần tử
        System.out.println("Nhập số bạn muốn tìm kiếm:");
        int searchNumber = scanner.nextInt();
        if (numbers.contains(searchNumber)) {
            System.out.println("Số " + searchNumber + " có trong ArrayList.");
        } else {
            System.out.println("Số " + searchNumber + " không có trong ArrayList.");
        }

        // Xóa một phần tử
        System.out.println("Nhập số bạn muốn xóa:");
        int removeNumber = scanner.nextInt();
        if (numbers.remove((Integer) removeNumber)) {
            System.out.println("Đã xóa số " + removeNumber + " khỏi ArrayList.");
        } else {
            System.out.println("Số " + removeNumber + " không có trong ArrayList để xóa.");
        }

        // Hiển thị các phần tử sau khi xóa
        System.out.println("Các phần tử trong ArrayList sau khi xóa:");
        for (int number : numbers) {
            System.out.print(number + " ");
        }
        System.out.println();

        scanner.close();
    }
}
```

### Giải thích:
- **Tạo ArrayList**: Sử dụng `ArrayList<Integer>` để lưu trữ các số nguyên.
- **Nhập dữ liệu**: Sử dụng `Scanner` để nhập dữ liệu từ người dùng.
- **Thêm phần tử**: Sử dụng `add()` để thêm phần tử vào `ArrayList`.
- **Hiển thị phần tử**: Sử dụng vòng lặp `for-each` để duyệt qua và in ra các phần tử.
- **Tìm kiếm phần tử**: Sử dụng `contains()` để kiểm tra sự tồn tại của một phần tử.
- **Xóa phần tử**: Sử dụng `remove()` để xóa phần tử khỏi `ArrayList`.

Mã này minh họa cách sử dụng cơ bản của `ArrayList` trong Java với các thao tác nhập, xuất, tìm kiếm và xóa.
