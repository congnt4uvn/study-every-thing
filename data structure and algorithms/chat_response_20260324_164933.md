Dưới đây là một ví dụ về danh sách liên kết vòng tròn (circular linked list) trong Java, bao gồm các chức năng chính như thêm, xóa và hiển thị các phần tử. Mã này cũng bao gồm phần nhập và xuất dữ liệu.

```java
class Node {
    int data;
    Node next;

    public Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class CircularLinkedList {
    private Node last;

    public CircularLinkedList() {
        this.last = null;
    }

    // Thêm phần tử vào cuối danh sách
    public void addToEnd(int data) {
        Node newNode = new Node(data);
        if (last == null) {
            last = newNode;
            last.next = last;
        } else {
            newNode.next = last.next;
            last.next = newNode;
            last = newNode;
        }
    }

    // Xóa phần tử đầu tiên
    public void deleteFirst() {
        if (last == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }

        if (last.next == last) {
            last = null;
        } else {
            last.next = last.next.next;
        }
    }

    // Hiển thị danh sách
    public void display() {
        if (last == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }

        Node first = last.next;
        Node temp = first;
        do {
            System.out.print(temp.data + " ");
            temp = temp.next;
        } while (temp != first);
        System.out.println();
    }
}

public class Main {
    public static void main(String[] args) {
        CircularLinkedList list = new CircularLinkedList();

        // Thêm phần tử vào danh sách
        list.addToEnd(10);
        list.addToEnd(20);
        list.addToEnd(30);

        // Hiển thị danh sách
        System.out.println("Danh sách sau khi thêm các phần tử:");
        list.display();

        // Xóa phần tử đầu tiên
        list.deleteFirst();

        // Hiển thị danh sách sau khi xóa
        System.out.println("Danh sách sau khi xóa phần tử đầu tiên:");
        list.display();
    }
}
```

### Giải thích:
- **Node**: Lớp này đại diện cho một nút trong danh sách liên kết, chứa dữ liệu và con trỏ đến nút tiếp theo.
- **CircularLinkedList**: Lớp này quản lý danh sách liên kết vòng tròn, với các phương thức để thêm, xóa và hiển thị các phần tử.
- **addToEnd**: Thêm một phần tử vào cuối danh sách.
- **deleteFirst**: Xóa phần tử đầu tiên trong danh sách.
- **display**: Hiển thị tất cả các phần tử trong danh sách.
- **Main**: Lớp này chứa phương thức `main` để chạy chương trình, thêm và xóa các phần tử, và hiển thị danh sách.

Chạy chương trình này sẽ cho bạn thấy cách hoạt động của danh sách liên kết vòng tròn với các thao tác cơ bản.
