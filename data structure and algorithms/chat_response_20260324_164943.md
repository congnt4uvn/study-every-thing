Dưới đây là một ví dụ về mã Java sử dụng cấu trúc dữ liệu ngăn xếp (stack) và bao gồm các chức năng chính như nhập dữ liệu, xử lý và xuất dữ liệu. Mã này sẽ minh họa cách sử dụng ngăn xếp để đảo ngược một chuỗi ký tự.

```java
import java.util.Scanner;
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        // Tạo một đối tượng Scanner để nhập dữ liệu từ bàn phím
        Scanner scanner = new Scanner(System.in);

        // Yêu cầu người dùng nhập một chuỗi
        System.out.print("Nhập một chuỗi: ");
        String input = scanner.nextLine();

        // Tạo một ngăn xếp để lưu trữ các ký tự
        Stack<Character> stack = new Stack<>();

        // Đẩy từng ký tự của chuỗi vào ngăn xếp
        for (char ch : input.toCharArray()) {
            stack.push(ch);
        }

        // Tạo một StringBuilder để xây dựng chuỗi đảo ngược
        StringBuilder reversed = new StringBuilder();

        // Lấy từng ký tự từ ngăn xếp và thêm vào StringBuilder
        while (!stack.isEmpty()) {
            reversed.append(stack.pop());
        }

        // Xuất chuỗi đảo ngược
        System.out.println("Chuỗi đảo ngược: " + reversed.toString());

        // Đóng Scanner
        scanner.close();
    }
}
```

### Giải thích:
- **Nhập dữ liệu**: Sử dụng `Scanner` để nhập một chuỗi từ người dùng.
- **Xử lý dữ liệu**: Sử dụng ngăn xếp để đảo ngược chuỗi. Mỗi ký tự của chuỗi được đẩy vào ngăn xếp, sau đó được lấy ra theo thứ tự ngược lại.
- **Xuất dữ liệu**: In ra chuỗi đã được đảo ngược.

Chương trình này minh họa cách sử dụng ngăn xếp để thực hiện một thao tác đơn giản là đảo ngược chuỗi, đồng thời bao gồm các bước nhập và xuất dữ liệu cơ bản.
