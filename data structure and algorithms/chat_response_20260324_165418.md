```java
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class QueueExample {

    public static void main(String[] args) {
        // Create a queue using LinkedList
        Queue<Integer> queue = new LinkedList<>();

        // Scanner for user input
        Scanner scanner = new Scanner(System.in);

        // Display menu
        while (true) {
            System.out.println("\nQueue Operations:");
            System.out.println("1. Enqueue");
            System.out.println("2. Dequeue");
            System.out.println("3. Display Queue");
            System.out.println("4. Exit");
            System.out.print("Choose an option: ");

            int choice = scanner.nextInt();

            switch (choice) {
                case 1:
                    System.out.print("Enter a number to enqueue: ");
                    int number = scanner.nextInt();
                    queue.add(number);
                    System.out.println(number + " enqueued to the queue.");
                    break;
                case 2:
                    if (queue.isEmpty()) {
                        System.out.println("Queue is empty. Cannot dequeue.");
                    } else {
                        int dequeued = queue.poll();
                        System.out.println(dequeued + " dequeued from the queue.");
                    }
                    break;
                case 3:
                    System.out.println("Current Queue: " + queue);
                    break;
                case 4:
                    System.out.println("Exiting...");
                    scanner.close();
                    return;
                default:
                    System.out.println("Invalid choice. Please try again.");
            }
        }
    }
}
```

### Explanation:

1. **Queue Implementation**: The program uses Java's `LinkedList` class to implement a queue. The `Queue` interface is used to define the queue operations.

2. **Main Function**: The `main` method contains a loop that displays a menu for the user to perform different queue operations.

3. **Operations**:
   - **Enqueue**: Adds an element to the end of the queue using the `add` method.
   - **Dequeue**: Removes an element from the front of the queue using the `poll` method. It also checks if the queue is empty before attempting to dequeue.
   - **Display Queue**: Prints the current elements in the queue.

4. **User Input**: The program uses a `Scanner` to read user input for performing operations on the queue.

5. **Exit**: The program exits when the user selects the option to exit.

This program provides a basic demonstration of queue operations and can be expanded with additional features or error handling as needed.
