Dưới đây là một ví dụ về danh sách liên kết đôi (doubly linked list) trong Java, bao gồm các chức năng chính như thêm, xóa, và hiển thị các phần tử. Mã này cũng bao gồm một ví dụ về cách nhập và xuất dữ liệu.

```java
class Node {
    int data;
    Node prev;
    Node next;

    public Node(int data) {
        this.data = data;
    }
}

class DoublyLinkedList {
    private Node head;
    private Node tail;

    // Thêm một phần tử vào cuối danh sách
    public void addLast(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = tail = newNode;
        } else {
            tail.next = newNode;
            newNode.prev = tail;
            tail = newNode;
        }
    }

    // Thêm một phần tử vào đầu danh sách
    public void addFirst(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = tail = newNode;
        } else {
            newNode.next = head;
            head.prev = newNode;
            head = newNode;
        }
    }

    // Xóa một phần tử từ đầu danh sách
    public void removeFirst() {
        if (head == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }
        if (head == tail) {
            head = tail = null;
        } else {
            head = head.next;
            head.prev = null;
        }
    }

    // Xóa một phần tử từ cuối danh sách
    public void removeLast() {
        if (tail == null) {
            System.out.println("Danh sách rỗng.");
            return;
        }
        if (head == tail) {
            head = tail = null;
        } else {
            tail = tail.prev;
            tail.next = null;
        }
    }

    // Hiển thị danh sách từ đầu đến cuối
    public void displayForward() {
        Node current = head;
        while (current != null) {
            System.out.print(current.data + " ");
            current = current.next;
        }
        System.out.println();
    }

    // Hiển thị danh sách từ cuối đến đầu
    public void displayBackward() {
        Node current = tail;
        while (current != null) {
            System.out.print(current.data + " ");
            current = current.prev;
        }
        System.out.println();
    }
}

public class Main {
    public static void main(String[] args) {
        DoublyLinkedList dll = new DoublyLinkedList();

        // Thêm phần tử vào danh sách
        dll.addLast(10);
        dll.addLast(20);
        dll.addFirst(5);

        // Hiển thị danh sách
        System.out.println("Danh sách từ đầu đến cuối:");
        dll.displayForward();

        System.out.println("Danh sách từ cuối đến đầu:");
        dll.displayBackward();

        // Xóa phần tử
        dll.removeFirst();
        System.out.println("Sau khi xóa phần tử đầu:");
        dll.displayForward();

        dll.removeLast();
        System.out.println("Sau khi xóa phần tử cuối:");
        dll.displayForward();
    }
}
```

### Giải thích:
- **Node**: Lớp này đại diện cho một nút trong danh sách liên kết đôi, chứa dữ liệu và hai con trỏ `prev` và `next`.
- **DoublyLinkedList**: Lớp này chứa các phương thức để thao tác với danh sách liên kết đôi như thêm, xóa, và hiển thị các phần tử.
- **Main**: Lớp này chứa phương thức `main` để chạy chương trình, thực hiện các thao tác trên danh sách và hiển thị kết quả.
