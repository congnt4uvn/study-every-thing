Dưới đây là một ví dụ về mã Java cho thuật toán sắp xếp Merge Sort, bao gồm tất cả các hàm chính, cấu trúc dữ liệu và thuật toán, cũng như phần nhập và xuất dữ liệu. Mã này được chú thích bằng tiếng Việt để bạn có thể dễ dàng hiểu cách hoạt động của nó.

```java
import java.util.Arrays;
import java.util.Scanner;

public class MergeSortExample {

    // Hàm để hợp nhất hai mảng con
    public static void merge(int[] array, int left, int middle, int right) {
        // Tính kích thước của hai mảng con
        int n1 = middle - left + 1;
        int n2 = right - middle;

        // Tạo mảng tạm thời
        int[] leftArray = new int[n1];
        int[] rightArray = new int[n2];

        // Sao chép dữ liệu vào mảng tạm thời
        for (int i = 0; i < n1; i++) {
            leftArray[i] = array[left + i];
        }
        for (int j = 0; j < n2; j++) {
            rightArray[j] = array[middle + 1 + j];
        }

        // Hợp nhất các mảng tạm thời
        int i = 0, j = 0;
        int k = left;
        while (i < n1 && j < n2) {
            if (leftArray[i] <= rightArray[j]) {
                array[k] = leftArray[i];
                i++;
            } else {
                array[k] = rightArray[j];
                j++;
            }
            k++;
        }

        // Sao chép các phần tử còn lại của leftArray nếu có
        while (i < n1) {
            array[k] = leftArray[i];
            i++;
            k++;
        }

        // Sao chép các phần tử còn lại của rightArray nếu có
        while (j < n2) {
            array[k] = rightArray[j];
            j++;
            k++;
        }
    }

    // Hàm sắp xếp Merge Sort
    public static void mergeSort(int[] array, int left, int right) {
        if (left < right) {
            // Tìm điểm giữa
            int middle = (left + right) / 2;

            // Sắp xếp nửa đầu và nửa sau
            mergeSort(array, left, middle);
            mergeSort(array, middle + 1, right);

            // Hợp nhất hai nửa đã sắp xếp
            merge(array, left, middle, right);
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Nhập dữ liệu từ người dùng
        System.out.print("Nhập số lượng phần tử: ");
        int n = scanner.nextInt();
        int[] array = new int[n];

        System.out.println("Nhập các phần tử:");
        for (int i = 0; i < n; i++) {
            array[i] = scanner.nextInt();
        }

        // Gọi hàm sắp xếp Merge Sort
        mergeSort(array, 0, array.length - 1);

        // Xuất mảng đã sắp xếp
        System.out.println("Mảng sau khi sắp xếp:");
        System.out.println(Arrays.toString(array));

        scanner.close();
    }
}
```

### Giải thích:
- **merge**: Hàm này hợp nhất hai mảng con đã được sắp xếp thành một mảng duy nhất.
- **mergeSort**: Hàm này chia mảng thành hai nửa, sắp xếp từng nửa và sau đó hợp nhất chúng lại.
- **main**: Hàm chính để nhập dữ liệu từ người dùng, gọi hàm sắp xếp và xuất kết quả.
