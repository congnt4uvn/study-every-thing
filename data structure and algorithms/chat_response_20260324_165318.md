Dưới đây là một ví dụ về cách triển khai thuật toán KMP (Knuth-Morris-Pratt) trong Java để tìm kiếm chuỗi con một cách hiệu quả. Mã này bao gồm tất cả các hàm chính, cấu trúc dữ liệu và thuật toán cần thiết, cùng với phần nhập và xuất dữ liệu.

```java
public class KMPAlgorithm {

    // Hàm để tính toán mảng LPS (Longest Prefix Suffix)
    private static int[] computeLPSArray(String pattern) {
        int m = pattern.length();
        int[] lps = new int[m];
        int len = 0;
        int i = 1;

        lps[0] = 0; // LPS của ký tự đầu tiên luôn là 0

        while (i < m) {
            if (pattern.charAt(i) == pattern.charAt(len)) {
                len++;
                lps[i] = len;
                i++;
            } else {
                if (len != 0) {
                    len = lps[len - 1];
                } else {
                    lps[i] = 0;
                    i++;
                }
            }
        }
        return lps;
    }

    // Hàm KMP để tìm kiếm chuỗi con
    public static void KMPSearch(String text, String pattern) {
        int n = text.length();
        int m = pattern.length();

        int[] lps = computeLPSArray(pattern);

        int i = 0; // chỉ số cho text
        int j = 0; // chỉ số cho pattern

        while (i < n) {
            if (pattern.charAt(j) == text.charAt(i)) {
                i++;
                j++;
            }

            if (j == m) {
                System.out.println("Chuỗi con được tìm thấy tại chỉ số " + (i - j));
                j = lps[j - 1];
            } else if (i < n && pattern.charAt(j) != text.charAt(i)) {
                if (j != 0) {
                    j = lps[j - 1];
                } else {
                    i++;
                }
            }
        }
    }

    public static void main(String[] args) {
        String text = "ABABDABACDABABCABAB";
        String pattern = "ABABCABAB";

        System.out.println("Văn bản: " + text);
        System.out.println("Mẫu: " + pattern);

        KMPSearch(text, pattern);
    }
}
```

### Giải thích:

- **computeLPSArray**: Hàm này tính toán mảng LPS, giúp xác định độ dài của tiền tố dài nhất của mẫu cũng là hậu tố. Mảng này được sử dụng để tránh việc so sánh lại các ký tự đã biết là trùng khớp.

- **KMPSearch**: Hàm này thực hiện tìm kiếm chuỗi con bằng cách sử dụng mảng LPS để tối ưu hóa quá trình so sánh.

- **main**: Hàm chính để chạy chương trình, nơi bạn có thể thay đổi `text` và `pattern` để kiểm tra các trường hợp khác nhau.

### Đầu vào và đầu ra:

- **Đầu vào**: Chuỗi `text` và chuỗi `pattern` mà bạn muốn tìm kiếm.
- **Đầu ra**: Chỉ số bắt đầu của mỗi lần xuất hiện của `pattern` trong `text`.
