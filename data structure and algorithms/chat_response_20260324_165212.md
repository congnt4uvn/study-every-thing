Dưới đây là một ví dụ về cách triển khai thuật toán Prim để tìm cây khung nhỏ nhất (Minimum Spanning Tree - MST) bằng Java. Chương trình này bao gồm các cấu trúc dữ liệu và hàm chính cần thiết để thực hiện thuật toán. Đầu vào là một ma trận trọng số của đồ thị, và đầu ra là cây khung nhỏ nhất.

```java
import java.util.Arrays;

public class PrimAlgorithm {
    private static final int V = 5; // Số lượng đỉnh trong đồ thị

    // Hàm tìm đỉnh có trọng số nhỏ nhất từ tập hợp các đỉnh chưa được bao gồm trong MST
    int minKey(int[] key, boolean[] mstSet) {
        int min = Integer.MAX_VALUE, minIndex = -1;

        for (int v = 0; v < V; v++) {
            if (!mstSet[v] && key[v] < min) {
                min = key[v];
                minIndex = v;
            }
        }
        return minIndex;
    }

    // Hàm in ra cây khung nhỏ nhất được lưu trữ trong parent[]
    void printMST(int[] parent, int[][] graph) {
        System.out.println("Cạnh \tTrọng số");
        for (int i = 1; i < V; i++) {
            System.out.println(parent[i] + " - " + i + "\t" + graph[i][parent[i]]);
        }
    }

    // Hàm thực hiện thuật toán Prim để tìm cây khung nhỏ nhất
    void primMST(int[][] graph) {
        int[] parent = new int[V]; // Mảng lưu trữ cây khung
        int[] key = new int[V]; // Giá trị trọng số nhỏ nhất cho mỗi đỉnh
        boolean[] mstSet = new boolean[V]; // Đỉnh đã được bao gồm trong MST

        // Khởi tạo tất cả các giá trị key là vô cùng
        Arrays.fill(key, Integer.MAX_VALUE);
        key[0] = 0; // Đỉnh đầu tiên được chọn
        parent[0] = -1; // Đỉnh đầu tiên là gốc của MST

        for (int count = 0; count < V - 1; count++) {
            int u = minKey(key, mstSet);
            mstSet[u] = true;

            for (int v = 0; v < V; v++) {
                if (graph[u][v] != 0 && !mstSet[v] && graph[u][v] < key[v]) {
                    parent[v] = u;
                    key[v] = graph[u][v];
                }
            }
        }

        printMST(parent, graph);
    }

    public static void main(String[] args) {
        PrimAlgorithm t = new PrimAlgorithm();
        int[][] graph = {
            {0, 2, 0, 6, 0},
            {2, 0, 3, 8, 5},
            {0, 3, 0, 0, 7},
            {6, 8, 0, 0, 9},
            {0, 5, 7, 9, 0}
        };

        t.primMST(graph);
    }
}
```

### Giải thích:
- **minKey**: Hàm này tìm đỉnh có giá trị key nhỏ nhất từ tập hợp các đỉnh chưa được bao gồm trong MST.
- **printMST**: Hàm này in ra các cạnh của cây khung nhỏ nhất cùng với trọng số của chúng.
- **primMST**: Hàm chính thực hiện thuật toán Prim. Nó sử dụng mảng `key` để theo dõi trọng số nhỏ nhất của cạnh kết nối mỗi đỉnh với cây khung. Mảng `parent` lưu trữ cây khung.
- **main**: Hàm này khởi tạo một đồ thị dưới dạng ma trận trọng số và gọi hàm `primMST` để tìm cây khung nhỏ nhất.

Chương trình này sẽ in ra các cạnh của cây khung nhỏ nhất và trọng số tương ứng của chúng.
