Dưới đây là một ví dụ về cách triển khai hàng đợi (queue) trong Java, bao gồm các chức năng chính của cấu trúc dữ liệu và thuật toán, cùng với đầu vào và đầu ra. Mã này sẽ được chú thích bằng tiếng Việt để bạn dễ hiểu hơn.

```java
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class QueueExample {
    public static void main(String[] args) {
        // Tạo một hàng đợi sử dụng LinkedList
        Queue<Integer> queue = new LinkedList<>();

        // Sử dụng Scanner để nhận đầu vào từ người dùng
        Scanner scanner = new Scanner(System.in);

        // Thêm phần tử vào hàng đợi
        System.out.println("Nhập số lượng phần tử bạn muốn thêm vào hàng đợi:");
        int n = scanner.nextInt();
        System.out.println("Nhập các phần tử:");
        for (int i = 0; i < n; i++) {
            int element = scanner.nextInt();
            queue.add(element); // Thêm phần tử vào cuối hàng đợi
        }

        // Hiển thị các phần tử trong hàng đợi
        System.out.println("Các phần tử trong hàng đợi:");
        for (int elem : queue) {
            System.out.print(elem + " ");
        }
        System.out.println();

        // Xóa phần tử khỏi hàng đợi
        System.out.println("Xóa phần tử khỏi hàng đợi:");
        if (!queue.isEmpty()) {
            int removedElement = queue.poll(); // Lấy và xóa phần tử đầu tiên
            System.out.println("Phần tử đã xóa: " + removedElement);
        } else {
            System.out.println("Hàng đợi rỗng, không thể xóa phần tử.");
        }

        // Hiển thị phần tử đầu tiên của hàng đợi
        if (!queue.isEmpty()) {
            int head = queue.peek(); // Lấy phần tử đầu tiên mà không xóa
            System.out.println("Phần tử đầu tiên trong hàng đợi: " + head);
        } else {
            System.out.println("Hàng đợi rỗng.");
        }

        // Đóng Scanner
        scanner.close();
    }
}
```

### Giải thích:
- **Queue Interface**: Sử dụng `Queue` interface và `LinkedList` để triển khai hàng đợi.
- **Thêm phần tử**: Sử dụng `add()` để thêm phần tử vào cuối hàng đợi.
- **Xóa phần tử**: Sử dụng `poll()` để lấy và xóa phần tử đầu tiên của hàng đợi.
- **Xem phần tử đầu tiên**: Sử dụng `peek()` để xem phần tử đầu tiên mà không xóa nó.
- **Đầu vào/đầu ra**: Sử dụng `Scanner` để nhận đầu vào từ người dùng và in kết quả ra màn hình.

Chạy chương trình này sẽ cho phép bạn thêm các phần tử vào hàng đợi, hiển thị chúng, xóa phần tử đầu tiên và xem phần tử đầu tiên hiện tại trong hàng đợi.
