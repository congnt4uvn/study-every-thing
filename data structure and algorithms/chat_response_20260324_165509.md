To represent a graph in Java using both an adjacency matrix and an adjacency list (using a hash table), you can create a class for each representation. Below is a sample code that includes the main functions for both data structures, along with input and output examples.

### Adjacency Matrix Representation

```java
import java.util.Scanner;

class GraphAdjMatrix {
    private int[][] adjMatrix;
    private int numVertices;

    public GraphAdjMatrix(int numVertices) {
        this.numVertices = numVertices;
        adjMatrix = new int[numVertices][numVertices];
    }

    public void addEdge(int i, int j) {
        adjMatrix[i][j] = 1;
        adjMatrix[j][i] = 1; // For undirected graph
    }

    public void printMatrix() {
        System.out.println("Adjacency Matrix:");
        for (int i = 0; i < numVertices; i++) {
            for (int j = 0; j < numVertices; j++) {
                System.out.print(adjMatrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```

### Adjacency List Representation using Hash Table

```java
import java.util.HashMap;
import java.util.LinkedList;
import java.util.List;
import java.util.Map;

class GraphAdjList {
    private Map<Integer, List<Integer>> adjList;

    public GraphAdjList() {
        adjList = new HashMap<>();
    }

    public void addEdge(int i, int j) {
        adjList.computeIfAbsent(i, k -> new LinkedList<>()).add(j);
        adjList.computeIfAbsent(j, k -> new LinkedList<>()).add(i); // For undirected graph
    }

    public void printList() {
        System.out.println("Adjacency List:");
        for (Map.Entry<Integer, List<Integer>> entry : adjList.entrySet()) {
            System.out.print(entry.getKey() + ": ");
            for (Integer vertex : entry.getValue()) {
                System.out.print(vertex + " ");
            }
            System.out.println();
        }
    }
}
```

### Main Function with Input and Output

```java
public class GraphExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter number of vertices: ");
        int numVertices = scanner.nextInt();

        GraphAdjMatrix graphMatrix = new GraphAdjMatrix(numVertices);
        GraphAdjList graphList = new GraphAdjList();

        System.out.print("Enter number of edges: ");
        int numEdges = scanner.nextInt();

        System.out.println("Enter edges (format: u v):");
        for (int i = 0; i < numEdges; i++) {
            int u = scanner.nextInt();
            int v = scanner.nextInt();
            graphMatrix.addEdge(u, v);
            graphList.addEdge(u, v);
        }

        graphMatrix.printMatrix();
        graphList.printList();

        scanner.close();
    }
}
```

### How to Use

1. Compile the code using a Java compiler.
2. Run the program.
3. Input the number of vertices and edges.
4. Enter each edge as a pair of integers (e.g., `0 1` for an edge between vertex 0 and vertex 1).
5. The program will output both the adjacency matrix and the adjacency list representation of the graph.
