


Nội dung trên nói về AWS CDK (Cloud Development Kit) và so sánh với CloudFormation và SAM:

1. **AWS CDK là gì?**  
   - Là bộ công cụ cho phép định nghĩa hạ tầng AWS bằng ngôn ngữ lập trình quen thuộc: TypeScript, JavaScript, Python, Java, .NET.  
   - Bạn viết code (các “construct” cấp cao như VPC, ECS Cluster, Fargate Service…) thay vì viết YAML/JSON.

2. **Cách hoạt động với CloudFormation**  
   - CDK **không thay thế** mà **xây trên** CloudFormation.  
   - Code CDK được biên dịch (cdk synth) thành **template CloudFormation** (JSON/YAML).  
   - Nếu code không biên dịch hoặc sai kiểu, bạn biết lỗi sớm; nếu hợp lệ, nó sinh ra template CloudFormation để triển khai.

3. **Ưu điểm của CDK**  
   - Dùng ngôn ngữ lập trình nên:
     - Có kiểm tra kiểu, IDE hỗ trợ, tái sử dụng, vòng lặp, hàm, module…  
     - Dễ kết hợp triển khai **hạ tầng** và **mã ứng dụng** (Lambda, container ECS/EKS) cùng nhau.  
   - Các construct cấp cao giúp tạo nhanh VPC, ECS Cluster, Fargate Service, ALB… với cấu hình mặc định hợp lý.

4. **So sánh CDK và SAM**  
   - **SAM**:
     - Tập trung vào **serverless** (Lambda, API Gateway, DynamoDB, v.v.).  
     - Viết template **khai báo** bằng YAML/JSON.  
     - Cũng dùng CloudFormation ở backend.  
     - Phù hợp để bắt đầu nhanh với Lambda.  
   - **CDK**:
     - Là “siêu bộ” của CloudFormation, hỗ trợ **mọi dịch vụ AWS**.  
     - Định nghĩa hạ tầng bằng **ngôn ngữ lập trình** (TS, Python, Java, .NET).  
     - Cũng sinh ra CloudFormation ở backend.

5. **Kết hợp CDK và SAM**  
   - Có thể dùng **SAM CLI** để test cục bộ ứng dụng viết bằng CDK:
     - Chạy `cdk synth` để tạo template CloudFormation.  
     - Dùng `sam local` tham chiếu template đó để gọi thử các hàm Lambda.

6. **Bài thực hành tiếp theo (demo kiến trúc)**  
   - Sẽ triển khai bằng CDK một ứng dụng gồm:
     - S3 bucket cho người dùng upload ảnh.  
     - S3 trigger một Lambda.  
     - Lambda gọi Amazon Rekognition để phân tích ảnh.  
     - Lambda lưu kết quả vào DynamoDB.  
   - Toàn bộ hạ tầng trên được định nghĩa trong một script CDK.
