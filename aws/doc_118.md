


Người hướng dẫn đang minh họa cách dùng Amazon S3 để lưu trữ một website tĩnh:

1. Trong S3 bucket, đã có 2 file (trong đó có file ảnh coffee.jpg, beach.jpg).
2. Vào phần Properties của bucket, kéo xuống mục “Static website hosting”.
3. Nhấn Edit, bật “Static website hosting”, chọn kiểu “Host a static website”.
4. Nhập tên file chỉ mục là `index.html` (trang chủ mặc định của website).
5. Lưu cấu hình, sau đó quay lại tab Objects và upload file `index.html` lên bucket.
6. Do bucket đã được thiết lập public (bài giảng trước), tất cả nội dung có thể được truy cập công khai.
7. Quay lại Properties, kéo xuống sẽ thấy “Bucket website endpoint” – đây là URL của website tĩnh.
8. Mở URL này trong trình duyệt sẽ thấy nội dung trang `index.html` (ví dụ: “I like coffee. Hello world!”) và ảnh coffee.jpg.
9. Có thể mở trực tiếp ảnh coffee.jpg hoặc beach.jpg bằng URL công khai của chúng.
10. Kết luận: Bucket S3 đã được cấu hình thành công để lưu trữ website tĩnh, và nhờ chính sách public, mọi file trong bucket có thể được truy cập qua web.
