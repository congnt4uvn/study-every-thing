


Dưới đây là tóm tắt nội dung:

1. Khái niệm SSH  
   - SSH là giao thức cho phép điều khiển máy/ server từ xa qua terminal (dòng lệnh).  
   - Trong ví dụ, dùng SSH để truy cập vào EC2 instance (Amazon Linux 2) qua Internet.

2. Điều kiện để SSH vào EC2  
   - EC2 instance phải có Public IPv4.  
   - Security Group phải mở cổng 22 (SSH) cho IP của bạn (trong ví dụ là 0.0.0.0/0 – từ mọi nơi).  
   - Có file key pair `.pem` đã tải về khi tạo EC2 (ví dụ: `EC2Tutorial.pem`, không được có khoảng trắng trong tên).

3. Chuẩn bị trên máy tính (Linux/Mac)  
   - Đặt file `EC2Tutorial.pem` vào một thư mục, ví dụ: `aws-course`.  
   - Dùng các lệnh Linux cơ bản để vào đúng thư mục chứa file:
     - `pwd` – xem đang ở thư mục nào.  
     - `ls` hoặc `ll` – liệt kê file/thư mục.  
     - `cd <tên_thư_mục>` – di chuyển vào thư mục (ví dụ `cd aws-course`).  

4. Câu lệnh SSH cơ bản  
   - Ban đầu thử:  
     `ssh ec2-user@<Public_IP>`  
     → bị lỗi vì chưa chỉ định file key `.pem`.  
   - Câu lệnh đúng:  
     `ssh -i EC2Tutorial.pem ec2-user@<Public_IP>`  
   - Nếu file key “không được bảo vệ” (permission quá rộng), sẽ có lỗi.

5. Sửa quyền file key  
   - Dùng lệnh:  
     `chmod 0400 EC2Tutorial.pem`  
   - Sau đó chạy lại:  
     `ssh -i EC2Tutorial.pem ec2-user@<Public_IP>`  
   - Lần đầu có thể được hỏi “yes/no” để tin cậy host → nhập `yes`.

6. Sau khi SSH thành công  
   - Prompt sẽ giống: `ec2-user@<IP> ...` → nghĩa là bạn đang ở bên trong EC2.  
   - Có thể chạy lệnh ví dụ:
     - `whoami` → trả về `ec2-user`.  
     - `ping google.com` → kiểm tra kết nối mạng.  
   - Dừng lệnh đang chạy: `Ctrl + C`.  
   - Thoát khỏi EC2: gõ `exit` hoặc đóng terminal.

7. Lưu ý về IP công khai  
   - Nếu bạn “Stop” rồi “Start” lại EC2 instance, Public IP có thể thay đổi.  
   - Khi đó, phải dùng IP mới trong lệnh SSH:  
     `ssh -i EC2Tutorial.pem ec2-user@<New_Public_IP>`

Nội dung bài giảng: hướng dẫn chi tiết cách dùng SSH từ Linux/Mac để kết nối vào EC2 (Amazon Linux 2) bằng key `.pem`, cách xử lý lỗi thường gặp (chưa dùng key, quyền file key sai), và một số lệnh cơ bản sau khi đăng nhập.
