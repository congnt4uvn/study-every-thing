


Người hướng dẫn đang hướng dẫn cách cài đặt AWS CLI v2 trên Linux:

1. Vào Google, tìm “Install AWS CLI version 2 on Linux” và mở trang hướng dẫn chính thức.
2. Cuộn xuống phần cài đặt và chạy lần lượt 3 lệnh:
   - Tải file ZIP trình cài đặt AWS CLI.
   - Giải nén file ZIP vừa tải.
   - Chạy trình cài đặt với quyền root (dùng `sudo`), nhập mật khẩu khi được yêu cầu.
3. Sau khi cài xong, có thể kiểm tra bằng lệnh:
   - `aws --version` (nếu `~/.local/bin` đã nằm trong PATH), hoặc
   - `/usr/local/bin/aws --version`.
4. Kết quả hiển thị phiên bản AWS CLI 2, kèm phiên bản Python, Linux và Botocore → xác nhận cài đặt thành công.
5. Khi AWS CLI đã hoạt động, có thể chạy bất kỳ lệnh AWS CLI nào và tiếp tục các bài học sau. Nếu gặp lỗi, nên đọc phần tài liệu/troubleshooting mà người hướng dẫn nhắc đến.
