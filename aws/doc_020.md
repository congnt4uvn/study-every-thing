


Nội dung trên hướng dẫn cách cài đặt AWS CLI phiên bản 2 trên macOS:

1. Mở Google, tìm và chọn liên kết cài đặt AWS CLI v2 cho macOS.
2. Tải xuống file cài đặt dạng `.pkg`.
3. Mở file `.pkg`, làm theo trình cài đặt đồ họa:
   - Nhấn “Tiếp tục” nhiều lần.
   - Đồng ý điều khoản.
   - Chọn cài đặt cho tất cả người dùng trên máy.
   - Nhấn “Cài đặt” và chờ quá trình hoàn tất.
4. Sau khi cài xong, có thể chuyển file cài đặt vào thùng rác.
5. Mở Terminal (hoặc iTerm).
6. Gõ lệnh `aws --version`:
   - Nếu hiện phiên bản, ví dụ `aws-cli/2.0.10`, nghĩa là cài đặt thành công.
7. Nếu gặp vấn đề, xem lại tài liệu/hướng dẫn kèm theo để tìm câu trả lời.
