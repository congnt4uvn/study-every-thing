


Dưới đây là tóm tắt nội dung:

1. Mục tiêu  
   - Hướng dẫn cách SSH vào EC2 instance (Amazon Linux 2) từ máy Windows (Windows 7/8, cũng dùng được cho Windows 10) bằng PuTTY.

2. Khái niệm SSH  
   - SSH cho phép điều khiển máy từ xa qua dòng lệnh.  
   - EC2 instance có IP public và security group đã mở cổng 22 (SSH) cho phép máy Windows kết nối.

3. Cài đặt PuTTY và PuTTYgen  
   - Tải PuTTY (client SSH miễn phí cho Windows) từ internet.  
   - Cài đặt PuTTY, sau đó có 2 công cụ: PuTTY và PuTTYgen.

4. Chuyển file PEM sang PPK bằng PuTTYgen  
   - Mở PuTTYgen.  
   - Nhấn “Load”, chọn hiển thị “All files” để thấy file .pem.  
   - Chọn file .pem (key tải từ AWS).  
   - Sau khi import thành công, nhấn “Save private key” để lưu thành file .ppk (ví dụ: EC2tutorial.ppk).  
   - Đây là file key riêng dùng cho PuTTY.

5. Cấu hình PuTTY để SSH vào EC2  
   - Mở PuTTY.  
   - Ở mục “Host Name”, nhập:  
     `ec2-user@<IPv4 Public Address>`  
     (ec2-user là user mặc định của Amazon Linux 2, IP là IPv4 public của instance).  
   - Lưu cấu hình này với tên, ví dụ “EC2 Instance” (Save).

6. Gán private key trong PuTTY  
   - Trong PuTTY, chọn cấu hình đã lưu, nhấn “Load”.  
   - Vào mục SSH → Auth.  
   - Chọn “Browse” và chọn file .ppk vừa tạo (EC2tutorial.ppk).  
   - Quay lại mục Session, nhấn “Save” để lưu lại toàn bộ cấu hình (host + key).

7. Kết nối SSH  
   - Nhấn “Open” để kết nối.  
   - Lần đầu sẽ hiện cảnh báo về host key, chọn “Yes” để chấp nhận.  
   - PuTTY sẽ dùng user ec2-user và file .ppk để xác thực.  
   - Khi đăng nhập thành công, bạn đang ở trong Amazon Linux 2 AMI.

8. Thao tác cơ bản sau khi SSH  
   - Có thể chạy lệnh `whoami` để kiểm tra user (ec2-user).  
   - Có thể thử lệnh như `ping google.com`.  
   - Dừng lệnh bằng Ctrl + C.  
   - Thoát phiên bằng cách đóng cửa sổ PuTTY.

9. Lưu cấu hình để dùng lại  
   - Khi mở lại PuTTY, chỉ cần chọn cấu hình “EC2 Instance” và nhấn “Open” là có thể SSH ngay, không cần cấu hình lại.  

10. Ghi chú  
   - Bài giảng này dành cho Windows 7/8 (và có thể dùng cho Windows 10).  
   - Bài giảng tiếp theo sẽ hướng dẫn cách SSH trên Windows 10 bằng phương pháp khác.
