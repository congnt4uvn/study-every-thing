


Danh sách trên là một “checklist dọn dẹp” các dịch vụ AWS đã dùng, để sau khi học/lab xong bạn không quên xóa tài nguyên, tránh tốn chi phí.  

Các nhóm chính cần kiểm tra và xóa nếu không dùng nữa:

- **Compute & networking**:  
  - EC2 instances, Elastic Load Balancer, Launch Configurations, Auto Scaling Groups  
  - Elastic IPs, Security Groups, EBS Volumes, Key Pairs  
  - ECS (cluster, service, task definition, ECR)  
  - Elastic Beanstalk (applications & environments)  
  - AWS Lambda functions  

- **Storage & database**:  
  - S3 buckets & files  
  - RDS databases  
  - DynamoDB tables & indexes  
  - ElastiCache clusters  

- **Networking & integration**:  
  - Route 53  
  - API Gateway  
  - Kinesis Streams  
  - SNS Topics  
  - SQS Queues  

- **DevOps & monitoring**:  
  - CodeCommit, CodeBuild, CodePipeline  
  - CloudWatch Logs, Metrics, Dashboards, Alerts, Rules  
  - CloudFormation Stacks  
  - SSM Parameter Store  

- **Bảo mật & quyền truy cập**:  
  - IAM Users, Groups, Roles, Policies  

Cuối cùng, nội dung nhắc rằng đây là một khối lượng kiến thức lớn và chúc mừng bạn đã hoàn thành.
