


Dưới đây là tóm tắt nội dung về Amazon CodeGuru:

1. Tổng quan  
- Amazon CodeGuru là dịch vụ AI hỗ trợ hai chức năng chính:  
  1) Tự động kiểm tra mã nguồn (CodeGuru Reviewer)  
  2) Đề xuất tối ưu hiệu suất ứng dụng (CodeGuru Profiler)

2. CodeGuru Reviewer  
- Thực hiện phân tích mã nguồn tĩnh khi bạn đẩy code lên các kho như CodeCommit, GitHub, Bitbucket.  
- Tự động xem xét các commit, chỉ ra những dòng mã có vấn đề.  
- Phát hiện:  
  - Lỗi logic, bug khó thấy  
  - Rò rỉ tài nguyên/bộ nhớ  
  - Lỗ hổng bảo mật  
  - Vấn đề về kiểm tra dữ liệu đầu vào  
  - Vi phạm các best practice lập trình  
- Dùng machine learning và suy luận tự động, được “huấn luyện” từ:  
  - Hàng nghìn kho mã nguồn mở  
  - Kho mã nguồn nội bộ của Amazon  
- Hiện hỗ trợ Java và Python (có thể thay đổi theo thời gian).

3. CodeGuru Profiler  
- Dùng khi ứng dụng đang chạy ở môi trường tiền sản xuất hoặc sản xuất.  
- Phân tích hành vi thực thi của ứng dụng để:  
  - Xác định đoạn mã tiêu tốn nhiều CPU (ví dụ logging quá mức)  
  - Giảm sử dụng CPU, giảm chi phí tính toán  
  - Cung cấp tóm tắt heap memory để biết đối tượng nào chiếm nhiều bộ nhớ  
  - Phát hiện bất thường trong hoạt động ứng dụng  
- Hỗ trợ ứng dụng chạy trên AWS Cloud hoặc on-premises.  
- Tác động tài nguyên lên ứng dụng được giám sát là tối thiểu.

4. Ý chính cần nhớ  
- CodeGuru = Reviewer + Profiler.  
- Reviewer: kiểm tra mã tự động, tìm bug, bảo mật, best practice.  
- Profiler: tối ưu hiệu suất, chi phí, tài nguyên khi ứng dụng đang chạy.
