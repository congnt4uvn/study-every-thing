


Dưới đây là tóm tắt nội dung:

- Amazon ECR (Elastic Container Registry) là dịch vụ dùng để lưu trữ và quản lý Docker image trên AWS, tương tự Docker Hub nhưng thuộc hệ sinh thái AWS.  
- Có 2 loại kho:  
  - Private repository: chỉ tài khoản (hoặc các tài khoản) của bạn truy cập.  
  - Public repository: xuất bản image lên thư viện công cộng của Amazon ECR.  
- ECR tích hợp chặt chẽ với Amazon ECS; các Docker image trong ECR được lưu trữ trên Amazon S3.  
- Cụm ECS (ví dụ các EC2 instance trong cụm) sẽ “kéo” (pull) image từ ECR. Để làm được điều này, EC2 cần được gán IAM Role phù hợp.  
- Mọi truy cập vào ECR đều được kiểm soát bằng IAM; nếu gặp lỗi quyền, cần kiểm tra lại IAM policy.  
- ECR hỗ trợ thêm: quét lỗ hổng bảo mật của image, versioning, tagging và lifecycle policy cho image.  
- Kết luận: bất cứ khi nào cần lưu trữ Docker image trên AWS, hãy nghĩ đến Amazon ECR.
