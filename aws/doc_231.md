


Dưới đây là tóm tắt nội dung bài giảng:

- Amazon Managed Service for Apache Flink (trước đây là Kinesis Data Analytics for Apache Flink) là dịch vụ được quản lý để chạy các ứng dụng Apache Flink trên AWS.  
- Apache Flink là framework xử lý luồng dữ liệu thời gian thực, thường dùng Java, SQL hoặc Scala.  
- Dịch vụ này cho phép đọc dữ liệu từ:
  - Amazon Kinesis Data Streams  
  - Amazon MSK (Managed Streaming for Apache Kafka – dịch vụ quản lý cho Apache Kafka)  
- AWS chịu trách nhiệm:
  - Cung cấp tài nguyên compute  
  - Hỗ trợ xử lý song song, tự động mở rộng  
  - Quản lý backup ứng dụng thông qua checkpoint và snapshot  
- Bạn có thể dùng mọi tính năng lập trình mà Apache Flink hỗ trợ để chuyển đổi dữ liệu (rất linh hoạt về loại transformation).  
- Lưu ý quan trọng (dễ ra thi): Flink có thể đọc từ Kinesis Data Streams nhưng **không** đọc được từ Kinesis Data Firehose.  
- Dịch vụ này chỉ dùng cho **xử lý luồng dữ liệu** (stream processing), không phải batch.
