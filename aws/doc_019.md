


Nội dung trên hướng dẫn cách cài đặt AWS CLI phiên bản 2 trên Windows:

1. Trên Google, tìm “aws cli install windows” và vào trang chính thức của AWS.
2. Chọn phần cài đặt cho Windows, tải về trình cài đặt MSI của AWS CLI v2.
3. Chạy file MSI:
   - Nhấn Next.
   - Chấp nhận điều khoản (I accept the license terms).
   - Nhấn Next, rồi Install.
   - Chờ quá trình cài đặt hoàn tất, sau đó nhấn Finish.
4. Mở Command Prompt (Dấu nhắc lệnh) trên Windows.
5. Gõ lệnh:
   - `aws --version`
   - Nếu kết quả hiển thị aws-cli với số phiên bản bắt đầu bằng 2 (kèm Python, Windows), nghĩa là cài đặt thành công.
6. Để nâng cấp AWS CLI sau này, chỉ cần tải lại file MSI mới nhất và chạy lại, nó sẽ tự động nâng cấp.

Sau khi thấy kết quả `aws --version` đúng, bạn đã sẵn sàng dùng AWS CLI và tiếp tục theo dõi bài giảng tiếp theo.
