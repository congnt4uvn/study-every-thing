


Giảng viên minh họa cách sử dụng **AWS Lambda Layer** để chạy thư viện **Pandas** trong hàm Lambda:

1. Tạo một hàm Lambda mới (runtime Python, ví dụ 3.13), đặt tên `Lambda-layer-demo`.
2. Sao chép mã từ file `layer-demo-panda.py` (trong thư mục Lambda) vào phần code của hàm.  
   - Mã này: import Pandas, tạo dữ liệu mẫu, rồi lọc dữ liệu bằng Pandas (lọc các phần tử có tuổi > 30).
3. Triển khai (Deploy) hàm và chạy Test lần đầu:
   - Hàm bị lỗi vì **không tìm thấy module `pandas`** (Pandas chưa được đóng gói trong Lambda).
4. Để giải quyết:
   - Vào mục **Layers** của hàm Lambda → **Add a layer**.
   - Chọn **AWS layers** và chọn lớp **`AWSSDKPandas-Python313`** (hoặc phiên bản Pandas phù hợp).
   - Thêm layer này vào hàm.
5. Chạy lại Test:
   - Lần này hàm chạy thành công, trả về dữ liệu đã được lọc (các phần tử có tuổi > 30).
6. Kết luận:
   - Lambda Layer giúp đóng gói và tái sử dụng thư viện (như Pandas) mà không cần tự đóng gói thủ công.
   - Có thể áp dụng Layer cho nhiều thư viện và nhiều trường hợp sử dụng khác.
