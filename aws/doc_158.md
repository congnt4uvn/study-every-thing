


Dưới đây là tóm tắt nội dung:

1. **Hành vi mặc định & cache key**
   - Phân phối CloudFront có một *default behavior* (pattern `*`) không sửa được path pattern.
   - Trong behavior có thể gán:
     - **Cache policy** (bắt buộc).
     - **Origin request policy** (tùy chọn).
   - Cache policy cho phép:
     - Thiết lập TTL: min TTL, max TTL, default TTL.
     - Xác định cache key gồm những gì:
       - Header nào được đưa vào cache key.
       - Query string nào (tất cả hoặc danh sách cụ thể).
       - Cookie nào (tất cả hoặc danh sách cụ thể).
   - Những thành phần được chọn (header, query string, cookie) sẽ vừa ảnh hưởng cache key, vừa được gửi đến origin.

2. **Origin request policy**
   - Dùng để chỉ định thêm:
     - Header nào gửi đến origin.
     - Query string nào gửi đến origin.
     - Cookie nào gửi đến origin.
   - Mục đích: điều khiển chi tiết dữ liệu kèm theo khi CloudFront gọi về origin, ngoài những gì đã nằm trong cache key.

3. **Nhiều behavior**
   - Có thể tạo behavior mới, ví dụ cho `/images/*`, trỏ đến một origin khác (S3 bucket khác, EC2, v.v.).
   - Mỗi behavior có thể có cache policy và origin request policy riêng.
   - Khi nhiều behavior cùng tồn tại, pattern cụ thể hơn sẽ được ưu tiên.

4. **Ví dụ về TTL & cache**
   - File `index.html` trong S3 được chỉnh sửa nội dung:
     - Từ “I really like coffee” thành “I really like coffee every morning”.
   - Upload file mới lên S3 (không bật versioning nên file cũ bị ghi đè).
   - Truy cập trực tiếp từ S3 thấy nội dung mới.
   - Truy cập qua CloudFront vẫn thấy nội dung cũ vì:
     - CloudFront đã cache file với TTL (ví dụ 1 ngày) nên chưa yêu cầu lại từ S3.

5. **Invalidation (vô hiệu hóa cache)**
   - Để buộc CloudFront lấy phiên bản mới:
     - Vào tab **Invalidations** trong CloudFront.
     - Tạo invalidation với pattern `/*` (hoặc `*`), nghĩa là xóa mọi object trong cache.
   - Sau khi invalidation hoàn tất:
     - Truy cập lại URL CloudFront sẽ thấy nội dung mới “I really like coffee every morning”.
   - Đây là minh họa cách hoạt động của cache và invalidation trong CloudFront.
