


Dưới đây là tóm tắt ngắn gọn, theo đúng “góc độ đi thi”:

1. Khái niệm & mục đích  
- AWS AppSync là dịch vụ **managed GraphQL** trên AWS.  
- Dùng để xây dựng API GraphQL, có thể kết hợp dữ liệu từ nhiều nguồn (NoSQL, RDB, HTTP API, Lambda, …) thành một “graph” dữ liệu.  
- Hỗ trợ **real-time** qua WebSockets/MQTT và **offline sync** cho ứng dụng mobile/web.

2. GraphQL (trong bối cảnh AppSync)  
- Là kiểu API cho phép client **chỉ yêu cầu đúng các field cần** và nhận lại đúng như vậy.  
- Có thể **kết hợp nhiều nguồn dữ liệu** phía sau thành một schema thống nhất.  
- AppSync có tích hợp sẵn với: **DynamoDB, Aurora, OpenSearch**, và có thể mở rộng qua **Lambda** hoặc **HTTP endpoint**.

3. Real-time & Offline  
- AppSync hỗ trợ **WebSockets** và **MQTT over WebSockets** cho ứng dụng thời gian thực (dashboard real-time, chat, IoT, …).  
- Hỗ trợ **offline data sync** cho ứng dụng mobile, là lựa chọn thay thế cho cơ chế đồng bộ cũ (như “caching/cookie-based sync”).

4. Cách hoạt động (ở mức cao)  
- Bạn **tải lên một GraphQL schema** vào AppSync.  
- Client gửi **query/mutation/subscription** đến AppSync, chỉ định các field cần.  
- AppSync dùng **resolver** để truy xuất dữ liệu từ các nguồn (DynamoDB, Aurora, OpenSearch, Lambda, HTTP…).  
- AppSync trả về dữ liệu dạng **JSON**, đúng cấu trúc và field mà client yêu cầu.

5. Tích hợp & giám sát  
- Dùng cho: web app, mobile app, real-time dashboard, ứng dụng cần offline sync.  
- Tích hợp logging/metrics với **CloudWatch Logs** và **CloudWatch Metrics**.

6. Bảo mật (rất quan trọng cho kỳ thi)  
Có **4 cơ chế ủy quyền chính** cho API GraphQL AppSync:  
1) **API_KEY** – đơn giản, giống API Gateway API key.  
2) **AWS_IAM** – dùng IAM user/role/account để truy cập (giống API Gateway IAM auth).  
3) **OPENID_CONNECT** – tích hợp với nhà cung cấp OpenID Connect, dùng JWT.  
4) **AMAZON_COGNITO_USER_POOL** – tích hợp với Cognito User Pool (có thể liên kết social login, v.v.).

7. Miền tùy chỉnh & HTTPS  
- Nếu muốn **custom domain + HTTPS** cho AppSync, giải pháp khuyến nghị: đặt **CloudFront** phía trước AppSync.

Nếu bạn muốn, tôi có thể rút ra riêng một “cheat sheet” vài dòng chỉ để ôn thi.
