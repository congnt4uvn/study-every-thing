## Tóm tắt nội dung

- **Postman** và **curl** là các công cụ thường dùng để làm việc với **API** (REST, HTTP).
- Dùng để:
  - Gửi request (GET, POST, PUT, DELETE, …) đến server.
  - Kiểm thử API trong quá trình phát triển Java backend.
  - Xem request/response, header, body, status code.
- Thường được dùng song song với ứng dụng Java (Spring Boot, Jakarta EE, v.v.) để test các endpoint.

---

## Ví dụ với `curl`

Giả sử bạn có một API Java (Spring Boot) chạy ở `http://localhost:8080`:

### 1. Endpoint GET (lấy danh sách user)

**Java (Spring Boot)**

```java
@RestController
@RequestMapping("/users")
public class UserController {

    @GetMapping
    public List<String> getUsers() {
        return List.of("Alice", "Bob", "Charlie");
    }
}
```

**Gọi bằng curl (input)**

```bash
curl -X GET http://localhost:8080/users
```

**Output (response)**

```json
["Alice","Bob","Charlie"]
```

---

### 2. Endpoint POST (tạo user mới)

**Java (Spring Boot)**

```java
@RestController
@RequestMapping("/users")
public class UserController {

    @PostMapping
    public String createUser(@RequestBody String name) {
        return "Created user: " + name;
    }
}
```

**Gọi bằng curl (input)**

```bash
curl -X POST http://localhost:8080/users \
     -H "Content-Type: application/json" \
     -d "\"David\""
```

**Output (response)**

```text
Created user: David
```

---

## Ví dụ với Postman

Vẫn với các endpoint trên:

### 1. GET /users

- Method: `GET`
- URL: `http://localhost:8080/users`
- Body: (để trống)
- Nhấn **Send**

**Output (response)**

```json
["Alice","Bob","Charlie"]
```

### 2. POST /users

- Method: `POST`
- URL: `http://localhost:8080/users`
- Tab **Body**:
  - Chọn **raw**
  - Chọn **JSON**
  - Nội dung:

    ```json
    "David"
    ```

- Nhấn **Send**

**Output (response)**

```text
Created user: David
```

---

Tóm lại: Postman và curl là hai công cụ quan trọng trong hệ sinh thái phát triển Java để test và debug API, giúp gửi request và xem response một cách nhanh chóng và trực quan.
