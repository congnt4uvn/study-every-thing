
Dưới đây là tóm tắt ngắn gọn về **Collections & Data Structures: ArrayList, LinkedList** trong Java, kèm ví dụ mã và input/output.

---

## 1. Khái niệm chung

- Cả **ArrayList** và **LinkedList** đều là các lớp triển khai `List` trong Java.
- Cùng hỗ trợ:
  - Lưu trữ phần tử theo **thứ tự** (có index).
  - Cho phép **phần tử trùng lặp**.
  - Các thao tác cơ bản: `add`, `get`, `remove`, `size`, duyệt bằng `for`/`foreach`.

---

## 2. ArrayList

### Đặc điểm chính
- Dựa trên **mảng động** (dynamic array).
- Truy cập theo chỉ số (index) **rất nhanh**: `get(i)`, `set(i, value)` là O(1) trung bình.
- Thêm/xóa **ở cuối** danh sách thường nhanh.
- Thêm/xóa **ở giữa/đầu** tốn chi phí (phải dời phần tử).

### Khi nên dùng
- Khi cần **truy cập ngẫu nhiên** (random access) nhiều.
- Khi chủ yếu **thêm ở cuối**, ít chèn/xóa ở giữa.

### Ví dụ ArrayList

```java
import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        // Tạo ArrayList lưu Integer
        ArrayList<Integer> numbers = new ArrayList<>();

        // Input: thêm phần tử
        numbers.add(10);   // index 0
        numbers.add(20);   // index 1
        numbers.add(30);   // index 2
        numbers.add(1, 15); // chèn 15 vào index 1

        // Lấy phần tử
        int first = numbers.get(0);   // 10
        int second = numbers.get(1);  // 15

        // Xóa phần tử
        numbers.remove(2); // xóa phần tử tại index 2 (lúc này là 20)

        // In danh sách
        System.out.println("ArrayList: " + numbers);
        System.out.println("Phần tử đầu: " + first);
        System.out.println("Phần tử thứ 2: " + second);
    }
}
```

**Output:**
```text
ArrayList: [10, 15, 30]
Phần tử đầu: 10
Phần tử thứ 2: 15
```

## 3. LinkedList

### Đặc điểm chính
- Dựa trên **danh sách liên kết đôi** (doubly linked list).
- Mỗi phần tử (node) chứa:
  - dữ liệu
  - con trỏ đến node trước và node sau.
- Thêm/xóa **ở đầu/cuối** hoặc **ở vị trí đã có node tham chiếu** rất nhanh (O(1)).
- Truy cập theo index **chậm hơn** ArrayList (phải duyệt từ đầu/cuối đến vị trí đó).

### Khi nên dùng
- Khi cần **thêm/xóa nhiều** ở đầu/cuối hoặc giữa danh sách.
- Khi không cần truy cập ngẫu nhiên quá thường xuyên.

### Ví dụ LinkedList

```java
import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        // Tạo LinkedList lưu String
        LinkedList<String> names = new LinkedList<>();

        // Input: thêm phần tử
        names.add("An");
        names.add("Bình");
        names.add("Chi");

        // Thêm ở đầu và cuối
        names.addFirst("Đầu");
        names.addLast("Cuối");

        // Lấy phần tử
        String first = names.getFirst(); // "Đầu"
        String last = names.getLast();   // "Cuối"
        String middle = names.get(2);    // phần tử tại index 2

        // Xóa phần tử
        names.removeFirst(); // xóa "Đầu"
        names.removeLast();  // xóa "Cuối"

        // In danh sách
        System.out.println("LinkedList: " + names);
        System.out.println("Phần tử đầu ban đầu: " + first);
        System.out.println("Phần tử cuối ban đầu: " + last);
        System.out.println("Phần tử giữa: " + middle);
    }
}
```

**Output:**
```text
LinkedList: [An, Bình, Chi]
Phần tử đầu ban đầu: Đầu
Phần tử cuối ban đầu: Cuối
Phần tử giữa: An
```

## 4. So sánh nhanh

| Tiêu chí                     | ArrayList                         | LinkedList                          |
|-----------------------------|-----------------------------------|-------------------------------------|
| Cấu trúc                    | Mảng động                        | Danh sách liên kết đôi              |
| Truy cập theo index         | Rất nhanh (O(1) trung bình)      | Chậm hơn (O(n))                     |
| Thêm/xóa ở cuối             | Nhanh (thường O(1) amortized)    | Nhanh (O(1))                        |
| Thêm/xóa ở đầu/giữa         | Chậm (phải dời phần tử)          | Nhanh nếu đã có node tham chiếu     |
| Dùng khi                    | Cần truy cập ngẫu nhiên nhiều    | Cần thêm/xóa nhiều ở đầu/giữa/cuối |
