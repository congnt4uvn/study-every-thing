
Dưới đây là tóm tắt ngắn gọn về các loại test trong Spring Boot và ví dụ minh họa.

---

## 1. @SpringBootTest – Test tích hợp toàn bộ ứng dụng

- Dùng để test **tích hợp** (integration test).
- Khởi động **toàn bộ context Spring Boot** (gần giống chạy app thật).
- Phù hợp khi bạn muốn test nhiều layer cùng lúc: controller, service, repository, config,…

### Ví dụ

```java
// src/test/java/com/example/demo/DemoApplicationTests.java
@SpringBootTest
class DemoApplicationTests {

    @Autowired
    private MyService myService;

    @Test
    void testBusinessLogic() {
        String result = myService.process("Hello");
        assertEquals("HELLO", result);
    }
}
```

**Input:** `"Hello"`  
**Output mong đợi:** `"HELLO"`

## 2. @WebMvcTest – Test layer Web (Controller)

- Chỉ load **Web layer** (controller, filter, advice,…), không load service, repository thật.
- Thường dùng với **MockMvc** để test REST API / controller.
- Nhanh hơn @SpringBootTest vì không khởi động toàn bộ context.

### Ví dụ

```java
// src/main/java/com/example/demo/GreetingController.java
@RestController
@RequestMapping("/greet")
public class GreetingController {

    @GetMapping
    public String greet(@RequestParam String name) {
        return "Hello " + name;
    }
}
```

```java
// src/test/java/com/example/demo/GreetingControllerTest.java
@WebMvcTest(GreetingController.class)
class GreetingControllerTest {

    @Autowired
    private MockMvc mockMvc;

    @Test
    void testGreetEndpoint() throws Exception {
        mockMvc.perform(get("/greet")
                .param("name", "John"))
            .andExpect(status().isOk())
            .andExpect(content().string("Hello John"));
    }
}
```

**HTTP Input:** `GET /greet?name=John`  
**HTTP Output mong đợi:** `200 OK` với body `"Hello John"`

## 3. @DataJpaTest – Test layer JPA (Repository)

- Dùng để test **repository** và tương tác với database.
- Chỉ load các bean liên quan đến JPA: `EntityManager`, `Repository`, cấu hình JPA.
- Mặc định dùng **in-memory database** (H2) trừ khi cấu hình khác.

### Ví dụ

```java
// src/main/java/com/example/demo/User.java
@Entity
public class User {
    @Id @GeneratedValue
    private Long id;
    private String username;

    // getters/setters
}
```

```java
// src/main/java/com/example/demo/UserRepository.java
public interface UserRepository extends JpaRepository<User, Long> {
    Optional<User> findByUsername(String username);
}
```

```java
// src/test/java/com/example/demo/UserRepositoryTest.java
@DataJpaTest
class UserRepositoryTest {

    @Autowired
    private UserRepository userRepository;

    @Test
    void testFindByUsername() {
        User user = new User();
        user.setUsername("alice");
        userRepository.save(user);

        Optional<User> found = userRepository.findByUsername("alice");

        assertTrue(found.isPresent());
        assertEquals("alice", found.get().getUsername());
    }
}
```

**Input:** Lưu user có `username = "alice"`  
**Output mong đợi:** `findByUsername("alice")` trả về `Optional<User>` chứa user đó

---

## Tóm tắt nhanh

- `@SpringBootTest`:  
  - Test tích hợp toàn bộ ứng dụng.  
  - Dùng khi cần kiểm tra nhiều layer cùng lúc.

- `@WebMvcTest`:  
  - Test **controller / REST API**.  
  - Chỉ load web layer, dùng `MockMvc`.

- `@DataJpaTest`:  
  - Test **repository / JPA**.  
  - Chỉ load JPA layer, thường dùng in-memory DB.
