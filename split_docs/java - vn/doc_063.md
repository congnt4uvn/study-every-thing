
Dưới đây là tóm tắt ngắn gọn về **Spring Boot Starter Dependencies** (phụ thuộc starter trong Spring Boot), kèm ví dụ code và input/output.

---

## 1. Spring Boot Starter là gì?

- Là các **gói phụ thuộc (dependency)** được đóng gói sẵn, giúp bạn **khởi tạo nhanh** một loại ứng dụng Spring cụ thể.
- Thay vì phải tự tìm và khai báo nhiều thư viện, bạn chỉ cần thêm **1 starter** vào `pom.xml` (Maven) hoặc `build.gradle` (Gradle).
- Mỗi starter thường có tên dạng:  
  `spring-boot-starter-<tính_năng>`

---

## 2. Một số starter phổ biến

- `spring-boot-starter-web`: Xây dựng ứng dụng web, REST API (Spring MVC, Tomcat, JSON…)
- `spring-boot-starter-data-jpa`: Làm việc với database bằng JPA/Hibernate
- `spring-boot-starter-thymeleaf`: View layer với Thymeleaf
- `spring-boot-starter-security`: Bảo mật (authentication, authorization)
- `spring-boot-starter-test`: Thư viện test (JUnit, Mockito, Spring Test…)
- `spring-boot-starter-actuator`: Theo dõi, giám sát ứng dụng (health, metrics…)

---

## 3. Ví dụ với Maven

### 3.1. `pom.xml` dùng starter web + starter test

```xml
<project>
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.example</groupId>
    <artifactId>demo</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>3.2.0</version>
    </parent>

    <dependencies>
        <!-- Starter cho ứng dụng web / REST -->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

        <!-- Starter cho test -->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
    </build>
</project>
```

## 4. Ví dụ code REST API đơn giản

### 4.1. Controller

```java
// DemoApplication.java
package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication  // bật auto-configuration, component scan, v.v.
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

```java
// HelloController.java
package com.example.demo;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String hello() {
        return "Xin chào Spring Boot!";
    }
}
```

### 4.2. Chạy và kiểm tra

- Chạy ứng dụng (Maven):

```bash
mvn spring-boot:run
```

- Gửi request HTTP:

**Input (HTTP request):**

```http
GET http://localhost:8080/hello
```

**Output (HTTP response):**

```http
200 OK
Content-Type: text/plain;charset=UTF-8

Xin chào Spring Boot!
```

## 5. Ví dụ starter khác: `spring-boot-starter-data-jpa`

Thêm vào `pom.xml`:

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>

<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <scope>runtime</scope>
</dependency>
```

Entity + Repository:

```java
import jakarta.persistence.Entity;
import jakarta.persistence.GeneratedValue;
import jakarta.persistence.Id;

@Entity
public class User {
    @Id
    @GeneratedValue
    private Long id;
    private String name;

    // getters/setters
}
```

```java
import org.springframework.data.jpa.repository.JpaRepository;

public interface UserRepository extends JpaRepository<User, Long> {
}
```

Controller:

```java
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/users")
public class UserController {
    private final UserRepository repo;

    public UserController(UserRepository repo) {
        this.repo = repo;
    }

    @PostMapping
    public User create(@RequestBody User user) {
        return repo.save(user);
    }

    @GetMapping
    public List<User> findAll() {
        return repo.findAll();
    }
}
```

**Input (tạo user):**

```http
POST http://localhost:8080/users
Content-Type: application/json

{
  "name": "Nguyen Van A"
}
```

**Output (response):**

```json
{
  "id": 1,
  "name": "Nguyen Van A"
}
```

## 6. Ý chính cần nhớ

- Starter dependencies giúp:
  - Cấu hình nhanh, ít phải khai báo thư viện lẻ.
  - Tránh xung đột version nhờ Spring Boot quản lý.
  - Tập trung vào logic nghiệp vụ thay vì cấu hình phức tạp.
- Chọn starter phù hợp với loại ứng dụng: web, data, security, template engine, test, actuator, v.v.
