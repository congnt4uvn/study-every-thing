
Dưới đây là tóm tắt ngắn gọn về các tính năng mới nổi bật trong **Java 8**, kèm ví dụ code và input/output minh họa.

---

## 1. Lambda Expressions (Biểu thức Lambda)

Giúp viết code ngắn gọn hơn, đặc biệt với các interface chỉ có 1 method (functional interface).

**Ví dụ:**

```java
import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("An", "Bình", "Châu");

        // Dùng lambda để duyệt danh sách
        names.forEach(name -> System.out.println(name));
    }
}
```

**Output:**
```
An
Bình
Châu
```

## 2. Functional Interfaces (Interface hàm)

Là interface chỉ có **một** phương thức trừu tượng, ví dụ: `Runnable`, `Callable`, `Comparator`, và các interface mới trong `java.util.function` như `Function`, `Predicate`, `Consumer`.

**Ví dụ:**

```java
import java.util.function.Predicate;

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        Predicate<Integer> isEven = x -> x % 2 == 0;

        System.out.println(isEven.test(4)); // true
        System.out.println(isEven.test(5)); // false
    }
}
```

**Output:**
```
true
false
```

## 3. Stream API

Cung cấp cách xử lý dữ liệu theo kiểu “luồng” (stream) với các thao tác như `filter`, `map`, `collect`, hỗ trợ lập trình hàm và dễ song song hóa.

**Ví dụ: lọc số chẵn và nhân đôi**

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

        List<Integer> result = numbers.stream()
                .filter(n -> n % 2 == 0)   // lọc số chẵn
                .map(n -> n * 2)           // nhân đôi
                .collect(Collectors.toList());

        System.out.println(result);
    }
}
```

**Output:**
```
[4, 8, 12]
```

## 4. Default Methods trong Interface

Cho phép định nghĩa **method có thân** (có implementation) trong interface với từ khóa `default`. Giúp mở rộng interface mà không làm hỏng code cũ.

**Ví dụ:**

```java
interface MyInterface {
    void doSomething();

    default void log(String message) {
        System.out.println("LOG: " + message);
    }
}

public class DefaultMethodExample implements MyInterface {
    @Override
    public void doSomething() {
        log("Đang làm gì đó...");
    }

    public static void main(String[] args) {
        DefaultMethodExample ex = new DefaultMethodExample();
        ex.doSomething();
    }
}
```

**Output:**
```
LOG: Đang làm gì đó...
```

## 5. Method References (Tham chiếu method)

Cú pháp ngắn gọn để tham chiếu đến method đã tồn tại, thay cho lambda.

**Ví dụ:**

```java
import java.util.Arrays;
import java.util.List;

public class MethodRefExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("An", "Bình", "Châu");

        // Thay vì: names.forEach(name -> System.out.println(name));
        names.forEach(System.out::println);
    }
}
```

**Output:**
```
An
Bình
Châu
```

## 6. New Date and Time API (java.time)

Thư viện thời gian mới: `LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime`, an toàn luồng (thread-safe), dễ dùng hơn `Date`/`Calendar`.

**Ví dụ:**

```java
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class DateTimeExample {
    public static void main(String[] args) {
        LocalDate today = LocalDate.now();
        LocalDateTime now = LocalDateTime.now();

        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm");

        System.out.println("Today: " + today);
        System.out.println("Now: " + now.format(formatter));
    }
}
```

**Output (ví dụ):**
```
Today: 2026-05-20
Now: 20/05/2026 14:35
```

## 7. Optional

Giúp xử lý giá trị có thể null một cách an toàn, giảm `NullPointerException`.

**Ví dụ:**

```java
import java.util.Optional;

public class OptionalExample {
    public static void main(String[] args) {
        Optional<String> name = Optional.ofNullable(getName());

        String result = name.orElse("Không có tên");
        System.out.println(result);
    }

    static String getName() {
        return null; // giả sử lấy từ DB nhưng không có dữ liệu
    }
}
```

**Output:**
```
Không có tên
```

## 8. Parallel Streams (Stream song song)

Dễ dàng xử lý song song để tận dụng đa lõi CPU.

**Ví dụ:**

```java
import java.util.stream.LongStream;

public class ParallelStreamExample {
    public static void main(String[] args) {
        long sum = LongStream.rangeClosed(1, 10)
                .parallel()   // xử lý song song
                .sum();

        System.out.println(sum);
    }
}
```

**Output:**
```
55
```

---

### Tóm tắt ngắn

Java 8 giới thiệu:
- Lambda expressions và method references → code ngắn gọn, hỗ trợ lập trình hàm.
- Functional interfaces → chuẩn hóa các kiểu hàm (Function, Predicate, Consumer…).
- Stream API và parallel streams → xử lý tập dữ liệu mạnh mẽ, dễ song song hóa.
- Default methods trong interface → mở rộng API mà không phá vỡ code cũ.
- Date/Time API mới (`java.time`) → làm việc với thời gian hiện đại, an toàn.
- Optional → giảm lỗi null, code rõ ràng hơn.

Nếu bạn muốn, tôi có thể viết thêm ví dụ chi tiết cho từng tính năng hoặc so sánh Java 7 vs Java 8.
