
Dưới đây là tóm tắt ngắn gọn về các cấu trúc dữ liệu Java: HashSet, TreeSet, HashMap, LinkedHashMap, kèm ví dụ code và input/output.

---

## 1. HashSet

- Lưu trữ các phần tử **không trùng lặp**.
- Không đảm bảo **thứ tự** phần tử.
- Dựa trên **hash table**, thao tác thêm/tìm/xóa thường O(1).

### Ví dụ

```java
import java.util.HashSet;

public class HashSetExample {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();

        set.add("Apple");
        set.add("Banana");
        set.add("Orange");
        set.add("Apple"); // trùng, sẽ không được thêm

        System.out.println(set);
        System.out.println("Contains 'Banana'? " + set.contains("Banana"));
    }
}
```

**Output (thứ tự có thể khác):**
```text
[Orange, Banana, Apple]
Contains 'Banana'? true
```

## 2. TreeSet

- Lưu trữ các phần tử **không trùng lặp**.
- **Có sắp xếp** (theo thứ tự tự nhiên hoặc Comparator).
- Dựa trên **cây đỏ-đen**, thao tác O(log n).

### Ví dụ

```java
import java.util.TreeSet;

public class TreeSetExample {
    public static void main(String[] args) {
        TreeSet<Integer> set = new TreeSet<>();

        set.add(5);
        set.add(1);
        set.add(3);
        set.add(3); // trùng, không thêm

        System.out.println(set);
        System.out.println("First: " + set.first());
        System.out.println("Last: " + set.last());
    }
}
```

**Output:**
```text
[1, 3, 5]
First: 1
Last: 5
```

## 3. HashMap

- Lưu trữ dữ liệu dạng **key–value**.
- Key **không trùng lặp**, value có thể trùng.
- Không đảm bảo **thứ tự** các phần tử.
- Dựa trên **hash table**, thao tác get/put thường O(1).

### Ví dụ

```java
import java.util.HashMap;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap<String, Integer> map = new HashMap<>();

        map.put("Alice", 25);
        map.put("Bob", 30);
        map.put("Charlie", 28);
        map.put("Alice", 26); // ghi đè value của key "Alice"

        System.out.println(map);
        System.out.println("Age of Bob: " + map.get("Bob"));
        System.out.println("Contains key 'David'? " + map.containsKey("David"));
    }
}
```

**Output (thứ tự có thể khác):**
```text
{Bob=30, Alice=26, Charlie=28}
Age of Bob: 30
Contains key 'David'? false
```

## 4. LinkedHashMap

- Giống HashMap nhưng:
  - **Giữ thứ tự** chèn phần tử (insertion order) hoặc access order (nếu cấu hình).
- Hữu ích khi cần vừa map vừa giữ thứ tự.

### Ví dụ

```java
import java.util.LinkedHashMap;
import java.util.Map;

public class LinkedHashMapExample {
    public static void main(String[] args) {
        LinkedHashMap<String, Integer> map = new LinkedHashMap<>();

        map.put("A", 1);
        map.put("C", 3);
        map.put("B", 2);

        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }
    }
}
```

**Output (giữ thứ tự chèn):**
```text
A -> 1
C -> 3
B -> 2
```

---

## So sánh nhanh

- **HashSet**: Tập hợp không trùng, không thứ tự, nhanh.
- **TreeSet**: Tập hợp không trùng, có sắp xếp, chậm hơn HashSet.
- **HashMap**: Map key–value, không thứ tự, nhanh.
- **LinkedHashMap**: Map key–value, **giữ thứ tự chèn**, chậm hơn chút so với HashMap.
