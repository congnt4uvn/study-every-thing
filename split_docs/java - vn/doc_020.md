
Dưới đây là tóm tắt ngắn gọn về các interface chính trong Java Collections: `List`, `Set`, `Map`, `Queue`, `Deque`, kèm ví dụ code và input/output.

---

## 1. List
- **Đặc điểm**:
  - Lưu các phần tử **có thứ tự** (ordered).
  - **Cho phép trùng lặp** (duplicates).
  - Truy cập bằng **chỉ số (index)**.
- **Cài đặt phổ biến**: `ArrayList`, `LinkedList`, `Vector`.

### Ví dụ (ArrayList)

```java
import java.util.*;

public class ListExample {
    public static void main(String[] args) {
        List<String> names = new ArrayList<>();
        names.add("An");
        names.add("Bình");
        names.add("An"); // trùng lặp

        System.out.println(names);        // [An, Bình, An]
        System.out.println(names.get(1)); // Bình
    }
}
```

**Output:**
```text
[An, Bình, An]
Bình
```

---

## 2. Set
- **Đặc điểm**:
  - **Không cho phép trùng lặp**.
  - Không đảm bảo thứ tự (với `HashSet`).
  - Một số Set có thứ tự:
    - `LinkedHashSet`: giữ thứ tự thêm vào.
    - `TreeSet`: sắp xếp tăng dần (theo tự nhiên hoặc comparator).

### Ví dụ (HashSet)

```java
import java.util.*;

public class SetExample {
    public static void main(String[] args) {
        Set<Integer> numbers = new HashSet<>();
        numbers.add(3);
        numbers.add(1);
        numbers.add(3); // bị bỏ qua vì trùng

        System.out.println(numbers);
    }
}
```

**Output (thứ tự có thể khác):**
```text
[1, 3]
```

---

## 3. Map
- **Đặc điểm**:
  - Lưu dữ liệu dạng **key–value**.
  - **Key không trùng lặp**, value có thể trùng.
  - Truy xuất value bằng key.
- **Cài đặt phổ biến**: `HashMap`, `LinkedHashMap`, `TreeMap`.

### Ví dụ (HashMap)

```java
import java.util.*;

public class MapExample {
    public static void main(String[] args) {
        Map<String, Integer> scores = new HashMap<>();
        scores.put("An", 9);
        scores.put("Bình", 8);
        scores.put("An", 10); // ghi đè value của key "An"

        System.out.println(scores);
        System.out.println("Điểm của An: " + scores.get("An"));
    }
}
```

**Output (thứ tự có thể khác):**
```text
{Bình=8, An=10}
Điểm của An: 10
```

## 4. Queue
- **Đặc điểm**:
  - Hàng đợi, thường theo kiểu **FIFO** (First In First Out).
  - Dùng cho xử lý theo thứ tự đến trước – xử lý trước.
- **Cài đặt phổ biến**: `LinkedList`, `PriorityQueue`.

### Ví dụ (Queue với LinkedList)

```java
import java.util.*;

public class QueueExample {
    public static void main(String[] args) {
        Queue<String> queue = new LinkedList<>();
        queue.add("A");
        queue.add("B");
        queue.add("C");

        System.out.println(queue);          // [A, B, C]
        System.out.println(queue.poll());   // lấy và xóa phần tử đầu: A
        System.out.println(queue);          // [B, C]
        System.out.println(queue.peek());   // xem phần tử đầu: B
    }
}
```

**Output:**
```text
[A, B, C]
A
[B, C]
B
```

## 5. Deque (Double-Ended Queue)
- **Đặc điểm**:
  - Hàng đợi hai đầu: thêm/xóa ở **cả đầu và cuối**.
  - Có thể dùng như **Queue** hoặc **Stack**.
- **Cài đặt phổ biến**: `ArrayDeque`, `LinkedList`.

### Ví dụ (Deque như Stack & Queue)

```java
import java.util.*;

public class DequeExample {
    public static void main(String[] args) {
        Deque<Integer> deque = new ArrayDeque<>();

        // Dùng như Queue (FIFO)
        deque.addLast(1);
        deque.addLast(2);
        deque.addLast(3);
        System.out.println(deque);          // [1, 2, 3]
        System.out.println(deque.pollFirst()); // 1

        // Dùng như Stack (LIFO)
        deque.push(4); // thêm vào đầu
        System.out.println(deque);          // [4, 2, 3]
        System.out.println(deque.pop());    // 4
    }
}
```

**Output:**
```text
[1, 2, 3]
1
[4, 2, 3]
4
```

---

## Tóm tắt nhanh

- **List**: Có thứ tự, cho phép trùng lặp, truy cập bằng index.  
- **Set**: Không trùng lặp, thường không đảm bảo thứ tự.  
- **Map**: Cặp key–value, key không trùng.  
- **Queue**: Hàng đợi FIFO.  
- **Deque**: Hàng đợi hai đầu, dùng được như Queue hoặc Stack.
