
Dưới đây là tóm tắt ngắn gọn các tính năng mới nổi bật từ Java 9 đến Java 17, kèm ví dụ code và input/output khi phù hợp.

---

## Java 9

### 1. Module System (JPMS – Project Jigsaw)
Giúp chia ứng dụng thành các module rõ ràng, dễ quản lý và bảo mật hơn.

**Ví dụ: module-info.java**
```java
module com.example.app {
    requires java.sql;
    exports com.example.service;
}
```

---

### 2. JShell (REPL)
Cho phép chạy thử từng dòng code Java trong terminal.

**Ví dụ (trong JShell):**
```text
jshell> int x = 5;
x ==> 5

jshell> x * 2
$2 ==> 10
```

---

### 3. Stream API cải tiến
Thêm các method như `takeWhile`, `dropWhile`, `ofNullable`.

```java
import java.util.stream.Stream;

public class Main {
    public static void main(String[] args) {
        Stream<Integer> s = Stream.of(1, 2, 3, 0, 4, 5);

        s.takeWhile(n -> n < 3)
         .forEach(System.out::println);
    }
}
```

**Output:**
```text
1
2
```

---

### 4. Optional.ifPresentOrElse
```java
import java.util.Optional;

public class Main {
    public static void main(String[] args) {
        Optional<String> opt = Optional.of("Hello");

        opt.ifPresentOrElse(
            v -> System.out.println("Value: " + v),
            () -> System.out.println("Empty")
        );
    }
}
```

**Output:**
```text
Value: Hello
```

## Java 10

### 1. `var` – Local Variable Type Inference
Giúp code ngắn gọn hơn, nhưng vẫn là kiểu tĩnh.

```java
public class Main {
    public static void main(String[] args) {
        var message = "Xin chào";
        var number = 10;
        System.out.println(message + " " + number);
    }
}
```

**Output:**
```text
Xin chào 10
```

## Java 11 (LTS)

### 1. Chuỗi – `isBlank`, `lines`, `strip`, `repeat`
```java
public class Main {
    public static void main(String[] args) {
        String s = "  Hello\nWorld  ";

        System.out.println(s.isBlank());          // false
        s.lines().forEach(System.out::println);   // Hello / World
        System.out.println(s.strip());            // "Hello\nWorld"
        System.out.println("ha".repeat(3));       // "hahaha"
    }
}
```

---

### 2. `var` trong lambda parameters
```java
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> list = List.of("a", "b", "c");
        list.forEach((var x) -> System.out.println(x.toUpperCase()));
    }
}
```

**Output:**
```text
A
B
C
```

---

### 3. HTTP Client API mới (java.net.http)
Thay thế HttpURLConnection, hỗ trợ HTTP/2.

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class Main {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://httpbin.org/get"))
                .build();

        HttpResponse<String> response =
                client.send(request, HttpResponse.BodyHandlers.ofString());

        System.out.println(response.statusCode());
        System.out.println(response.body().substring(0, 60) + "...");
    }
}
```

**Output (ví dụ):**
```text
200
{
  "args": {},
  "headers": {
    "Accept-Encoding": "gzip,...
```

## Java 12–13 (một số điểm chính)

### 1. Switch Expressions (preview → chính thức ở Java 14)
Giúp `switch` gọn hơn, có thể trả về giá trị.

```java
public class Main {
    public static void main(String[] args) {
        String day = "MONDAY";
        int num = switch (day) {
            case "MONDAY", "TUESDAY", "WEDNESDAY" -> 1;
            case "THURSDAY", "FRIDAY" -> 2;
            case "SATURDAY", "SUNDAY" -> 3;
            default -> 0;
        };
        System.out.println(num);
    }
}
```

**Output:**
```text
1
```

## Java 14

### 1. `record` (preview → chính thức ở Java 16)
Kiểu dữ liệu bất biến, dùng để chứa dữ liệu, tự sinh constructor, getter, `equals`, `hashCode`, `toString`.

```java
public record Person(String name, int age) {}

public class Main {
    public static void main(String[] args) {
        Person p = new Person("An", 20);
        System.out.println(p.name());
        System.out.println(p);
    }
}
```

**Output:**
```text
An
Person[name=An, age=20]
```

---

### 2. `instanceof` pattern matching (preview → chính thức ở Java 16)
Giảm code cast thủ công.

```java
public class Main {
    public static void main(String[] args) {
        Object obj = "Hello";

        if (obj instanceof String s) {
            System.out.println(s.toUpperCase());
        }
    }
}
```

**Output:**
```text
HELLO
```

## Java 15

### 1. Text Blocks (chuỗi nhiều dòng)
Dùng `"""` để viết chuỗi nhiều dòng dễ đọc.

```java
        String json = """

                }
                """;
        System.out.println(json);
    }
}
```

**Output:**
```text
{
  "name": "An",
  "age": 20
}
```

## Java 17 (LTS)

### 1. Sealed Classes
Giới hạn lớp nào được phép kế thừa, giúp thiết kế domain rõ ràng.

```java
public sealed interface Shape
        permits Circle, Rectangle {}

public final class Circle implements Shape {
    double radius;
    public Circle(double radius) { this.radius = radius; }
}

public final class Rectangle implements Shape {
    double w, h;
    public Rectangle(double w, double h) { this.w = w; this.h = h; }
}
```

---

### 2. Switch + pattern matching (preview)
Kết hợp `switch` với pattern, giúp xử lý kiểu dữ liệu rõ ràng hơn.

```java
public class Main {
    static String formatShape(Object obj) {
        return switch (obj) {
            case Circle c -> "Circle radius=" + c.radius;
            case Rectangle r -> "Rectangle " + r.w + "x" + r.h;
            default -> "Unknown";
        };
    }

    public static void main(String[] args) {
        System.out.println(formatShape(new Circle(2.5)));
    }
}
```

**Output:**
```text
Circle radius=2.5
```

## Tóm tắt ý chính

- **Java 9**: Module system, JShell, cải tiến Stream & Optional.  
- **Java 10**: `var` cho biến cục bộ.  
- **Java 11 (LTS)**: API chuỗi mới, HTTP Client chuẩn, `var` trong lambda.  
- **Java 12–13**: Switch expressions (gọn, trả về giá trị).  
- **Java 14–16**: `record`, pattern matching cho `instanceof`, text blocks.  
- **Java 17 (LTS)**: Sealed classes, tiếp tục mở rộng pattern matching, dọn dẹp các API cũ.

Nếu bạn muốn, tôi có thể viết một chương trình nhỏ dùng **record + sealed + switch expression** kết hợp trong một ví dụ hoàn chỉnh.
