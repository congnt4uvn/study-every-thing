
Dưới đây là tóm tắt ngắn gọn về JVM, JDK, JRE (bằng tiếng Việt) kèm ví dụ:

---

## 1. JVM là gì?

**JVM (Java Virtual Machine)**  
- Là “máy ảo” chạy chương trình Java.  
- Nhiệm vụ chính:
  - Nạp file `.class` (bytecode).
  - Kiểm tra, tối ưu, và thực thi bytecode.
  - Quản lý bộ nhớ (garbage collection).
- JVM là phần giúp Java có tính “Write once, run anywhere” (viết một lần, chạy mọi nơi).

---

## 2. JRE là gì?

**JRE (Java Runtime Environment)**  
- Môi trường chạy Java, dùng cho **chạy** ứng dụng Java (không dùng để biên dịch).  
- Bao gồm:
  - JVM
  - Thư viện chuẩn Java (Java Standard Library)
  - Một số file cấu hình, resource cần thiết.
- Nếu bạn chỉ cần **chạy** chương trình Java (không lập trình), cài JRE là đủ.

---

## 3. JDK là gì?

**JDK (Java Development Kit)**  
- Bộ công cụ dành cho **lập trình viên Java**.  
- Bao gồm:
  - JRE (tức là có cả JVM + thư viện chuẩn)
  - Công cụ phát triển: `javac`, `java`, `javadoc`, `jar`, v.v.
- Dùng JDK để:
  - Biên dịch mã nguồn `.java` thành `.class` (bytecode).
  - Chạy, debug, đóng gói ứng dụng Java.

---

## 4. Quan hệ giữa JDK – JRE – JVM

Có thể hình dung như sau:

- **JVM**: lõi để chạy bytecode.
- **JRE = JVM + thư viện chuẩn** → dùng để chạy chương trình.
- **JDK = JRE + công cụ phát triển** → dùng để phát triển + chạy chương trình.

---

## 5. Ví dụ code Java đơn giản

### File: `HelloWorld.java`

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào Java!");
    }
}
```

### Bước 1: Biên dịch (dùng JDK – công cụ `javac`)

**Input (lệnh):**
```bash
javac HelloWorld.java
```

**Output:**
- Tạo ra file bytecode: `HelloWorld.class`

### Bước 2: Chạy chương trình (dùng JVM thông qua lệnh `java`)

**Input (lệnh):**
```bash
java HelloWorld
```

**Output trên màn hình:**
```text
Xin chào Java!
```

Quy trình:
1. `javac` (thuộc JDK) biên dịch `.java` → `.class`.
2. `java` (JRE/JVM) nạp file `.class` và thực thi bytecode trên JVM.

---

## 6. Tóm tắt cực ngắn

- **JVM**: Chạy bytecode Java.
- **JRE**: JVM + thư viện → để **chạy** ứng dụng Java.
- **JDK**: JRE + công cụ phát triển → để **viết, biên dịch, chạy** ứng dụng Java.
