
Dưới đây là tóm tắt ngắn gọn về **Text Blocks** trong Java (từ Java 15 trở đi).

---

## 1. Text Block là gì?

**Text Block** là một cách mới để viết chuỗi nhiều dòng trong Java, dùng cú pháp `"""` (ba dấu ngoặc kép):

- Bắt đầu bằng `"""` trên một dòng riêng.
- Kết thúc bằng `"""`.
- Tự động xử lý xuống dòng và dấu ngoặc kép bên trong dễ dàng hơn.

---

## 2. Lợi ích chính

1. **Dễ đọc, dễ viết**:  
   - Không cần nối chuỗi bằng `+` nhiều lần.
   - Không cần escape nhiều ký tự như `\"` hoặc `\n`.

2. **Giữ định dạng**:  
   - Phù hợp cho SQL, JSON, HTML, XML, template, v.v.

3. **Ít lỗi hơn**:  
   - Code gọn, ít ký tự escape → ít bug.

---

## 3. Ví dụ so sánh

### Trước đây (chuỗi thường)

```java
String json = "{\n" +
              "  \"name\": \"An\",\n" +
              "  \"age\": 25,\n" +
              "  \"city\": \"Hanoi\"\n" +
              "}";
System.out.println(json);
```

**Output:**
```json
{
  "name": "An",
  "age": 25,
  "city": "Hanoi"
}
```

### Với Text Block

```java
String json = """
    {
      "name": "An",
      "age": 25,
      "city": "Hanoi"
    }
    """;
System.out.println(json);
```

**Output giống hệt**:
```json
{
  "name": "An",
  "age": 25,
  "city": "Hanoi"
}
```

## 4. Ví dụ với HTML

```java
String html = """
    <html>
        <head>
            <title>Trang ví dụ</title>
        </head>
        <body>
            <h1>Xin chào</h1>
            <p>Đây là ví dụ về Text Block.</p>
        </body>
    </html>
    """;

System.out.println(html);
```

**Output:**
```html
<html>
    <head>
        <title>Trang ví dụ</title>
    </head>
    <body>
        <h1>Xin chào</h1>
        <p>Đây là ví dụ về Text Block.</p>
    </body>
</html>
```

## 5. Ví dụ với SQL

```java
String sql = """
    SELECT id, name, age
    FROM users
    WHERE age > 18
    ORDER BY name;
    """;

System.out.println(sql);
```

**Output:**
```sql
SELECT id, name, age
FROM users
WHERE age > 18
ORDER BY name;
```

---

## 6. Một số lưu ý

- Text Block vẫn là kiểu `String` bình thường.
- Có thể dùng `String.format()` hoặc `formatted()`:

```java
String name = "An";
int age = 25;

String info = """
    Tên: %s
    Tuổi: %d
    """.formatted(name, age);

System.out.println(info);
```

**Output:**
```
Tên: An
Tuổi: 25
```

---

**Tóm tắt:**  
Text Blocks là cú pháp mới dùng `"""` để viết chuỗi nhiều dòng trong Java, giúp code gọn hơn, dễ đọc hơn, đặc biệt hữu ích cho JSON, SQL, HTML, XML và các template văn bản phức tạp.
