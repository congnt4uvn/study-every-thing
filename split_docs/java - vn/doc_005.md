
Dưới đây là tóm tắt ngắn gọn về **Lambda** và **Method Reference** trong Java 8+ (bằng tiếng Việt), kèm ví dụ code và input/output.

---

## 1. Lambda Expression

### Ý chính
- Là cách viết ngắn gọn cho **anonymous class** (class vô danh) implement interface có **1 method trừu tượng** (Functional Interface).
- Giúp code ngắn hơn, dễ đọc hơn, đặc biệt khi dùng với `Collections`, `Streams`, `Runnable`, v.v.

### Cú pháp cơ bản
```java
(parameters) -> expression
(parameters) -> { statements }
```

### Ví dụ 1: So sánh trước và sau khi dùng lambda

**Trước (Java cũ – anonymous class):**
```java
import java.util.Arrays;
import java.util.List;

public class OldStyle {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("An", "Bình", "Chi");

        // Dùng anonymous class
        names.forEach(new java.util.function.Consumer<String>() {
            @Override
            public void accept(String s) {
                System.out.println(s);
            }
        });
    }
}
```

**Sau (Java 8 – lambda):**
```java
import java.util.Arrays;
import java.util.List;

public class LambdaStyle {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("An", "Bình", "Chi");

        // Dùng lambda
        names.forEach(s -> System.out.println(s));
        // hoặc ngắn hơn nữa với method reference (phần dưới)
    }
}
```

**Input:**
```text
Danh sách: ["An", "Bình", "Chi"]
```

**Output:**
```text
An
Bình
Chi
```

---

### Ví dụ 2: Lambda với Functional Interface tự định nghĩa

```java
@FunctionalInterface
interface Calculator {
    int operate(int a, int b);
}

public class LambdaCalculator {
    public static void main(String[] args) {
        // Cộng
        Calculator add = (a, b) -> a + b;
        // Nhân
        Calculator multiply = (a, b) -> a * b;

        int x = 5;
        int y = 3;

        System.out.println("5 + 3 = " + add.operate(x, y));
        System.out.println("5 * 3 = " + multiply.operate(x, y));
    }
}
```

**Output:**
```text
5 + 3 = 8
5 * 3 = 15
```

## 2. Method Reference

### Ý chính
- Là cú pháp rút gọn của lambda khi **lambda chỉ đơn giản gọi lại một method đã có**.
- Giúp code ngắn gọn, dễ đọc hơn.
- Dạng tổng quát:  
  - `ClassName::staticMethod`
  - `objectRef::instanceMethod`
  - `ClassName::instanceMethod` (dùng cho đối tượng được truyền vào)
  - `ClassName::new` (constructor reference)

### Ví dụ 1: In phần tử danh sách

**Lambda:**
```java
names.forEach(s -> System.out.println(s));
```

**Method reference tương đương:**
```java
names.forEach(System.out::println);
```

**Input:**
```text
["An", "Bình", "Chi"]
```

**Output:**
```text
An
Bình
Chi
```

---

### Ví dụ 2: Method reference tới static method

```java
import java.util.Arrays;
import java.util.List;

public class StaticMethodRef {
    public static void printUpper(String s) {
        System.out.println(s.toUpperCase());
    }

    public static void main(String[] args) {
        List<String> names = Arrays.asList("an", "bình", "chi");

        // Dùng lambda
        names.forEach(s -> StaticMethodRef.printUpper(s));

        // Dùng method reference
        names.forEach(StaticMethodRef::printUpper);
    }
}
```

**Output:**
```text
AN
BÌNH
CHI
```

---

### Ví dụ 3: Constructor reference

```java
import java.util.function.Function;

class Person {
    String name;
    Person(String name) {
        this.name = name;
    }
}

public class ConstructorRef {
    public static void main(String[] args) {
        // Lambda
        Function<String, Person> personCreator1 = name -> new Person(name);

        // Constructor reference
        Function<String, Person> personCreator2 = Person::new;

        Person p = personCreator2.apply("An");
        System.out.println(p.name);
    }
}
```

**Output:**
```text
An
```

## Tóm tắt ngắn

- **Lambda**:
  - Viết gọn cho anonymous class của Functional Interface.
  - Cú pháp: `(params) -> { body }`.
  - Dùng nhiều với `Collections`, `Streams`, `Runnable`, `Callable`, v.v.

- **Method Reference**:
  - Viết gọn cho lambda chỉ gọi một method có sẵn.
  - Các dạng chính:
    - `System.out::println`
    - `ClassName::staticMethod`
    - `objectRef::instanceMethod`
    - `ClassName::new` (constructor)

Nếu bạn muốn, tôi có thể thêm ví dụ với `Stream` (filter, map, sort) dùng lambda và method reference.
