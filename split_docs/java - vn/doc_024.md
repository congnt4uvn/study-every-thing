
Dưới đây là tóm tắt ngắn gọn về độ phức tạp Big-O của các cấu trúc dữ liệu/Collections thường dùng trong Java, kèm ví dụ mã và input/output minh họa.

---

## 1. Array / ArrayList

**Độ phức tạp (trung bình):**

- Truy cập theo chỉ số: `O(1)`
- Thêm cuối danh sách: `O(1)` (amortized)
- Thêm/xóa ở giữa: `O(n)`
- Tìm kiếm tuyến tính: `O(n)`

**Ví dụ:**

```java
import java.util.ArrayList;
import java.util.List;

public class ArrayListDemo {
    public static void main(String[] args) {
        List<Integer> list = new ArrayList<>();

        // Thêm phần tử (append) – O(1) amortized
        list.add(10);
        list.add(20);
        list.add(30);

        // Truy cập theo chỉ số – O(1)
        int x = list.get(1); // 20

        // Chèn vào giữa – O(n)
        list.add(1, 15); // [10, 15, 20, 30]

        // Xóa ở giữa – O(n)
        list.remove(2); // [10, 15, 30]

        System.out.println(list);
        System.out.println("Element at index 1: " + x);
    }
}
```

**Input/Output minh họa:**

- Input (các thao tác): `add 10, add 20, add 30, add(1, 15), remove(2)`
- Output:
  - In ra list: `[10, 15, 30]`
  - In ra `x`: `Element at index 1: 20`

## 2. LinkedList

**Độ phức tạp (trung bình):**

- Thêm/xóa ở đầu/cuối (khi có reference): `O(1)`
- Thêm/xóa ở giữa (theo index): `O(n)` (phải duyệt)
- Truy cập theo chỉ số: `O(n)`
- Tìm kiếm tuyến tính: `O(n)`

**Ví dụ:**

```java
import java.util.LinkedList;

public class LinkedListDemo {
    public static void main(String[] args) {
        LinkedList<String> list = new LinkedList<>();

        // Thêm đầu/cuối – O(1)
        list.addFirst("B");
        list.addLast("C");
        list.addFirst("A"); // [A, B, C]

        // Truy cập theo index – O(n)
        String s = list.get(1); // "B"

        // Xóa đầu – O(1)
        list.removeFirst(); // [B, C]

        System.out.println(list);
        System.out.println("Element at index 1: " + s);
    }
}
```

## 3. HashMap / HashSet

**Độ phức tạp (trung bình):**

- Thêm (put/add): `O(1)`
- Tìm kiếm (get/contains): `O(1)`
- Xóa (remove): `O(1)`
- Trường hợp xấu nhất (nhiều collision): `O(n)`

**Ví dụ HashMap:**

```java
import java.util.HashMap;
import java.util.Map;

public class HashMapDemo {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();

        // put – O(1) trung bình
        map.put("Alice", 25);
        map.put("Bob", 30);

        // get – O(1) trung bình
        int age = map.get("Alice"); // 25

        // containsKey – O(1) trung bình
        boolean hasBob = map.containsKey("Bob");

        // remove – O(1) trung bình
        map.remove("Bob");

        System.out.println(map);
        System.out.println("Alice age: " + age);
        System.out.println("Has Bob? " + hasBob);
    }
}
```

**Input/Output minh họa:**

- Thao tác: `put(Alice,25)`, `put(Bob,30)`, `get(Alice)`, `containsKey(Bob)`, `remove(Bob)`
- Output:
  - Map: `{Alice=25}`
  - `Alice age: 25`
  - `Has Bob? true`

## 4. TreeMap / TreeSet (cây đỏ-đen)

**Độ phức tạp (trung bình & xấu nhất):**

- Thêm (put/add): `O(log n)`
- Tìm kiếm (get/contains): `O(log n)`
- Xóa (remove): `O(log n)`
- Duyệt theo thứ tự (sorted iteration): `O(n)`

**Ví dụ TreeMap:**

```java
import java.util.Map;
import java.util.TreeMap;

public class TreeMapDemo {
    public static void main(String[] args) {
        Map<Integer, String> map = new TreeMap<>();

        map.put(3, "C");
        map.put(1, "A");
        map.put(2, "B");

        // get – O(log n)
        String val = map.get(2); // "B"

        // Duyệt theo thứ tự key tăng dần – O(n)
        for (Map.Entry<Integer, String> e : map.entrySet()) {
            System.out.println(e.getKey() + " -> " + e.getValue());
        }

        System.out.println("Value at key 2: " + val);
    }
}
```

**Output:**

```
1 -> A
2 -> B
3 -> C
Value at key 2: B
```

## 5. Stack (thường dùng `ArrayDeque` hoặc `Stack`)

**Độ phức tạp (ArrayDeque làm stack):**

- push (addFirst): `O(1)`
- pop (removeFirst): `O(1)`
- peek (getFirst): `O(1)`

**Ví dụ:**

```java
import java.util.ArrayDeque;
import java.util.Deque;

public class StackDemo {
    public static void main(String[] args) {
        Deque<Integer> stack = new ArrayDeque<>();

        stack.push(10); // [10]
        stack.push(20); // [20, 10]
        stack.push(30); // [30, 20, 10]

        int top = stack.peek(); // 30 – O(1)
        int popped = stack.pop(); // 30 – O(1)

        System.out.println("Stack: " + stack);
        System.out.println("Top before pop: " + top);
        System.out.println("Popped: " + popped);
    }
}
```

## 6. Queue / Deque (ArrayDeque)

**Độ phức tạp (ArrayDeque):**

- Thêm cuối (offerLast/addLast): `O(1)`
- Xóa đầu (pollFirst/removeFirst): `O(1)`
- Peek đầu/cuối: `O(1)`

**Ví dụ Queue:**

```java
import java.util.ArrayDeque;
import java.util.Queue;

public class QueueDemo {
    public static void main(String[] args) {
        Queue<String> queue = new ArrayDeque<>();

        queue.offer("A");
        queue.offer("B");
        queue.offer("C"); // [A, B, C]

        String head = queue.peek(); // "A" – O(1)
        String removed = queue.poll(); // "A" – O(1)

        System.out.println("Queue: " + queue);
        System.out.println("Head before poll: " + head);
        System.out.println("Removed: " + removed);
    }
}
```

## 7. PriorityQueue (heap)

**Độ phức tạp:**

- Thêm (offer/add): `O(log n)`
- Lấy phần tử nhỏ nhất (peek): `O(1)`
- Xóa phần tử nhỏ nhất (poll): `O(log n)`

**Ví dụ:**

```java
import java.util.PriorityQueue;

public class PriorityQueueDemo {
    public static void main(String[] args) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();

        pq.offer(5);
        pq.offer(1);
        pq.offer(3); // heap bên trong

        int min = pq.peek(); // 1 – O(1)
        int removed = pq.poll(); // 1 – O(log n)

        System.out.println("PriorityQueue: " + pq);
        System.out.println("Min before poll: " + min);
        System.out.println("Removed: " + removed);
    }
}
```
