
Dưới đây là tóm tắt ngắn gọn về `java.util.concurrent` – Executors, tập trung vào `ExecutorService` và các loại thread pool chính, kèm ví dụ code và input/output.

---

## 1. ExecutorService là gì?

- Là một interface quản lý **nhóm thread** để thực thi task (Runnable/Callable).
- Giúp:
  - Tái sử dụng thread (không phải tạo/destroy liên tục).
  - Quản lý số lượng thread tối đa.
  - Lên lịch, hủy, chờ kết quả task.

Tạo `ExecutorService` thường thông qua `Executors` (factory methods).

---

## 2. newFixedThreadPool(int nThreads)

- Tạo thread pool với **số thread cố định**.
- Nếu có nhiều task hơn số thread, task sẽ xếp hàng đợi.

### Ví dụ

```java
import java.util.concurrent.*;

public class FixedThreadPoolExample {
    public static void main(String[] args) throws InterruptedException {
        ExecutorService executor = Executors.newFixedThreadPool(3);

        for (int i = 1; i <= 5; i++) {
            final int taskId = i;
            executor.submit(() -> {
                System.out.println("Task " + taskId + " chạy trên " + Thread.currentThread().getName());
            });
        }

        executor.shutdown();
    }
}
```

**Input (logic):** 5 task, pool 3 thread  
**Output (mẫu):**
```text
Task 1 chạy trên pool-1-thread-1
Task 2 chạy trên pool-1-thread-2
Task 3 chạy trên pool-1-thread-3
Task 4 chạy trên pool-1-thread-1
Task 5 chạy trên pool-1-thread-2
```
(Thứ tự có thể khác, nhưng luôn tối đa 3 thread hoạt động cùng lúc.)

## 3. newCachedThreadPool()

- Tạo thread pool **linh hoạt**:
  - Tạo thread mới khi cần.
  - Tái sử dụng thread rảnh.
  - Thread không dùng lâu sẽ bị hủy.
- Phù hợp khi có nhiều task nhỏ, ngắn, số lượng biến động.

### Ví dụ

```java
import java.util.concurrent.*;

public class CachedThreadPoolExample {
    public static void main(String[] args) {
        ExecutorService executor = Executors.newCachedThreadPool();

        for (int i = 1; i <= 5; i++) {
            final int taskId = i;
            executor.submit(() -> {
                System.out.println("Task " + taskId + " chạy trên " + Thread.currentThread().getName());
            });
        }

        executor.shutdown();
    }
}
```

**Output (mẫu):**
```text
Task 1 chạy trên pool-1-thread-1
Task 2 chạy trên pool-1-thread-2
Task 3 chạy trên pool-1-thread-3
Task 4 chạy trên pool-1-thread-4
Task 5 chạy trên pool-1-thread-5
```
(Có thể tạo nhiều thread, tùy số task và thời điểm.)

## 4. newScheduledThreadPool(int corePoolSize)

- Cho phép **lên lịch** chạy task:
  - Chạy sau một khoảng delay.
  - Chạy định kỳ (fixed rate / fixed delay).

### Ví dụ: chạy sau 2 giây, rồi chạy định kỳ mỗi 1 giây

```java
import java.util.concurrent.*;

public class ScheduledThreadPoolExample {
    public static void main(String[] args) throws InterruptedException {
        ScheduledExecutorService scheduler = Executors.newScheduledThreadPool(2);

        // Task chạy 1 lần sau 2 giây
        scheduler.schedule(() -> {
            System.out.println("Chạy 1 lần sau 2 giây");
        }, 2, TimeUnit.SECONDS);

        // Task chạy định kỳ: delay 1 giây, sau đó mỗi 1 giây
        scheduler.scheduleAtFixedRate(() -> {
            System.out.println("Task định kỳ: " + System.currentTimeMillis());
        }, 1, 1, TimeUnit.SECONDS);

        Thread.sleep(5000); // cho chương trình chạy thử 5 giây
        scheduler.shutdown();
    }
}
```

**Output (mẫu):**
```text
Task định kỳ: 1710000000000
Chạy 1 lần sau 2 giây
Task định kỳ: 1710000001000
Task định kỳ: 1710000002000
Task định kỳ: 1710000003000
...
```

## 5. ForkJoinPool

- Dùng cho mô hình **fork/join**:
  - Chia bài toán lớn thành nhiều bài toán nhỏ (fork).
  - Thực thi song song.
  - Gộp kết quả (join).
- Thường dùng với `RecursiveTask<V>` hoặc `RecursiveAction`.

### Ví dụ: tính tổng mảng bằng ForkJoinPool

```java
import java.util.concurrent.*;

class SumTask extends RecursiveTask<Long> {
    private final int[] arr;
    private final int start, end;
    private static final int THRESHOLD = 3; // ngưỡng chia nhỏ

    SumTask(int[] arr, int start, int end) {
        this.arr = arr;
        this.start = start;
        this.end = end;
    }

    @Override
    protected Long compute() {
        int length = end - start;
        if (length <= THRESHOLD) {
            long sum = 0;
            for (int i = start; i < end; i++) {
                sum += arr[i];
            }
            return sum;
        } else {
            int mid = (start + end) / 2;
            SumTask left = new SumTask(arr, start, mid);
            SumTask right = new SumTask(arr, mid, end);
            left.fork();                 // chạy left song song
            long rightResult = right.compute(); // tính right
            long leftResult = left.join();      // đợi left
            return leftResult + rightResult;
        }
    }
}

public class ForkJoinPoolExample {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5};
        ForkJoinPool pool = new ForkJoinPool();

        SumTask task = new SumTask(arr, 0, arr.length);
        long result = pool.invoke(task);

        System.out.println("Tổng = " + result);
    }
}
```

**Input:** mảng `{1, 2, 3, 4, 5}`  
**Output:**
```text
Tổng = 15
```

---

## 6. Tóm tắt ý chính

- `ExecutorService`: quản lý thread, thực thi task, hỗ trợ submit, shutdown, lấy kết quả.
- `newFixedThreadPool(n)`: số thread cố định, phù hợp tải ổn định.
- `newCachedThreadPool()`: số thread linh hoạt, phù hợp nhiều task nhỏ, ngắn.
- `newScheduledThreadPool(n)`: hỗ trợ lên lịch và chạy định kỳ.
- `ForkJoinPool`: tối ưu cho bài toán chia nhỏ – gộp lại (parallelism cao).
