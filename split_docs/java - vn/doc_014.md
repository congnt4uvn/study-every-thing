
Dưới đây là tóm tắt ngắn gọn về **Pattern Matching** trong Java (từ Java 16+), tập trung vào `instanceof` và `switch`.

---

## 1. Pattern Matching cho `instanceof`

### Ý tưởng chính
- Trước đây: phải kiểm tra `instanceof` rồi **ép kiểu** thủ công.
- Bây giờ: có thể **vừa kiểm tra vừa khai báo biến** kiểu đã ép trong cùng một câu lệnh.

### Trước Java 16 (cách cũ)

```java
Object obj = "Hello";

if (obj instanceof String) {
    String s = (String) obj;  // ép kiểu thủ công
    System.out.println(s.toUpperCase());
}
```

**Output:**
```text
HELLO
```

### Từ Java 16+ (pattern matching)

```java
Object obj = "Hello";

if (obj instanceof String s) {  // pattern matching
    System.out.println(s.toUpperCase());
}
```

**Điểm chính:**
- `obj instanceof String s`:
  - Nếu `obj` là `String`, Java tự động:
    - ép kiểu `obj` sang `String`
    - gán vào biến `s`
- Giảm lặp code, an toàn hơn, dễ đọc hơn.

## 2. Pattern Matching cho `switch` (Java 17+ / 21+)

### Ý tưởng chính
- `switch` giờ có thể:
  - Dùng với **kiểu tham chiếu** (Object, Number, …)
  - Dùng **pattern** để kiểm tra kiểu và gán biến
  - Hỗ trợ `null` trong `switch`
  - Dùng `yield` để trả về giá trị

### Ví dụ cơ bản: phân loại đối tượng

```java
static String formatObject(Object obj) {
    return switch (obj) {
        case String s      -> "Chuỗi: " + s.toUpperCase();
        case Integer i     -> "Số nguyên: " + (i * 2);
        case Double d      -> "Số thực: " + String.format("%.2f", d);
        case null          -> "Giá trị null";
        default            -> "Kiểu khác: " + obj.toString();
    };
}

public static void main(String[] args) {
    System.out.println(formatObject("hello"));
    System.out.println(formatObject(10));
    System.out.println(formatObject(3.14159));
    System.out.println(formatObject(null));
}
```

**Output:**
```text
Chuỗi: HELLO
Số nguyên: 20
Số thực: 3.14
Giá trị null
```

### Ví dụ nâng cao: pattern + điều kiện (guard)

```java
static String describeNumber(Number n) {
    return switch (n) {
        case Integer i && i > 0 -> "Số nguyên dương: " + i;
        case Integer i          -> "Số nguyên không dương: " + i;
        case Double d && d.isNaN() -> "Double: NaN";
        case Double d           -> "Double: " + d;
        case null               -> "null";
        default                 -> "Số kiểu khác: " + n;
    };
}

public static void main(String[] args) {
    System.out.println(describeNumber(5));
    System.out.println(describeNumber(-3));
    System.out.println(describeNumber(3.5));
    System.out.println(describeNumber(Double.NaN));
    System.out.println(describeNumber(null));
}
```

**Output:**
```text
Số nguyên dương: 5
Số nguyên không dương: -3
Double: 3.5
Double: NaN
null
```

## 3. Tóm tắt ý chính

1. **Pattern Matching cho `instanceof`**
   - Cú pháp: `if (obj instanceof Type var) { ... }`
   - Tự động ép kiểu và khai báo biến.
   - Code ngắn hơn, ít lỗi hơn.

2. **Pattern Matching cho `switch`**
   - `switch` trên `Object` với các pattern kiểu: `case String s -> ...`
   - Hỗ trợ `null` trong `switch`.
   - Có thể dùng điều kiện bổ sung: `case Integer i && i > 0 -> ...`
   - Thường dùng với `switch` expression để trả về giá trị.

Nếu bạn muốn, tôi có thể viết thêm ví dụ đầy đủ với `main` và hướng dẫn cách biên dịch/chạy bằng `javac` và `java`.
