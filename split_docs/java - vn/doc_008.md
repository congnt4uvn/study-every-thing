
Dưới đây là tóm tắt ngắn gọn về các Functional Interface chính trong Java 8+ và ví dụ minh họa.

---

## 1. Functional Interface là gì?

- Là interface chỉ có **1 phương thức trừu tượng** (SAM – Single Abstract Method).
- Dùng nhiều với **lambda expression** và **method reference**.
- Java 8 cung cấp sẵn nhiều functional interface trong package `java.util.function`.

---

## 2. Các Functional Interface chính

### 2.1. `Function<T, R>`
- Mô tả: Hàm nhận **1 input** kiểu `T` và trả về **1 output** kiểu `R`.
- Phương thức chính: `R apply(T t)`

**Ví dụ: chuyển String sang độ dài (Integer)**

```java
import java.util.function.Function;

public class FunctionExample {
    public static void main(String[] args) {
        Function<String, Integer> lengthFunc = s -> s.length();

        String input = "Hello";
        Integer output = lengthFunc.apply(input);

        System.out.println("Input: " + input);
        System.out.println("Output (length): " + output);
    }
}
```

**Kết quả:**
```
Input: Hello
Output (length): 5
```

---

### 2.2. `Predicate<T>`
- Mô tả: Hàm nhận **1 input** kiểu `T` và trả về **boolean** (điều kiện đúng/sai).
- Phương thức chính: `boolean test(T t)`

**Ví dụ: kiểm tra số chẵn**

```java
import java.util.function.Predicate;

public class PredicateExample {
    public static void main(String[] args) {
        Predicate<Integer> isEven = n -> n % 2 == 0;

        int input1 = 4;
        int input2 = 7;

        System.out.println("Input: " + input1 + " -> isEven: " + isEven.test(input1));
        System.out.println("Input: " + input2 + " -> isEven: " + isEven.test(input2));
    }
}
```

**Kết quả:**
```
Input: 4 -> isEven: true
Input: 7 -> isEven: false
```

---

### 2.3. `Supplier<T>`
- Mô tả: **Không nhận input**, trả về **1 giá trị** kiểu `T`.
- Phương thức chính: `T get()`
- Thường dùng để **cung cấp dữ liệu**, lazy initialization, v.v.

**Ví dụ: cung cấp một số ngẫu nhiên**

```java
import java.util.function.Supplier;
import java.util.Random;

public class SupplierExample {
    public static void main(String[] args) {
        Supplier<Integer> randomSupplier = () -> new Random().nextInt(10); // 0–9

        Integer value1 = randomSupplier.get();
        Integer value2 = randomSupplier.get();

        System.out.println("Random 1: " + value1);
        System.out.println("Random 2: " + value2);
    }
}
```

**Kết quả (ví dụ):**
```
Random 1: 3
Random 2: 8
```
(kết quả thực tế sẽ thay đổi mỗi lần chạy)

---

### 2.4. `Consumer<T>`
- Mô tả: Nhận **1 input** kiểu `T` và **không trả về gì** (void).
- Phương thức chính: `void accept(T t)`
- Thường dùng để **thực hiện hành động** (in ra, ghi log, lưu DB, ...).

**Ví dụ: in chuỗi ra màn hình**

```java
import java.util.function.Consumer;

public class ConsumerExample {
    public static void main(String[] args) {
        Consumer<String> printer = s -> System.out.println("Value: " + s);

        String input = "Java 8";
        printer.accept(input);
    }
}
```

**Kết quả:**
```
Value: Java 8
```

---

## 3. Tóm tắt nhanh

- `Function<T, R>`: T → R (có input, có output).
- `Predicate<T>`: T → boolean (dùng cho điều kiện).
- `Supplier<T>`: () → T (không input, có output).
- `Consumer<T>`: T → void (có input, không output).

Các interface này giúp:
- Viết code ngắn gọn với **lambda**.
- Dễ kết hợp với **Stream API**.
- Tăng tính **functional** trong Java 8+.
