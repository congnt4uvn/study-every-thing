
Dưới đây là tóm tắt ngắn gọn về tính năng `var` trong Java, kèm ví dụ:

---

## 1. `var` là gì?

- `var` là **từ khóa dùng cho local variable type inference** (suy luận kiểu cho biến cục bộ).
- Được giới thiệu từ **Java 10**.
- Giúp Java suy luận kiểu dữ liệu dựa trên **giá trị gán ban đầu**, nhưng **Java vẫn là ngôn ngữ kiểu tĩnh** (static type).

---

## 2. Quy tắc sử dụng `var`

- Chỉ dùng được cho **biến cục bộ**:
  - Trong method
  - Trong vòng lặp
  - Trong khối lệnh (block)
- **Bắt buộc phải khởi tạo ngay khi khai báo**.
- Không dùng được cho:
  - Tham số method
  - Field (biến instance / static)
  - Biến chưa khởi tạo

---

## 3. Ví dụ cơ bản

### Ví dụ 1: Suy luận kiểu đơn giản

```java
public class VarExample {
    public static void main(String[] args) {
        var number = 10;          // int
        var message = "Xin chào"; // String
        var pi = 3.14;            // double
        var isActive = true;      // boolean

        System.out.println(number);
        System.out.println(message);
        System.out.println(pi);
        System.out.println(isActive);
    }
}
```

**Output:**
```
10
Xin chào
3.14
true
```

---

### Ví dụ 2: Với List, Map

```java
import java.util.*;

public class VarCollectionExample {
    public static void main(String[] args) {
        var list = List.of("A", "B", "C"); // List<String>
        var map = Map.of(1, "Một", 2, "Hai"); // Map<Integer, String>

        for (var item : list) { // item là String
            System.out.println(item);
        }

        for (var entry : map.entrySet()) { // entry là Map.Entry<Integer, String>
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }
    }
}
```

**Output:**
```
A
B
C
1 -> Một
2 -> Hai
```

---

### Ví dụ 3: Trong vòng lặp

```java
public class VarLoopExample {
    public static void main(String[] args) {
        for (var i = 0; i < 3; i++) {
            System.out.println("i = " + i);
        }
    }
}
```

**Output:**
```
i = 0
i = 1
i = 2
```

---

## 4. Những điều **không được** làm với `var`

```java
public class InvalidVar {
    // var x = 10; // LỖI: không dùng var cho field

    public static void main(String[] args) {
        var x = 10;      // OK
        // var y;        // LỖI: phải khởi tạo ngay
        // var z = null; // LỖI: không suy luận được kiểu từ null
    }

    // public void test(var param) {} // LỖI: không dùng var cho tham số method
}
```

---

## 5. Khi nào nên dùng `var`?

- Khi kiểu dữ liệu **dài, phức tạp** nhưng vẫn rõ ràng từ ngữ cảnh:

```java
var map = new HashMap<String, List<Integer>>();
```

- Tránh dùng `var` nếu làm **mất rõ ràng** về kiểu, khiến code khó đọc.
