
Dưới đây là tóm tắt ngắn gọn về: **Access Modifiers, static, final, inner classes** trong Java, kèm ví dụ.

---

## 1. Access Modifiers (Các bộ truy cập)

Dùng để kiểm soát phạm vi truy cập của lớp, thuộc tính, phương thức.

### Các loại chính

1. **public**  
   - Truy cập được từ mọi nơi.
2. **private**  
   - Chỉ truy cập được trong cùng một lớp.
3. **protected**  
   - Truy cập trong cùng package hoặc lớp con (kể cả khác package).
4. (mặc định – không ghi gì, còn gọi là *package-private*)  
   - Truy cập trong cùng package.

### Ví dụ

```java
package demo;

public class Person {
    public String name;        // truy cập mọi nơi
    private int age;           // chỉ trong Person
    protected String address;  // cùng package hoặc subclass
    String phone;              // package-private

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    private void printAge() {
        System.out.println("Age: " + age);
    }
}
```

## 2. static

Từ khóa **static** gắn với lớp, không gắn với từng đối tượng.

- **Biến static**: dùng chung cho mọi đối tượng.
- **Phương thức static**: gọi mà không cần tạo đối tượng.
- **Khối static**: chạy một lần khi lớp được nạp.

### Ví dụ

```java
class Counter {
    static int count = 0; // biến static dùng chung

    public Counter() {
        count++;
    }

    static void printCount() {
        System.out.println("Count = " + count);
    }

    public static void main(String[] args) {
        new Counter();
        new Counter();
        Counter.printCount(); // gọi trực tiếp qua tên lớp
    }
}
```

**Input/Output (chạy main):**

- Tạo 2 đối tượng `Counter`
- Gọi `Counter.printCount()`

**Output:**
```text
Count = 2
```

## 3. final

Từ khóa **final** dùng để:

1. **final biến**: không thể thay đổi giá trị (hằng số).
2. **final phương thức**: không thể override trong lớp con.
3. **final lớp**: không thể bị kế thừa.

### Ví dụ

```java
class MathUtil {
    public static final double PI = 3.14159; // hằng số

    public final void printPI() {
        System.out.println("PI = " + PI);
    }
}

final class ConstantClass {
    // không thể extends lớp này
}
```

**Ví dụ dùng final biến:**

```java
public class FinalExample {
    public static void main(String[] args) {
        final int x = 10;
        System.out.println(x);
        // x = 20; // Lỗi: không thể gán lại
    }
}
```

**Output:**
```text
10
```

## 4. Inner Classes (Lớp lồng nhau)

Là lớp được khai báo bên trong một lớp khác. Dùng để nhóm logic liên quan, tăng tính đóng gói.

### Các loại chính

1. **Non-static inner class (inner class thường)**  
   - Gắn với đối tượng outer class.
2. **Static nested class**  
   - Giống lớp static, không cần đối tượng outer.
3. **Local inner class**  
   - Khai báo trong phương thức.
4. **Anonymous inner class**  
   - Lớp không tên, thường dùng để implement interface/abstract class nhanh.

---

### 4.1. Non-static inner class

```java
class Outer {
    private String message = "Hello";

    class Inner {
        void printMessage() {
            System.out.println("Message: " + message);
        }
    }

    public static void main(String[] args) {
        Outer outer = new Outer();
        Outer.Inner inner = outer.new Inner();
        inner.printMessage();
    }
}
```

**Output:**
```text
Message: Hello
```

---

### 4.2. Static nested class

```java
class Outer2 {
    static String msg = "Static nested";

    static class Nested {
        void print() {
            System.out.println("Msg: " + msg);
        }
    }

    public static void main(String[] args) {
        Outer2.Nested nested = new Outer2.Nested();
        nested.print();
    }
}
```

**Output:**
```text
Msg: Static nested
```

---

### 4.3. Anonymous inner class (ví dụ với interface)

```java
interface Greeting {
    void sayHello();
}

public class AnonymousExample {
    public static void main(String[] args) {
        Greeting g = new Greeting() {
            @Override
            public void sayHello() {
                System.out.println("Xin chào từ anonymous class!");
            }
        };

        g.sayHello();
    }
}
```

**Output:**
```text
Xin chào từ anonymous class!
```

---

## Tóm tắt nhanh

- **Access modifiers**: kiểm soát phạm vi truy cập (public, private, protected, default).
- **static**: thuộc về lớp, dùng chung, gọi không cần đối tượng.
- **final**: không thay đổi (biến), không override (phương thức), không kế thừa (lớp).
- **Inner classes**: lớp bên trong lớp (non-static, static nested, local, anonymous) giúp tổ chức mã gọn và đóng gói hơn.
