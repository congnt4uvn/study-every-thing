
Dưới đây là tóm tắt ngắn gọn về **khái niệm Auto-configuration trong Spring Boot**, kèm ví dụ mã và input/output.

---

## 1. Auto-configuration là gì?

- **Auto-configuration** là cơ chế của Spring Boot tự động cấu hình các bean trong ứng dụng dựa trên:
  - Các **thư viện** có trong classpath (ví dụ: spring-web, spring-data-jpa, spring-boot-starter-web, …)
  - Các **cấu hình** mà bạn đã khai báo (application.properties / application.yml)
  - Các **bean** mà bạn đã tự định nghĩa
- Mục tiêu: **giảm tối đa cấu hình thủ công**, giúp bạn tập trung vào logic nghiệp vụ.

---

## 2. Cách hoạt động (ý chính)

1. Khi ứng dụng Spring Boot khởi động, annotation `@SpringBootApplication` sẽ kích hoạt:
   - `@EnableAutoConfiguration` → bật cơ chế auto-configuration.
2. Spring Boot sẽ:
   - Quét các class auto-config (ví dụ: `WebMvcAutoConfiguration`, `DataSourceAutoConfiguration`, …)
   - Kiểm tra các điều kiện (`@ConditionalOnClass`, `@ConditionalOnMissingBean`, `@ConditionalOnProperty`, …)
   - Nếu điều kiện thỏa mãn → tạo bean tương ứng.
3. Nếu bạn **tự định nghĩa bean** cùng loại → auto-configuration sẽ **nhường** (không override bean của bạn).

## 3. Ví dụ cơ bản: Web application

### 3.1. Cấu trúc tối thiểu

```java
// DemoApplication.java
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication  // gồm @Configuration, @EnableAutoConfiguration, @ComponentScan
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

```java
// HelloController.java
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String hello() {
        return "Xin chào Spring Boot!";
    }
}
```

### 3.2. Cấu hình Maven (pom.xml – phần chính)

```xml
<dependencies>
    <!-- Starter web: kéo theo Spring MVC, Tomcat embedded, JSON, ... -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>

    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

### 3.3. Auto-configuration đã làm gì?

Vì bạn dùng `spring-boot-starter-web`, Spring Boot sẽ tự:

- Cấu hình **Tomcat embedded** làm web server.
- Cấu hình **Spring MVC**.
- Cấu hình **DispatcherServlet**, mapping URL, JSON converter, v.v.

Bạn **không cần** viết file `web.xml` hay cấu hình `DispatcherServlet` thủ công.

## 4. Ví dụ với DataSource (JPA + H2)

### 4.1. Thêm dependency

```xml
<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-jpa</artifactId>
    </dependency>

    <dependency>
        <groupId>com.h2database</groupId>
        <artifactId>h2</artifactId>
        <scope>runtime</scope>
    </dependency>
</dependencies>
```

### 4.2. Cấu hình đơn giản (application.properties)

```properties
spring.datasource.url=jdbc:h2:mem:testdb
spring.datasource.driverClassName=org.h2.Driver
spring.datasource.username=sa
spring.datasource.password=
spring.jpa.hibernate.ddl-auto=create-drop
spring.h2.console.enabled=true
```

Spring Boot sẽ tự:

- Tạo **DataSource**.
- Cấu hình **EntityManagerFactory**, **TransactionManager**.
- Tạo schema dựa trên entity (vì `ddl-auto=create-drop`).

### 4.3. Entity + Repository + Controller

```java
// User.java
import jakarta.persistence.Entity;
import jakarta.persistence.GeneratedValue;
import jakarta.persistence.Id;

@Entity
public class User {
    @Id
    @GeneratedValue
    private Long id;
    private String name;

    // getters/setters
}
```

```java
// UserRepository.java
import org.springframework.data.jpa.repository.JpaRepository;

public interface UserRepository extends JpaRepository<User, Long> {
}
```

```java
// UserController.java
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/users")
public class UserController {
    private final UserRepository repo;

    public UserController(UserRepository repo) {
        this.repo = repo;
    }

    @PostMapping
    public User create(@RequestBody User user) {
        return repo.save(user);
    }

    @GetMapping
    public List<User> findAll() {
        return repo.findAll();
    }
}
```

### 4.4. Ví dụ Input/Output

**Request 1 – Tạo user**

- HTTP Method: `POST`
- URL: `http://localhost:8080/users`
- Body (JSON):

```json
{
  "name": "Nguyen Van A"
}
```

**Response 1 – Output**

```json
{
  "id": 1,
  "name": "Nguyen Van A"
}
```

---

**Request 2 – Lấy danh sách user**

- HTTP Method: `GET`
- URL: `http://localhost:8080/users`

**Response 2 – Output**

```json
[
  {
    "id": 1,
    "name": "Nguyen Van A"
  }
]
```

Tất cả DataSource, JPA, transaction… đều được **auto-configure**, bạn không cần cấu hình thủ công.

## 5. Tùy chỉnh / Tắt auto-configuration

### 5.1. Tắt một auto-config cụ thể

```java
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.boot.autoconfigure.jdbc.DataSourceAutoConfiguration;

@SpringBootApplication(exclude = { DataSourceAutoConfiguration.class })
public class DemoApplication {
    // ...
}
```

### 5.2. Ghi đè bean auto-config

Nếu bạn định nghĩa bean cùng loại, Spring Boot sẽ dùng bean của bạn:

```java
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.client.RestTemplate;

@Configuration
public class MyConfig {

    @Bean
    public RestTemplate restTemplate() {
        // cấu hình custom
        return new RestTemplate();
    }
}
```

Nếu có auto-config cho `RestTemplate`, nó sẽ **không** tạo bean nữa vì đã có bean bạn định nghĩa.

---

## 6. Ý chính cần nhớ

- Auto-configuration:
  - Dựa trên **classpath**, **cấu hình**, **bean hiện có**.
  - Giúp **khởi động nhanh** ứng dụng với cấu hình mặc định hợp lý.
  - Có thể **tùy chỉnh** hoặc **tắt** khi cần.
- `@SpringBootApplication` là điểm vào chính, kích hoạt auto-configuration.
- Bạn chỉ cần:
  - Thêm **starter** phù hợp.
  - Viết **code nghiệp vụ** (controller, service, repository, entity).
  - Cấu hình **tối thiểu** trong `application.properties`/`application.yml`.
