
Here’s a concise summary of CDC (Change Data Capture) with Debezium + Kafka in a Spring Boot context, plus example code and sample messages.

---

## Key Ideas

1. **CDC (Change Data Capture)**  
   - Technique to capture changes (INSERT/UPDATE/DELETE) in a database and publish them as events.  
   - Avoids polling tables; instead, reads database logs (e.g., MySQL binlog, PostgreSQL WAL).

2. **Debezium**  
   - Open-source CDC platform that connects to databases and streams changes into Kafka topics.  
   - Runs as Kafka Connect connectors (source connectors).  
   - Each table typically maps to a Kafka topic (e.g., `dbserver1.inventory.customers`).

3. **Kafka**  
   - Acts as the event backbone.  
   - Debezium writes change events to Kafka topics.  
   - Other services (e.g., Spring Boot microservices) consume these topics to react to data changes.

4. **Spring Boot Integration**  
   - Use Spring Kafka to consume Debezium CDC events.  
   - Transform and process events (e.g., update caches, trigger workflows, build read models).  
   - Can also produce new events to other topics.

5. **Benefits**  
   - Near real-time data synchronization.  
   - Event-driven architecture without changing existing DB-based applications.  
   - Enables CQRS, materialized views, audit logs, and integration between services.

---

## Typical Architecture

1. **Database** (MySQL/PostgreSQL/etc.)  
2. **Debezium Connector** (via Kafka Connect) reads DB logs.  
3. **Kafka** receives CDC events in topics.  
4. **Spring Boot Service** consumes topics and processes events.

## Example CDC Event (Kafka Message)

When a row is inserted into `inventory.customers`:

**Input (DB operation):**

```sql
INSERT INTO customers (id, first_name, last_name, email)
VALUES (1001, 'John', 'Doe', 'john.doe@example.com');
```

**Output (Kafka message value, simplified Debezium format):**

```json
{
  "before": null,
  "after": {
    "id": 1001,
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com"
  },
  "source": {
    "db": "inventory",
    "table": "customers",
    "ts_ms": 1710000000000
  },
  "op": "c",          // c = create, u = update, d = delete
  "ts_ms": 1710000000100
}
```

## Spring Boot: Kafka Consumer Configuration

```java
import org.apache.kafka.clients.consumer.ConsumerConfig;
import org.apache.kafka.common.serialization.StringDeserializer;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.kafka.config.ConcurrentKafkaListenerContainerFactory;
import org.springframework.kafka.core.ConsumerFactory;
import org.springframework.kafka.core.DefaultKafkaConsumerFactory;

import java.util.HashMap;
import java.util.Map;

@Configuration
public class KafkaConsumerConfig {

    @Bean
    public ConsumerFactory<String, String> consumerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ConsumerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ConsumerConfig.GROUP_ID_CONFIG, "cdc-consumer-group");
        props.put(ConsumerConfig.KEY_DESERIALIZER_CLASS_CONFIG, StringDeserializer.class);
        props.put(ConsumerConfig.VALUE_DESERIALIZER_CLASS_CONFIG, StringDeserializer.class);
        return new DefaultKafkaConsumerFactory<>(props);
    }

    @Bean
    public ConcurrentKafkaListenerContainerFactory<String, String> kafkaListenerContainerFactory() {
        ConcurrentKafkaListenerContainerFactory<String, String> factory =
                new ConcurrentKafkaListenerContainerFactory<>();
        factory.setConsumerFactory(consumerFactory());
        return factory;
    }
}
```

## Spring Boot: CDC Event Listener

```java
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.springframework.kafka.annotation.KafkaListener;
import org.springframework.stereotype.Service;

@Service
public class CustomerCdcListener {

    private final ObjectMapper objectMapper = new ObjectMapper();

    @KafkaListener(topics = "dbserver1.inventory.customers", groupId = "cdc-consumer-group")
    public void onMessage(String message) throws Exception {
        JsonNode root = objectMapper.readTree(message);
        String op = root.get("op").asText(); // c, u, d

        JsonNode after = root.get("after");
        JsonNode before = root.get("before");

        if ("c".equals(op)) {
            // handle insert
            System.out.println("New customer: " + after);
        } else if ("u".equals(op)) {
            // handle update
            System.out.println("Updated customer from " + before + " to " + after);
        } else if ("d".equals(op)) {
            // handle delete
            System.out.println("Deleted customer: " + before);
        }
    }
}
```

**Example console output for the insert above:**

```text
New customer: {"id":1001,"first_name":"John","last_name":"Doe","email":"john.doe@example.com"}
```
