
Dưới đây là tóm tắt ngắn gọn về các khái niệm cơ bản PL/SQL: Anonymous Block, Procedure, Function, kèm ví dụ.

---

## 1. Anonymous Block (Khối ẩn danh)

- Là khối PL/SQL **không có tên**, thường dùng để:
  - Thử nghiệm nhanh logic
  - Chạy một lần, không lưu trong database
- Cấu trúc cơ bản:

```plsql
DECLARE
    v_message VARCHAR2(50);
BEGIN
    v_message := 'Xin chào PL/SQL';
    DBMS_OUTPUT.PUT_LINE(v_message);
EXCEPTION
    WHEN OTHERS THEN
        DBMS_OUTPUT.PUT_LINE('Có lỗi xảy ra: ' || SQLERRM);
END;
/
```

**Input:** không có tham số vào, chỉ chạy trực tiếp.  
**Output (DBMS_OUTPUT):**
```text
Xin chào PL/SQL
```

---

## 2. Procedure (Thủ tục)

- Là khối PL/SQL **có tên**, được lưu trong database.
- Dùng để **thực hiện một hành động** (insert, update, xử lý nghiệp vụ…).
- Có thể có tham số IN, OUT, IN OUT.
- Không bắt buộc phải trả về giá trị (khác với function).

### Ví dụ: Procedure chèn nhân viên mới

```plsql
CREATE OR REPLACE PROCEDURE add_employee (
    p_emp_id    IN  NUMBER,
    p_emp_name  IN  VARCHAR2,
    p_salary    IN  NUMBER
) AS
BEGIN
    INSERT INTO employees (emp_id, emp_name, salary)
    VALUES (p_emp_id, p_emp_name, p_salary);

    DBMS_OUTPUT.PUT_LINE('Đã thêm nhân viên: ' || p_emp_name);
EXCEPTION
    WHEN OTHERS THEN
        DBMS_OUTPUT.PUT_LINE('Lỗi khi thêm nhân viên: ' || SQLERRM);
END;
/
```

**Gọi procedure (input):**

```plsql
BEGIN
    add_employee(101, 'Nguyen Van A', 1500);
END;
/
```

**Output (DBMS_OUTPUT):**
```text
Đã thêm nhân viên: Nguyen Van A
```

## 3. Function (Hàm)

- Là khối PL/SQL **có tên**, được lưu trong database.
- **Bắt buộc** phải trả về một giá trị (kiểu dữ liệu cụ thể).
- Thường dùng để:
  - Tính toán và trả về kết quả
  - Có thể dùng trong câu lệnh SQL (SELECT, WHERE, …) nếu thỏa điều kiện.

### Ví dụ: Function tính lương sau thuế

```plsql
CREATE OR REPLACE FUNCTION calc_net_salary (
    p_gross_salary IN NUMBER
) RETURN NUMBER AS
    v_tax_rate   NUMBER := 0.1;  -- 10% thuế
    v_net_salary NUMBER;
BEGIN
    v_net_salary := p_gross_salary * (1 - v_tax_rate);
    RETURN v_net_salary;
EXCEPTION
    WHEN OTHERS THEN
        DBMS_OUTPUT.PUT_LINE('Lỗi khi tính lương: ' || SQLERRM);
        RETURN NULL;
END;
/
```

**Gọi function trong PL/SQL (input/output):**

```plsql
DECLARE
    v_result NUMBER;
BEGIN
    v_result := calc_net_salary(2000);
    DBMS_OUTPUT.PUT_LINE('Lương thực nhận: ' || v_result);
END;
/
```

**Output:**
```text
Lương thực nhận: 1800
```

**Gọi function trong SQL:**

```sql
SELECT calc_net_salary(3000) AS net_salary
FROM dual;
```

**Kết quả:**

| NET_SALARY |
|-----------:|
|       2700 |

---

## 4. So sánh nhanh

| Loại        | Có tên? | Lưu trong DB? | Trả về giá trị? | Dùng trong SQL?         |
|------------|---------|---------------|------------------|-------------------------|
| Anonymous  | Không   | Không         | Không bắt buộc   | Không                   |
| Procedure  | Có      | Có            | Không bắt buộc   | Không (chỉ gọi trong PL/SQL) |
| Function   | Có      | Có            | Bắt buộc         | Có thể (nếu phù hợp)    |

Nếu bạn muốn, tôi có thể thêm ví dụ có tham số OUT/IN OUT hoặc ví dụ xử lý EXCEPTION chi tiết hơn.
