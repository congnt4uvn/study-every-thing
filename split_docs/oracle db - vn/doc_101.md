
Dưới đây là tóm tắt ngắn gọn về nội dung “Common Tools: Using SQL*Loader or external tables for bulk load” trong Oracle, kèm ví dụ.

---

## Ý chính

- **Mục tiêu**: Nạp (import) dữ liệu số lượng lớn (bulk load) vào Oracle Database.
- **Hai công cụ phổ biến**:
  1. **SQL\*Loader**  
     - Công cụ dòng lệnh (command-line) chuyên dùng để nạp dữ liệu từ file (thường là CSV, text) vào bảng Oracle.  
     - Hỗ trợ nhiều tùy chọn: kiểm soát định dạng file, ánh xạ cột, xử lý lỗi, hiệu năng cao (direct path load).
  2. **External Tables (Bảng ngoài)**  
     - Cho phép định nghĩa một bảng “ảo” trỏ trực tiếp đến file dữ liệu bên ngoài (trên filesystem).  
     - Có thể dùng câu lệnh SQL (SELECT, INSERT) để đọc và nạp dữ liệu như bảng bình thường.  
     - Dễ tích hợp với các thao tác SQL, thuận tiện cho ETL.

---

## 1. Ví dụ SQL*Loader

### 1.1. File dữ liệu (data.csv)

```text
1,Nguyen Van A,5000
2,Tran Thi B,6000
3,Le Van C,7000
```

### 1.2. Bảng đích trong Oracle

```sql
CREATE TABLE employees (
    emp_id    NUMBER,
    emp_name  VARCHAR2(100),
    salary    NUMBER
);
```

### 1.3. File control (load.ctl)

```text
LOAD DATA
INFILE 'data.csv'
INTO TABLE employees
FIELDS TERMINATED BY ','
(
  emp_id    INTEGER EXTERNAL,
  emp_name  CHAR,
  salary    INTEGER EXTERNAL
)
```

### 1.4. Lệnh chạy SQL*Loader (input)

```bash
sqlldr userid=user/password@ORCL control=load.ctl log=load.log bad=load.bad
```

### 1.5. Kết quả (output – dữ liệu trong bảng)

```sql
SELECT * FROM employees;

-- Kết quả:
-- EMP_ID | EMP_NAME      | SALARY
--   1    | Nguyen Van A  | 5000
--   2    | Tran Thi B    | 6000
--   3    | Le Van C      | 7000
```

## 2. Ví dụ External Table

### 2.1. File dữ liệu (data_ext.csv)

```text
1,Nguyen Van A,5000
2,Tran Thi B,6000
3,Le Van C,7000
```

### 2.2. Tạo DIRECTORY trỏ đến thư mục chứa file

```sql
CREATE OR REPLACE DIRECTORY data_dir AS '/u01/data';
GRANT READ, WRITE ON DIRECTORY data_dir TO user;
```

### 2.3. Tạo external table

```sql
CREATE TABLE employees_ext (
    emp_id    NUMBER,
    emp_name  VARCHAR2(100),
    salary    NUMBER
)
ORGANIZATION EXTERNAL (
    TYPE ORACLE_LOADER
    DEFAULT DIRECTORY data_dir
    ACCESS PARAMETERS (
        RECORDS DELIMITED BY NEWLINE
        FIELDS TERMINATED BY ','
        ( emp_id, emp_name, salary )
    )
    LOCATION ('data_ext.csv')
)
REJECT LIMIT UNLIMITED;
```

### 2.4. Đọc dữ liệu từ external table (input)

```sql
SELECT * FROM employees_ext;
```

### 2.5. Kết quả (output)

```text
EMP_ID | EMP_NAME      | SALARY
  1    | Nguyen Van A  | 5000
  2    | Tran Thi B    | 6000
  3    | Le Van C      | 7000
```

### 2.6. Nạp dữ liệu từ external table vào bảng thường

```sql
INSERT INTO employees (emp_id, emp_name, salary)
SELECT emp_id, emp_name, salary
FROM employees_ext;

COMMIT;
```

---

## So sánh nhanh

- **SQL\*Loader**:  
  - Mạnh về hiệu năng, nhiều tùy chọn nạp dữ liệu.  
  - Chạy qua command-line, dùng file control.

- **External Tables**:  
  - Dễ dùng với SQL, thuận tiện cho xử lý/kiểm tra dữ liệu trước khi nạp.  
  - Dữ liệu vẫn nằm trong file, chỉ “đọc” qua bảng ngoài; muốn lưu trong DB thì INSERT vào bảng thường.
