
Dưới đây là tóm tắt ngắn gọn về các công cụ thường dùng với Oracle Database, kèm ví dụ code SQL (có input/output).

---

## 1. SQL Developer
- Công cụ miễn phí của Oracle.
- Dùng để:
  - Kết nối và quản lý Oracle Database.
  - Viết, chạy, debug SQL và PL/SQL.
  - Xem cấu trúc bảng, index, view, sequence, v.v.
  - Xuất dữ liệu (CSV, Excel, SQL script).

**Ví dụ truy vấn trong SQL Developer**

```sql
-- Input: chạy câu lệnh SQL này trong SQL Developer
SELECT employee_id, first_name, last_name, salary
FROM employees
WHERE department_id = 10;
```

**Output (kết quả dạng bảng)**

| EMPLOYEE_ID | FIRST_NAME | LAST_NAME | SALARY |
|------------:|------------|-----------|--------|
| 200         | Jennifer   | Whalen    | 4400   |

---

## 2. TOAD (Toad for Oracle)
- Công cụ thương mại, mạnh cho DBA và developer.
- Dùng để:
  - Tối ưu hiệu năng truy vấn (Explain Plan, Autotrace).
  - So sánh schema, tạo script deploy.
  - Quản trị database (user, quyền, backup, v.v.).

**Ví dụ xem kế hoạch thực thi (Explain Plan)**

```sql
EXPLAIN PLAN FOR
SELECT * FROM employees WHERE last_name = 'King';

SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```

**Output (rút gọn)**

| Id | Operation         | Name      |
|----|-------------------|-----------|
|  0 | SELECT STATEMENT  |           |
|  1 |  TABLE ACCESS FULL| EMPLOYEES |

---

## 3. PL/SQL Developer
- Công cụ chuyên cho lập trình PL/SQL.
- Dùng để:
  - Viết, debug, test package, procedure, function.
  - Quản lý version code PL/SQL.
  - Chạy test unit cho PL/SQL.

**Ví dụ tạo procedure trong PL/SQL Developer**

```sql
-- Input: tạo procedure
CREATE OR REPLACE PROCEDURE raise_salary (
    p_emp_id   IN employees.employee_id%TYPE,
    p_percent  IN NUMBER
) AS
BEGIN
    UPDATE employees
    SET salary = salary * (1 + p_percent / 100)
    WHERE employee_id = p_emp_id;
END;
/

-- Gọi procedure (test)
BEGIN
    raise_salary(p_emp_id => 200, p_percent => 10);
END;
/
```

**Output (kết quả)**  
- Lương của nhân viên `employee_id = 200` tăng 10%.  
- Có thể kiểm tra lại:

```sql
SELECT employee_id, salary
FROM employees
WHERE employee_id = 200;
```

---

### Tóm tắt ý chính
- **SQL Developer**: miễn phí, phổ biến, phù hợp hầu hết developer Oracle.
- **TOAD**: mạnh cho DBA và tối ưu hiệu năng, nhiều tính năng nâng cao.
- **PL/SQL Developer**: tập trung vào phát triển và debug PL/SQL.
