
Dưới đây là tóm tắt ngắn gọn về **Pipelined Table Functions** trong Oracle PL/SQL, kèm ví dụ mã và input/output.

---

## 1. Khái niệm chính

- **Pipelined table function** là hàm PL/SQL trả về **tập hợp (collection)** nhưng cho phép **trả từng dòng dần dần (streaming)** cho SQL, thay vì đợi xử lý xong toàn bộ.
- Dùng trong mệnh đề `FROM` như một **bảng ảo**.
- Giúp:
  - Xử lý dữ liệu lớn hiệu quả hơn (ít dùng bộ nhớ).
  - Cho phép **song song** (parallel) trong một số trường hợp.
  - Dễ dàng viết logic xử lý phức tạp trong PL/SQL rồi dùng trong SQL.

---

## 2. Cấu trúc cơ bản

### 2.1. Định nghĩa kiểu dữ liệu

```sql
-- Kiểu object cho 1 dòng
CREATE OR REPLACE TYPE t_emp_row AS OBJECT (
    emp_id    NUMBER,
    emp_name  VARCHAR2(100),
    salary    NUMBER
);
/

-- Kiểu bảng (collection) của object trên
CREATE OR REPLACE TYPE t_emp_tab AS TABLE OF t_emp_row;
/
```

### 2.2. Định nghĩa pipelined function

```sql
CREATE OR REPLACE FUNCTION f_get_emps (p_min_salary NUMBER)
  RETURN t_emp_tab
  PIPELINED
IS
BEGIN
  FOR r IN (
    SELECT employee_id, first_name || ' ' || last_name AS emp_name, salary
    FROM   employees
    WHERE  salary >= p_min_salary
  )
  LOOP
    PIPE ROW ( t_emp_row(r.employee_id, r.emp_name, r.salary) );
  END LOOP;

  RETURN;
END;
/
```

- Từ khóa quan trọng:
  - `PIPELINED` trong phần khai báo hàm.
  - `PIPE ROW (...)` để “đẩy” từng dòng ra ngoài.

---

## 3. Cách sử dụng trong SQL

### 3.1. Gọi như một bảng

```sql
SELECT *
FROM   TABLE(f_get_emps(5000));
```

**Input**: `p_min_salary = 5000`  
**Output (ví dụ)**:

| EMP_ID | EMP_NAME        | SALARY |
|--------|-----------------|--------|
| 101    | Neena Kochhar   | 17000  |
| 102    | Lex De Haan     | 17000  |
| 103    | Alexander Hunold| 9000   |
| ...    | ...             | ...    |

## 4. Ví dụ nâng cao: xử lý chuỗi CSV

### 4.1. Kiểu dữ liệu

```sql
CREATE OR REPLACE TYPE t_num_row AS OBJECT (
    val NUMBER
);
/

CREATE OR REPLACE TYPE t_num_tab AS TABLE OF t_num_row;
/
```

### 4.2. Hàm pipelined tách chuỗi

```sql
CREATE OR REPLACE FUNCTION f_split_numbers (p_csv IN VARCHAR2)
  RETURN t_num_tab
  PIPELINED
IS
  l_str   VARCHAR2(32767) := p_csv || ',';
  l_pos   PLS_INTEGER;
  l_piece VARCHAR2(100);
BEGIN
  LOOP
    l_pos := INSTR(l_str, ',');
    EXIT WHEN l_pos = 0;

    l_piece := TRIM(SUBSTR(l_str, 1, l_pos - 1));
    l_str   := SUBSTR(l_str, l_pos + 1);

    IF l_piece IS NOT NULL THEN
      PIPE ROW ( t_num_row(TO_NUMBER(l_piece)) );
    END IF;
  END LOOP;

  RETURN;
END;
/
```

### 4.3. Gọi hàm

```sql
SELECT *
FROM   TABLE(f_split_numbers('10,20,30, 40 ,50'));
```

**Input**: `'10,20,30, 40 ,50'`  
**Output**:

| VAL |
|-----|
| 10  |
| 20  |
| 30  |
| 40  |
| 50  |

---

## 5. Điểm cần nhớ

- Pipelined function:
  - Trả về `TABLE OF <object_type>`.
  - Dùng `PIPELINED` + `PIPE ROW`.
  - Gọi trong SQL: `SELECT ... FROM TABLE(function_name(...))`.
- Thích hợp cho:
  - Xử lý dữ liệu lớn, streaming.
  - Chuyển đổi/biến đổi dữ liệu phức tạp trong PL/SQL rồi dùng trong SQL.
