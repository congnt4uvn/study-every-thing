
Dưới đây là tóm tắt ngắn gọn về các lệnh SQL cơ bản trong Oracle: `SELECT`, `INSERT`, `UPDATE`, `DELETE`, `MERGE` kèm ví dụ.

---

## 1. SELECT – Truy vấn dữ liệu

Dùng để lấy dữ liệu từ bảng.

**Cú pháp cơ bản:**
```sql
SELECT danh_sach_cot
FROM ten_bang
WHERE dieu_kien
ORDER BY cot1 [ASC|DESC];
```

**Ví dụ:**
```sql
-- Bảng: EMPLOYEES(emp_id, first_name, last_name, salary, department_id)

SELECT emp_id, first_name, last_name, salary
FROM employees
WHERE salary > 10000
ORDER BY salary DESC;
```

**Kết quả (output ví dụ):**
```text
EMP_ID | FIRST_NAME | LAST_NAME | SALARY
----------------------------------------
101    | John       | Smith     | 15000
102    | Mary       | Brown     | 12000
```

---

## 2. INSERT – Thêm dữ liệu

Dùng để chèn (thêm) bản ghi mới vào bảng.

**Cú pháp 1 – Chèn đầy đủ cột:**
```sql
INSERT INTO ten_bang
VALUES (giatri1, giatri2, ...);
```

**Cú pháp 2 – Chỉ định cột:**
```sql
INSERT INTO ten_bang (cot1, cot2, ...)
VALUES (giatri1, giatri2, ...);
```

**Ví dụ:**
```sql
INSERT INTO employees (emp_id, first_name, last_name, salary, department_id)
VALUES (201, 'Anh', 'Nguyen', 18000, 10);
```

**Kiểm tra:**
```sql
SELECT * FROM employees WHERE emp_id = 201;
```

**Output ví dụ:**
```text
EMP_ID | FIRST_NAME | LAST_NAME | SALARY | DEPARTMENT_ID
--------------------------------------------------------
201    | Anh        | Nguyen    | 18000  | 10
```

---

## 3. UPDATE – Cập nhật dữ liệu

Dùng để sửa dữ liệu đã tồn tại.

**Cú pháp:**
```sql
UPDATE ten_bang
SET cot1 = giatri1,
    cot2 = giatri2
WHERE dieu_kien;
```

**Ví dụ:**
```sql
UPDATE employees
SET salary = salary * 1.1
WHERE department_id = 10;
```

**Kiểm tra:**
```sql
SELECT emp_id, salary
FROM employees
WHERE department_id = 10;
```

**Output ví dụ (sau khi tăng 10%):**
```text
EMP_ID | SALARY
---------------
201    | 19800
...    | ...
```

> Lưu ý: Luôn dùng `WHERE`. Nếu bỏ `WHERE`, toàn bộ bảng sẽ bị cập nhật.

---

## 4. DELETE – Xóa dữ liệu

Dùng để xóa bản ghi khỏi bảng.

**Cú pháp:**
```sql
DELETE FROM ten_bang
WHERE dieu_kien;
```

**Ví dụ:**
```sql
DELETE FROM employees
WHERE emp_id = 201;
```

**Kiểm tra:**
```sql
SELECT * FROM employees WHERE emp_id = 201;
```

**Output ví dụ:**
```text
no rows selected
```

> Lưu ý: `DELETE FROM ten_bang;` (không WHERE) sẽ xóa toàn bộ dữ liệu trong bảng.

---

## 5. MERGE – Đồng bộ (Upsert) dữ liệu

Dùng để **chèn mới** hoặc **cập nhật** dữ liệu dựa trên điều kiện khớp giữa bảng nguồn và bảng đích. Rất hay dùng trong ETL, đồng bộ dữ liệu.

**Cú pháp tổng quát:**
```sql
MERGE INTO bang_dich d
USING bang_nguon s
   ON (d.khoa_chinh = s.khoa_chinh)
WHEN MATCHED THEN
    UPDATE SET d.cot1 = s.cot1,
               d.cot2 = s.cot2
WHEN NOT MATCHED THEN
    INSERT (cot1, cot2, cot3)
    VALUES (s.cot1, s.cot2, s.cot3);
```

**Ví dụ:**

Giả sử:

- Bảng đích: `employees`  
- Bảng nguồn tạm: `employees_stage`

```sql
MERGE INTO employees e
USING employees_stage s
   ON (e.emp_id = s.emp_id)
WHEN MATCHED THEN
    UPDATE SET e.salary       = s.salary,
               e.department_id = s.department_id
WHEN NOT MATCHED THEN
    INSERT (emp_id, first_name, last_name, salary, department_id)
    VALUES (s.emp_id, s.first_name, s.last_name, s.salary, s.department_id);
```

**Ý nghĩa:**
- Nếu `emp_id` đã tồn tại trong `employees` → cập nhật lương và phòng ban.
- Nếu `emp_id` chưa tồn tại → chèn bản ghi mới.

---

## Tóm tắt ý chính

- `SELECT`: Lấy dữ liệu (có thể lọc bằng `WHERE`, sắp xếp bằng `ORDER BY`).
- `INSERT`: Thêm bản ghi mới vào bảng.
- `UPDATE`: Sửa dữ liệu hiện có (cần `WHERE` để tránh cập nhật toàn bộ).
- `DELETE`: Xóa bản ghi (cần `WHERE` để tránh xóa toàn bộ).
- `MERGE`: Kết hợp `INSERT` + `UPDATE` trong một lệnh, dùng để đồng bộ dữ liệu giữa hai nguồn.

Nếu bạn cho thêm cấu trúc bảng cụ thể (DDL), tôi có thể viết bộ ví dụ đầy đủ hơn với dữ liệu mẫu và kết quả chi tiết.
