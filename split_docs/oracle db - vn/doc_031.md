
Dưới đây là tóm tắt ngắn gọn về **EXECUTE IMMEDIATE** trong PL/SQL (Oracle), kèm ví dụ mã và input/output.

---

## 1. EXECUTE IMMEDIATE là gì?

- Là câu lệnh dùng để thực thi **SQL động** (Dynamic SQL) trong PL/SQL.
- Cho phép:
  - Xây dựng câu SQL dưới dạng chuỗi (string) tại thời điểm chạy.
  - Thực thi các lệnh DDL (CREATE, ALTER, DROP), DML (INSERT, UPDATE, DELETE), hoặc SELECT động.
  - Dùng khi:
    - Tên bảng/cột chỉ biết tại runtime.
    - Cần tạo/đổi cấu trúc bảng động.
    - Cần linh hoạt trong điều kiện WHERE, ORDER BY, v.v.

Cú pháp cơ bản:

```plsql
EXECUTE IMMEDIATE <chuỗi_sql> [INTO ...] [USING ...];
```

---

## 2. Ví dụ DDL động (CREATE TABLE, DROP TABLE)

### Ví dụ 1: Tạo bảng động

**Input**: Tên bảng `'EMP_TEMP'`  
**Mục tiêu**: Tạo bảng EMP_TEMP với 2 cột.

```plsql
DECLARE
    v_table_name  VARCHAR2(30) := 'EMP_TEMP';
    v_sql         VARCHAR2(4000);
BEGIN
    v_sql := 'CREATE TABLE ' || v_table_name || ' (
                 emp_id   NUMBER,
                 emp_name VARCHAR2(100)
              )';

    EXECUTE IMMEDIATE v_sql;
END;
/
```

**Output**: Bảng `EMP_TEMP` được tạo trong schema hiện tại.

---

### Ví dụ 2: Xóa bảng động

```plsql
DECLARE
    v_table_name VARCHAR2(30) := 'EMP_TEMP';
BEGIN
    EXECUTE IMMEDIATE 'DROP TABLE ' || v_table_name;
END;
/
```

**Output**: Bảng `EMP_TEMP` bị xóa (nếu tồn tại).

---

## 3. DML động (INSERT, UPDATE, DELETE)

### Ví dụ 3: INSERT động với USING

**Input**:  
- Bảng: `employees(emp_id, emp_name)`  
- Giá trị: `101, 'An Nguyen'`

```plsql
DECLARE
    v_sql   VARCHAR2(4000);
    v_id    NUMBER := 101;
    v_name  VARCHAR2(100) := 'An Nguyen';
BEGIN
    v_sql := 'INSERT INTO employees (emp_id, emp_name)
              VALUES (:1, :2)';

    EXECUTE IMMEDIATE v_sql
        USING v_id, v_name;
END;
/
```

**Output**:  
Dòng mới được chèn vào bảng `employees`:

```text
EMP_ID | EMP_NAME
--------------------
101    | An Nguyen
```

---

### Ví dụ 4: UPDATE động với điều kiện linh hoạt

**Input**:  
- Bảng: `employees`  
- Cột cập nhật: `salary`  
- Điều kiện: `department_id = 10`  
- Giá trị mới: `5000`

```plsql
DECLARE
    v_sql   VARCHAR2(4000);
    v_dept  NUMBER := 10;
    v_sal   NUMBER := 5000;
BEGIN
    v_sql := 'UPDATE employees
              SET salary = :1
              WHERE department_id = :2';

    EXECUTE IMMEDIATE v_sql
        USING v_sal, v_dept;
END;
/
```

**Output**:  
Tất cả nhân viên có `department_id = 10` được cập nhật `salary = 5000`.

## 4. SELECT động với INTO

### Ví dụ 5: SELECT một giá trị

**Input**:  
- Bảng: `employees`  
- `emp_id = 101`  
- Cần lấy `emp_name`

```plsql
DECLARE
    v_sql      VARCHAR2(4000);
    v_emp_id   NUMBER := 101;
    v_emp_name VARCHAR2(100);
BEGIN
    v_sql := 'SELECT emp_name
              FROM employees
              WHERE emp_id = :1';

    EXECUTE IMMEDIATE v_sql
        INTO v_emp_name
        USING v_emp_id;

    DBMS_OUTPUT.PUT_LINE('Tên nhân viên: ' || v_emp_name);
END;
/
```

**Output (DBMS_OUTPUT)**:

```text
Tên nhân viên: An Nguyen
```

---

### Ví dụ 6: SELECT nhiều cột vào record

```plsql
DECLARE
    TYPE emp_rec_type IS RECORD (
        emp_id   employees.emp_id%TYPE,
        emp_name employees.emp_name%TYPE
    );
    v_emp   emp_rec_type;
    v_sql   VARCHAR2(4000);
BEGIN
    v_sql := 'SELECT emp_id, emp_name
              FROM employees
              WHERE emp_id = :1';

    EXECUTE IMMEDIATE v_sql
        INTO v_emp.emp_id, v_emp.emp_name
        USING 101;

    DBMS_OUTPUT.PUT_LINE('ID: ' || v_emp.emp_id ||
                         ', Name: ' || v_emp.emp_name);
END;
/
```

**Output**:

```text
ID: 101, Name: An Nguyen
```

---

## 5. SELECT nhiều dòng: dùng BULK COLLECT

### Ví dụ 7: Lấy nhiều dòng vào collection

```plsql
DECLARE
    TYPE emp_tab_type IS TABLE OF employees.emp_name%TYPE;
    v_names emp_tab_type;
    v_sql   VARCHAR2(4000);
BEGIN
    v_sql := 'SELECT emp_name
              FROM employees
              WHERE department_id = :1';

    EXECUTE IMMEDIATE v_sql
        BULK COLLECT INTO v_names
        USING 10;

    FOR i IN 1 .. v_names.COUNT LOOP
        DBMS_OUTPUT.PUT_LINE('Name: ' || v_names(i));
    END LOOP;
END;
/
```

**Output**:  
In ra danh sách tên nhân viên thuộc `department_id = 10`.

## 6. USING, INTO, RETURNING

- `USING`: truyền giá trị vào các bind variable `:1, :2, ...`.
- `INTO`: nhận kết quả SELECT (1 dòng).
- `RETURNING INTO`: lấy giá trị sau DML (INSERT/UPDATE/DELETE).

### Ví dụ 8: INSERT với RETURNING INTO

```plsql
DECLARE
    v_sql   VARCHAR2(4000);
    v_id    NUMBER;
BEGIN
    v_sql := 'INSERT INTO employees (emp_id, emp_name)
              VALUES (employees_seq.NEXTVAL, :1)
              RETURNING emp_id INTO :2';

    EXECUTE IMMEDIATE v_sql
        USING 'Binh Tran'
        RETURNING INTO v_id;

    DBMS_OUTPUT.PUT_LINE('New ID: ' || v_id);
END;
/
```

**Output**:

```text
New ID: <giá trị mới từ sequence>
```

---

## 7. Lưu ý quan trọng

1. **Bảo mật**:  
   - Tránh nối chuỗi trực tiếp từ input người dùng → dễ bị SQL Injection.  
   - Dùng bind variable (`:1, :2, ...`) với `USING`.

2. **Hiệu năng**:  
   - SQL động không được tối ưu tốt như SQL tĩnh nếu lạm dụng.  
   - Dùng khi thực sự cần linh hoạt (tên bảng, cột, điều kiện phức tạp).

3. **Quyền (Privileges)**:  
   - Lệnh DDL/DML động vẫn cần quyền tương ứng (CREATE, ALTER, SELECT, v.v.).

---

Tóm lại, **EXECUTE IMMEDIATE** là công cụ chính để viết **PL/SQL nâng cao với SQL động**, cho phép tạo/chỉnh sửa cấu trúc, thao tác dữ liệu, và truy vấn linh hoạt dựa trên tham số runtime.
