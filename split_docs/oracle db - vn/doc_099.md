
Dưới đây là tóm tắt ngắn gọn về các công cụ thường dùng với Oracle Database: **SQL\*Plus** và **SQLcl**, kèm ví dụ.

---

## 1. SQL\*Plus

**SQL\*Plus** là công cụ dòng lệnh truyền thống của Oracle để:
- Kết nối tới Oracle Database  
- Chạy câu lệnh SQL, PL/SQL  
- Chạy script `.sql`  
- Xuất kết quả ra màn hình hoặc file  

### Cách kết nối

```bash
sqlplus username/password@hostname:1521/servicename
```

**Ví dụ:**

```bash
sqlplus scott/tiger@localhost:1521/orclpdb
```

### Ví dụ câu lệnh SQL

**Input (trong SQL\*Plus):**
```sql
SELECT empno, ename, sal
FROM   emp
WHERE  deptno = 10;
```

**Output (màn hình):**
```text
     EMPNO ENAME          SAL
---------- ---------- -------
      7782 CLARK         2450
      7839 KING          5000
      7934 MILLER        1300
```

### Chạy script

```bash
sqlplus scott/tiger@orclpdb @script.sql
```

---

## 2. SQLcl (SQL Command Line)

**SQLcl** là công cụ dòng lệnh mới hơn của Oracle, thay thế/ bổ sung cho SQL\*Plus, với:
- Cú pháp tương thích phần lớn với SQL\*Plus  
- Tính năng hiện đại hơn: autocomplete, history, định dạng JSON, hỗ trợ scripting tốt hơn  
- Hỗ trợ các lệnh tiện ích (ví dụ: `info`, `ddl`, `history`, …)

### Cách kết nối

```bash
sql /nolog
SQL> connect scott/tiger@localhost:1521/orclpdb
```

Hoặc trực tiếp:

```bash
sql scott/tiger@localhost:1521/orclpdb
```

### Ví dụ truy vấn trong SQLcl

**Input:**
```sql
SELECT empno, ename, sal
FROM   emp
FETCH FIRST 3 ROWS ONLY;
```

**Output (được SQLcl format đẹp hơn):**
```text
EMPNO  ENAME   SAL
-----  ------  ----
7782   CLARK   2450
7839   KING    5000
7934   MILLER  1300
```

### Một số lệnh tiện ích của SQLcl

```sql
-- Xem DDL của một bảng
ddl emp;

-- Xem thông tin đối tượng
info emp;

-- Xem lịch sử lệnh
history;
```

---

## Tóm tắt chính

- **SQL\*Plus**: công cụ dòng lệnh cổ điển, nhẹ, có sẵn, dùng rộng rãi cho script batch, admin.  
- **SQLcl**: công cụ dòng lệnh hiện đại hơn, tương thích SQL\*Plus nhưng thêm nhiều tính năng tiện lợi (autocomplete, history, lệnh tiện ích, format kết quả tốt hơn).  
- Cả hai đều dùng để kết nối Oracle Database và thực thi SQL/PLSQL, chạy script `.sql`.
