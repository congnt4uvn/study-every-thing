
Here’s a concise summary of Java *text blocks* (a newer Java feature), with examples.

---

## What are text blocks?

Text blocks are a way to write multi-line string literals more easily and readably.

- Introduced as a standard feature in **Java 15** (preview in 13/14).
- Written with **three double quotes** `"""`.
- Preserve line breaks and most formatting.
- Reduce the need for many `\n` and `\"` escape sequences.

---

## Basic syntax

```java
String text = """
        This is a text block.
        It can span multiple lines.
        "Quotes" don't need escaping.
        """;
```

Equivalent old-style string:

```java
String text = "This is a text block.\n" +
              "It can span multiple lines.\n" +
              "\"Quotes\" don't need escaping.\n";
```

---

## Indentation handling

Java automatically removes common leading indentation from all lines, based on the closing `"""` position.

```java
String json = """
        {
          "name": "Alice",
          "age": 30
        }
        """;
```

The actual content of `json` is:

```text
{
  "name": "Alice",
  "age": 30
}
```

(no leading 8 spaces in each line)

---

## Escapes and line control

You can still use escape sequences:

- `\n` – newline
- `\t` – tab
- `\"` – quote (usually not needed)
- `\s` – space
- `\` at end of line – **suppress** the newline

Example:

```java
String message = """
        First line
        Second line\
        (continued)
        """;
System.out.println(message);
```

**Output:**

```text
First line
Second line(continued)
```

Note: The backslash at the end of `Second line\` removes the newline that would normally follow.

## Common use cases

1. **JSON / XML / HTML templates**

```java
String html = """
        <html>
          <head>
            <title>Example</title>
          </head>
          <body>
            <h1>Hello, world!</h1>
          </body>
        </html>
        """;
System.out.println(html);
```

**Output:**

```html
<html>
  <head>
    <title>Example</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
```

2. **SQL queries**

```java
String sql = """
        SELECT id, name, email
        FROM users
        WHERE active = 1
        ORDER BY name
        """;
System.out.println(sql);
```

**Output:**

```sql
SELECT id, name, email
FROM users
WHERE active = 1
ORDER BY name
```

3. **Multi-line messages / help text**

```java
String help = """
        Usage: myapp [options]
        
        Options:
          -h, --help      Show this help message
          -v, --version   Show version information
        """;
System.out.println(help);
```

**Output:**

```text
Usage: myapp [options]

Options:
  -h, --help      Show this help message
  -v, --version   Show version information
```

---

## Key points

- Text blocks use `"""` and are available from **Java 15**.
- They make multi-line strings more readable and maintainable.
- They preserve line breaks and most formatting.
- Indentation is normalized based on the closing `"""`.
- Escapes still work; a trailing `\` can remove the newline.
