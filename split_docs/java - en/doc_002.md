## 1. Access Modifiers

Control visibility (where a class/member can be used).

### Levels

1. **public**
   - Visible everywhere (any package).
2. **protected**
   - Visible in the same package and in subclasses (even in other packages).
3. **(default) package‑private** (no keyword)
   - Visible only within the same package.
4. **private**
   - Visible only within the same class.

### Example

```java
// File: com/example/Person.java
package com.example;

public class Person {
    public String name;        // accessible everywhere
    protected int age;         // same package + subclasses
    String city;               // package-private (default)
    private String password;   // only inside Person

    public Person(String name, int age, String city, String password) {
        this.name = name;
        this.age = age;
        this.city = city;
        this.password = password;
    }

    private void showPassword() {
        System.out.println("Password: " + password);
    }

    public void showInfo() {
        System.out.println(name + " (" + age + "), " + city);
    }
}
```

```java
// File: com/example/Main.java
package com.example;

public class Main {
    public static void main(String[] args) {
        Person p = new Person("Alice", 30, "Seoul", "secret123");
        System.out.println(p.name);  // OK (public)
        System.out.println(p.age);   // OK (protected, same package)
        System.out.println(p.city);  // OK (package-private, same package)
        // System.out.println(p.password); // ERROR (private)
        p.showInfo();                // OK (public method)
    }
}
```

**Sample output:**
```
Alice
30
Seoul
Alice (30), Seoul
```

## 2. `static`

Belongs to the **class**, not to a specific object.

- `static` field: shared by all instances.
- `static` method: can be called without creating an object; cannot access non-static fields directly.
- `static` block: runs once when the class is loaded.
- `static` nested class: inner class that does not need an instance of the outer class.

### Example: static fields and methods

```java
public class Counter {
    private static int count = 0; // shared by all Counter objects

    public Counter() {
        count++;
    }

    public static int getCount() {
        return count;
    }

    public static void main(String[] args) {
        new Counter();
        new Counter();
        new Counter();
        System.out.println(Counter.getCount());
    }
}
```

**Output:**
```
3
```

## 3. `final`

Means “cannot be changed” in different ways:

- `final` variable: value cannot be reassigned.
- `final` method: cannot be overridden in subclasses.
- `final` class: cannot be extended (no subclasses).

### Example: final variable

```java
public class FinalExample {
    public static void main(String[] args) {
        final int MAX = 10;
        System.out.println(MAX);
        // MAX = 20; // ERROR: cannot assign a value to final variable
    }
}
```

**Output:**
```
10
```

### Example: final method and class

```java
class Animal {
    public final void breathe() {
        System.out.println("Breathing...");
    }
}

final class Dog extends Animal {
    // Cannot override breathe():
    // public void breathe() {} // ERROR
}

// class SuperDog extends Dog {} // ERROR: cannot subclass final class
```

## 4. Inner Classes

Classes defined **inside** another class.

Types:

1. **Non-static inner class (member inner class)**
   - Needs an instance of the outer class.
2. **Static nested class**
   - Does not need an instance of the outer class.
3. **Local inner class**
   - Defined inside a method.
4. **Anonymous inner class**
   - No name; used for quick one-time implementations (often with interfaces).

### 4.1 Non-static inner class

```java
public class Outer {
    private String message = "Hello from Outer";

    public class Inner {
        public void printMessage() {
            System.out.println(message); // can access outer fields
        }
    }

    public static void main(String[] args) {
        Outer outer = new Outer();
        Outer.Inner inner = outer.new Inner();
        inner.printMessage();
    }
}
```

**Output:**
```
Hello from Outer
```

### 4.2 Static nested class

```java
public class Outer2 {
    private static String staticMsg = "Static message";

    public static class Nested {
        public void print() {
            System.out.println(staticMsg);
        }
    }

    public static void main(String[] args) {
        Outer2.Nested nested = new Outer2.Nested(); // no Outer2 instance needed
        nested.print();
    }
}
```

**Output:**
```
Static message
```

### 4.3 Anonymous inner class (simple example)

```java
public class AnonymousExample {
    interface Greeter {
        void greet();
    }

    public static void main(String[] args) {
        Greeter g = new Greeter() {
            @Override
            public void greet() {
                System.out.println("Hello from anonymous class");
            }
        };

        g.greet();
    }
}
```

**Output:**
```
Hello from anonymous class
```

---

**Summary:**

- **Access modifiers** control visibility (`public`, `protected`, default, `private`).
- **`static`** members belong to the class, shared or callable without an instance.
- **`final`** prevents changes: variables (no reassignment), methods (no override), classes (no subclass).
- **Inner classes** allow grouping related classes: non-static inner, static nested, local, and anonymous classes.
