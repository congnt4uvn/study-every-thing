
Here’s a concise overview of important Java features introduced from Java 9 to 17, with small code examples and sample I/O where it makes sense.

---

## Java 9

### 1. Module System (Project Jigsaw)
- Introduces `module-info.java` to define modules and their dependencies.
- Helps with better encapsulation and smaller runtimes.

**Example**

```java
// module-info.java
module com.example.app {
    requires com.example.lib;
    exports com.example.app.api;
}
```

No direct I/O; this is about structuring your project.

---

### 2. JShell (REPL)
- Interactive tool to run Java code snippets without creating full classes.

**Example (JShell session)**

```text
jshell> int x = 5;
x ==> 5

jshell> x * 2
$2 ==> 10
```

---

### 3. Collection Factory Methods
- Easy ways to create small immutable collections.

**Example**

```java
import java.util.List;
import java.util.Set;
import java.util.Map;

public class CollectionsDemo {
    public static void main(String[] args) {
        List<String> list = List.of("a", "b", "c");
        Set<Integer> set = Set.of(1, 2, 3);
        Map<Integer, String> map = Map.of(1, "one", 2, "two");

        System.out.println(list);
        System.out.println(set);
        System.out.println(map);
    }
}
```

**Output**

```text
[a, b, c]
[1, 2, 3]
{1=one, 2=two}
```

---

### 4. Stream API Enhancements
- New methods: `takeWhile`, `dropWhile`, `iterate` with predicate.

**Example**

```java
import java.util.stream.Stream;

public class StreamDemo {
    public static void main(String[] args) {
        Stream.of(1, 2, 3, 4, 1, 2)
              .takeWhile(n -> n < 4)
              .forEach(System.out::println);
    }
}
```

**Output**

```text
1
2
3
```

## Java 10

### 1. Local-Variable Type Inference (`var`)
- `var` lets the compiler infer the type of local variables.

**Example**

```java
public class VarDemo {
    public static void main(String[] args) {
        var message = "Hello";
        var number = 42;
        System.out.println(message + " " + number);
    }
}
```

**Output**

```text
Hello 42
```

---

## Java 11 (LTS)

### 1. New String Methods
- `isBlank()`, `lines()`, `strip()`, `repeat(int)`.

**Example**

```java
public class StringDemo {
    public static void main(String[] args) {
        String s = "  hi  ";
        System.out.println(s.isBlank());      // false
        System.out.println(s.strip());        // "hi"
        System.out.println("ha".repeat(3));   // "hahaha"
    }
}
```

**Output**

```text
false
hi
hahaha
```

### 2. `var` in Lambda Parameters
- You can use `var` in lambda parameter lists.

```java
import java.util.List;

public class LambdaVarDemo {
    public static void main(String[] args) {
        List<String> list = List.of("a", "b");
        list.forEach((var x) -> System.out.println(x.toUpperCase()));
    }
}
```

**Output**

```text
A
B
```

### 3. HTTP Client (Standard)
- New `java.net.http.HttpClient` replaces old `HttpURLConnection` for most use cases.

**Example**

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class HttpClientDemo {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://httpbin.org/get"))
                .build();

        HttpResponse<String> response =
                client.send(request, HttpResponse.BodyHandlers.ofString());

        System.out.println(response.statusCode());
        System.out.println(response.body().substring(0, 60) + "...");
    }
}
```

**Sample Output (truncated)**

```text
200
{
  "args": {}, 
  "headers": {
    "Accept-Encoding": "gzip,...
```

## Java 12–14 (Selected)

### Switch Expressions (Preview in 12/13, Standard in 14)
- Switch can return a value and use `->` syntax.

**Example (Java 14+)**

```java
public class SwitchDemo {
    public static void main(String[] args) {
        String day = "MONDAY";
        int num = switch (day) {
            case "MONDAY", "TUESDAY", "WEDNESDAY", "THURSDAY", "FRIDAY" -> 1;
            case "SATURDAY", "SUNDAY" -> 2;
            default -> 0;
        };
        System.out.println(num);
    }
}
```

**Output**

```text
1
```

## Java 15

### Text Blocks (Standard)
- Multi-line strings with `"""`.

**Example**

```java
public class TextBlockDemo {

                {
```

```text
{
```

## Java 16

### Records (Standard)
- Concise syntax for immutable data carriers.

**Example**

```java
public record Point(int x, int y) {}

class RecordDemo {
    public static void main(String[] args) {
        Point p = new Point(3, 4);
        System.out.println(p.x());
        System.out.println(p);
    }
}
```

**Output**

```text
3
Point[x=3, y=4]
```

## Java 17 (LTS)

### 1. Sealed Classes
- Restrict which classes can extend or implement a type.

**Example**

```java
public sealed interface Shape
        permits Circle, Rectangle {}

public final class Circle implements Shape {
    double radius;
    public Circle(double radius) { this.radius = radius; }
}

public final class Rectangle implements Shape {
    double w, h;
    public Rectangle(double w, double h) { this.w = w; this.h = h; }
}
```

No direct I/O; this is about type hierarchies and safety.

---

### 2. Pattern Matching for `instanceof`
- Simplifies casting after `instanceof`.

**Example**

```java
public class PatternInstanceofDemo {
    public static void main(String[] args) {
        Object obj = "hello";
        if (obj instanceof String s) {
            System.out.println(s.toUpperCase());
        }
    }
}
```

**Output**

```text
HELLO
```

---

### 3. Enhanced `switch` + Pattern Matching (Preview in later versions)
- Combined with sealed types, makes exhaustive handling easier (in later versions, but conceptually tied to 17+).

## Summary of Main Ideas (9–17)

- **Modularity (Java 9)**: `module-info.java` for better structure and encapsulation.
- **Developer Productivity**:
  - JShell (Java 9)
  - `var` for local variables and lambdas (Java 10, 11)
  - Collection factory methods, new String methods.
- **Modern APIs**:
  - New HTTP client (Java 11)
  - Stream enhancements (Java 9+).
- **Language Improvements**:
  - Switch expressions (Java 14)
  - Text blocks (Java 15)
  - Records (Java 16)
  - Sealed classes and pattern matching for `instanceof` (Java 17).
