
Here’s a concise summary of Big-O complexity for common Java collections and data structures, plus small code examples with sample input/output.

---

## 1. Array

- Access by index: **O(1)**
- Search (unsorted): **O(n)**
- Insert/delete at end (if space): **O(1)**
- Insert/delete in middle: **O(n)** (shift elements)

```java
int[] arr = {10, 20, 30, 40};
System.out.println(arr[2]); // access index 2
```

**Output:**
```text
30
```

---

## 2. `ArrayList<E>`

- Access by index: **O(1)**
- Search by value: **O(n)**
- Add at end (amortized): **O(1)**
- Insert/remove at index (middle): **O(n)** (shift elements)

```java
import java.util.*;

public class ArrayListExample {
    public static void main(String[] args) {
        List<Integer> list = new ArrayList<>();
        list.add(10); // O(1) amortized
        list.add(20);
        list.add(30);
        list.add(1, 15); // insert at index 1: O(n)

        System.out.println(list.get(2)); // O(1)
        System.out.println(list);        // print all
    }
}
```

**Output:**
```text
30
[10, 15, 20, 30]
```

---

## 3. `LinkedList<E>`

- Access by index: **O(n)** (must traverse)
- Add/remove at beginning or end (with reference): **O(1)**
- Insert/remove in middle (with node reference): **O(1)**, but finding position is **O(n)**
- Search by value: **O(n)**

```java
import java.util.*;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList<String> list = new LinkedList<>();
        list.add("A");        // O(1)
        list.addFirst("Start"); // O(1)
        list.addLast("End");    // O(1)

        System.out.println(list.get(1)); // O(n)
        System.out.println(list);
    }
}
```

**Output:**
```text
A
[Start, A, End]
```

---

## 4. `HashMap<K,V>` / `HashSet<E>`

(Backed by hash table)

- Insert: **O(1)** average, **O(n)** worst (hash collisions)
- Search (containsKey / contains): **O(1)** average
- Remove: **O(1)** average
- Iteration over all elements: **O(n)**

```java
import java.util.*;

public class HashMapExample {
    public static void main(String[] args) {
        Map<String, Integer> map = new HashMap<>();
        map.put("Alice", 25); // O(1) avg
        map.put("Bob", 30);
        map.put("Carol", 28);

        System.out.println(map.get("Bob"));        // O(1) avg
        System.out.println(map.containsKey("Dan")); // O(1) avg
    }
}
```

**Output:**
```text
30
false
```

```java
import java.util.*;

public class HashSetExample {
    public static void main(String[] args) {
        Set<Integer> set = new HashSet<>();
        set.add(1); // O(1) avg
        set.add(2);
        set.add(3);

        System.out.println(set.contains(2)); // O(1) avg
    }
}
```

**Output:**
```text
true
```

## 5. `TreeMap<K,V>` / `TreeSet<E>`

(Usually Red-Black tree)

- Insert: **O(log n)**
- Search: **O(log n)**
- Remove: **O(log n)**
- Iteration in sorted order: **O(n)**

```java
import java.util.*;

public class TreeMapExample {
    public static void main(String[] args) {
        TreeMap<Integer, String> map = new TreeMap<>();
        map.put(3, "C"); // O(log n)
        map.put(1, "A");
        map.put(2, "B");

        System.out.println(map.get(2));   // O(log n)
        System.out.println(map.firstKey()); // O(log n)
        System.out.println(map.lastKey());  // O(log n)
        System.out.println(map);          // sorted by key
    }
}
```

**Output:**
```text
B
1
3
{1=A, 2=B, 3=C}
```

## 6. `PriorityQueue<E>` (binary heap)

- Insert (`offer`/`add`): **O(log n)**
- Get min/max (`peek`): **O(1)**
- Remove min/max (`poll`): **O(log n)**
- Search arbitrary element: **O(n)**

```java
import java.util.*;

public class PriorityQueueExample {
    public static void main(String[] args) {
        PriorityQueue<Integer> pq = new PriorityQueue<>(); // min-heap
        pq.offer(5);  // O(log n)
        pq.offer(1);
        pq.offer(3);

        System.out.println(pq.peek()); // O(1) -> smallest
        System.out.println(pq.poll()); // O(log n) -> remove smallest
        System.out.println(pq);
    }
}
```

**Output:**
```text
1
1
[3, 5]
```

---

## 7. `Stack` / `Deque` as stack

- Push: **O(1)**
- Pop: **O(1)**
- Peek: **O(1)**

```java
import java.util.*;

public class StackExample {
    public static void main(String[] args) {
        Deque<Integer> stack = new ArrayDeque<>();
        stack.push(10); // O(1)
        stack.push(20);
        stack.push(30);

        System.out.println(stack.peek()); // O(1)
        System.out.println(stack.pop());  // O(1)
        System.out.println(stack);
    }
}
```

**Output:**
```text
30
30
[20, 10]
```

---

## 8. `Queue` / `Deque` as queue

- Enqueue (`offer`/`addLast`): **O(1)**
- Dequeue (`poll`/`removeFirst`): **O(1)**
- Peek front: **O(1)**

```java
import java.util.*;

public class QueueExample {
    public static void main(String[] args) {
        Deque<String> queue = new ArrayDeque<>();
        queue.offer("A"); // O(1)
        queue.offer("B");
        queue.offer("C");

        System.out.println(queue.peek()); // O(1)
        System.out.println(queue.poll()); // O(1)
        System.out.println(queue);
    }
}
```

**Output:**
```text
A
A
[B, C]
```

---

### Core Idea

- **Arrays / ArrayList**: Fast random access (**O(1)**), slow middle insert/delete (**O(n)**).
- **LinkedList**: Fast insert/delete at ends (**O(1)**), slow random access (**O(n)**).
- **HashMap / HashSet**: Average **O(1)** for insert/search/delete, no ordering.
- **TreeMap / TreeSet**: Sorted, but **O(log n)** operations.
- **PriorityQueue**: Efficient min/max retrieval (**O(1)** peek, **O(log n)** insert/remove).
- **Stack / Queue**: Constant-time operations at ends (**O(1)**).
