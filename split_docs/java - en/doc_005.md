
Here’s a concise summary of Java 8+ lambdas and method references, with examples.

---

## 1. Lambdas (Lambda Expressions)

**Idea:**  
A lambda is a short way to write an implementation of a functional interface (an interface with exactly one abstract method, like `Runnable`, `Comparator`, `Function`, etc.).

**Basic syntax:**

```java
(parameters) -> expression
(parameters) -> { statements }
```

### Example 1: Sorting a list with a lambda

```java
import java.util.*;

public class LambdaSortExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Bob", "Alice", "Charlie");

        // Before Java 8: anonymous class
        Collections.sort(names, new Comparator<String>() {
            @Override
            public int compare(String a, String b) {
                return a.compareTo(b);
            }
        });

        // Java 8: lambda
        Collections.sort(names, (a, b) -> a.compareTo(b));

        System.out.println(names);
    }
}
```

**Output:**
```text
[Alice, Bob, Charlie]
```

### Example 2: Using `forEach` with a lambda

```java
import java.util.*;

public class LambdaForEachExample {
    public static void main(String[] args) {
        List<Integer> nums = Arrays.asList(1, 2, 3, 4);

        nums.forEach(n -> System.out.println("Number: " + n));
    }
}
```

**Output:**
```text
Number: 1
Number: 2
Number: 3
Number: 4
```

## 2. Method References

**Idea:**  
Method references are a shorter form of lambdas that just call an existing method.  
They are used when a lambda only calls one method.

**Forms:**

1. `ClassName::staticMethod`
2. `instanceRef::instanceMethod`
3. `ClassName::instanceMethod` (for a parameter’s method)
4. `ClassName::new` (constructor reference)

### Example 1: Rewriting a lambda with a method reference

Lambda:

```java
nums.forEach(n -> System.out.println(n));
```

Method reference:

```java
nums.forEach(System.out::println);
```

### Example 2: Static method reference

```java
import java.util.*;
import java.util.stream.*;

public class StaticMethodRefExample {
    public static void main(String[] args) {
        List<String> numbers = Arrays.asList("1", "2", "3");

        // Lambda: s -> Integer.parseInt(s)
        List<Integer> ints = numbers.stream()
                                    .map(Integer::parseInt) // static method reference
                                    .collect(Collectors.toList());

        System.out.println(ints);
    }
}
```

**Output:**
```text
[1, 2, 3]
```

### Example 3: Constructor reference

```java
import java.util.function.Supplier;

public class ConstructorRefExample {
    static class MyObject {
        MyObject() {
            System.out.println("MyObject created");
        }
    }

    public static void main(String[] args) {
        // Lambda: () -> new MyObject()
        Supplier<MyObject> supplier = MyObject::new;

        MyObject obj = supplier.get(); // calls constructor
    }
}
```

**Output:**
```text
MyObject created
```

---

**Key points:**

- Lambdas provide concise implementations of functional interfaces.
- Method references are a shorthand for lambdas that only call an existing method.
- Both features make code shorter, clearer, and more functional-style in Java 8+.
