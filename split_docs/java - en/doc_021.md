
Here’s a concise summary and examples for the Java `ArrayList` and `LinkedList`:

---

## 1. ArrayList

**Key ideas:**
- Backed by a **dynamic array**.
- **Fast random access** by index: `get(i)` is O(1).
- **Adding/removing at the end** is usually O(1) amortized.
- **Inserting/removing in the middle or beginning** is slower (O(n)) because elements must be shifted.
- Good when you:
  - Access elements by index often.
  - Mostly add/remove at the end.

**Basic example:**

```java
import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();

        // Input: operations
        list.add("A");          // [A]
        list.add("B");          // [A, B]
        list.add("C");          // [A, B, C]
        list.add(1, "X");       // insert at index 1 -> [A, X, B, C]
        list.remove(2);         // remove element at index 2 -> [A, X, C]

        // Output
        System.out.println("List: " + list);
        System.out.println("Element at index 1: " + list.get(1));
        System.out.println("Size: " + list.size());
    }
}
```

**Sample output:**
```
List: [A, X, C]
Element at index 1: X
Size: 3
```

---

## 2. LinkedList

**Key ideas:**
- Implemented as a **doubly linked list**.
- Each element (node) holds:
  - The value
  - A reference to the previous node
  - A reference to the next node
- **Fast insertions/removals at the beginning or middle** if you already have a reference/iterator to the node (O(1)).
- **Slower random access by index**: `get(i)` is O(n) because it must traverse nodes.
- Also implements `Deque`, so it can be used as a **queue** or **stack**.
- Good when you:
  - Frequently add/remove at the beginning or in the middle.
  - Traverse sequentially more than you randomly access by index.

**Basic example:**

```java
import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList<String> list = new LinkedList<>();

        // Input: operations
        list.add("A");          // [A]
        list.add("B");          // [A, B]
        list.addFirst("Start"); // [Start, A, B]
        list.addLast("End");    // [Start, A, B, End]
        list.remove("A");       // [Start, B, End]
        String first = list.getFirst(); // "Start"
        String last = list.getLast();   // "End"

        // Output
        System.out.println("List: " + list);
        System.out.println("First: " + first);
        System.out.println("Last: " + last);
        System.out.println("Size: " + list.size());
    }
}
```

**Sample output:**
```
List: [Start, B, End]
First: Start
Last: End
Size: 3
```

---

## 3. When to use which?

- **Use ArrayList when:**
  - You need **fast random access** by index.
  - You mostly **append** elements at the end.
- **Use LinkedList when:**
  - You frequently **insert/remove at the beginning** or in the middle.
  - You use it as a **queue/deque** (e.g., `addFirst`, `addLast`, `removeFirst`, `removeLast`).
