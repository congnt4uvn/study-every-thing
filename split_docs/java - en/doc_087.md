
Here’s a concise summary and some Java-focused examples for the mentioned NoSQL databases.

---

## Key Ideas

- **NoSQL** databases are non-relational and are often used for:
  - High scalability and performance
  - Flexible schemas
  - Handling large volumes of data

- You should have **basic understanding of at least one** of:
  - **MongoDB** – Document-oriented (JSON-like documents)
  - **Cassandra** – Wide-column store, highly scalable, distributed
  - **Redis** – In-memory key-value store, very fast, often used for caching

---

## 1. MongoDB (Java Example)

**Concepts**: Collections, documents (BSON/JSON), flexible schema.

**Maven dependency:**
```xml
<dependency>
  <groupId>org.mongodb</groupId>
  <artifactId>mongodb-driver-sync</artifactId>
  <version>4.11.1</version>
</dependency>
```

**Java code:**
```java
import com.mongodb.client.*;
import org.bson.Document;

public class MongoExample {
    public static void main(String[] args) {
        try (MongoClient client = MongoClients.create("mongodb://localhost:27017")) {
            MongoDatabase db = client.getDatabase("testdb");
            MongoCollection<Document> users = db.getCollection("users");

            // INSERT
            Document user = new Document("name", "Alice")
                    .append("age", 30)
                    .append("email", "alice@example.com");
            users.insertOne(user);

            // QUERY
            Document found = users.find(new Document("name", "Alice")).first();
            System.out.println(found);
        }
    }
}
```

**Example input (document inserted):**
```json
{ "name": "Alice", "age": 30, "email": "alice@example.com" }
```

**Example output (console):**
```text
Document{{_id=..., name=Alice, age=30, email=alice@example.com}}
```

## 2. Cassandra (Java Example)

**Concepts**: Keyspace, tables, partition keys, clustering keys, high write throughput.

**Maven dependency:**
```xml
<dependency>
  <groupId>com.datastax.oss</groupId>
  <artifactId>java-driver-core</artifactId>
  <version>4.17.0</version>
</dependency>
```

**Java code:**
```java
import com.datastax.oss.driver.api.core.CqlSession;
import com.datastax.oss.driver.api.core.cql.*;

import java.net.InetSocketAddress;

public class CassandraExample {
    public static void main(String[] args) {
        try (CqlSession session = CqlSession.builder()
                .addContactPoint(new InetSocketAddress("127.0.0.1", 9042))
                .withLocalDatacenter("datacenter1")
                .build()) {

            // CREATE KEYSPACE & TABLE (usually done once)
            session.execute("CREATE KEYSPACE IF NOT EXISTS demo " +
                    "WITH replication = {'class': 'SimpleStrategy', 'replication_factor': 1}");
            session.execute("CREATE TABLE IF NOT EXISTS demo.users (" +
                    "id uuid PRIMARY KEY, name text, age int)");

            // INSERT
            session.execute("INSERT INTO demo.users (id, name, age) " +
                    "VALUES (uuid(), 'Bob', 25)");

            // QUERY
            ResultSet rs = session.execute("SELECT name, age FROM demo.users");
            for (Row row : rs) {
                System.out.println(row.getString("name") + " - " + row.getInt("age"));
            }
        }
    }
}
```

**Example output (console):**
```text
Bob - 25
```

## 3. Redis (Java Example)

**Concepts**: Key-value, in-memory, supports strings, lists, sets, hashes, etc. Often used for caching, sessions, counters.

**Maven dependency (Jedis):**
```xml
<dependency>
  <groupId>redis.clients</groupId>
  <artifactId>jedis</artifactId>
  <version>5.1.0</version>
</dependency>
```

**Java code:**
```java
import redis.clients.jedis.Jedis;

public class RedisExample {
    public static void main(String[] args) {
        try (Jedis jedis = new Jedis("localhost", 6379)) {
            // SET
            jedis.set("greeting", "Hello, Redis!");

            // GET
            String value = jedis.get("greeting");
            System.out.println(value);

            // COUNTER
            jedis.incr("page_views");
            System.out.println("Page views: " + jedis.get("page_views"));
        }
    }
}
```

**Example output (console):**
```text
Hello, Redis!
Page views: 1
```
