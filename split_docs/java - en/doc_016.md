
Here’s a concise summary of key new features introduced in **Java 8**, with short code examples and sample inputs/outputs.

---

## 1. Lambda Expressions

Anonymous functions that make it easier to pass behavior as data (especially for collections and streams).

**Example:**

```java
import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

        // Lambda used in forEach
        names.forEach(name -> System.out.println("Hello, " + name));
    }
}
```

**Output:**
```
Hello, Alice
Hello, Bob
Hello, Charlie
```

---

## 2. Functional Interfaces

Interfaces with exactly one abstract method, used as targets for lambda expressions.  
Common ones: `Predicate<T>`, `Function<T,R>`, `Consumer<T>`, `Supplier<T>`.

**Example:**

```java
@FunctionalInterface
interface MyOperation {
    int apply(int x, int y);
}

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        MyOperation add = (a, b) -> a + b;
        MyOperation multiply = (a, b) -> a * b;

        System.out.println(add.apply(3, 5));       // 8
        System.out.println(multiply.apply(3, 5));  // 15
    }
}
```

**Output:**
```
8
15
```

---

## 3. Stream API

A new way to process collections in a declarative, functional style (filter, map, reduce, etc.).

**Example:**

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

        // Filter even numbers and square them
        List<Integer> result = numbers.stream()
                .filter(n -> n % 2 == 0)
                .map(n -> n * n)
                .collect(Collectors.toList());

        System.out.println(result);
    }
}
```

**Output:**
```
[4, 16, 36]
```

---

## 4. Default and Static Methods in Interfaces

Interfaces can now have method implementations using `default` and `static` methods.

**Example:**

```java
interface Vehicle {
    void move();

    default void start() {
        System.out.println("Vehicle is starting");
    }

    static void info() {
        System.out.println("This is a vehicle interface");
    }
}

class Car implements Vehicle {
    public void move() {
        System.out.println("Car is moving");
    }
}

public class DefaultMethodExample {
    public static void main(String[] args) {
        Car car = new Car();
        car.start();      // default method
        car.move();
        Vehicle.info();   // static method
    }
}
```

**Output:**
```
Vehicle is starting
Car is moving
This is a vehicle interface
```

---

## 5. Method References

Shorter syntax for lambdas that just call an existing method.

**Example:**

```java
import java.util.Arrays;
import java.util.List;

public class MethodReferenceExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

        // Using method reference instead of name -> System.out.println(name)
        names.forEach(System.out::println);
    }
}
```

**Output:**
```
Alice
Bob
Charlie
```

---

## 6. Optional

A container object that may or may not contain a non-null value, helping to avoid `NullPointerException`.

**Example:**

```java
import java.util.Optional;

public class OptionalExample {
    public static void main(String[] args) {
        Optional<String> maybeName = Optional.of("Alice");
        Optional<String> empty = Optional.empty();

        System.out.println(maybeName.orElse("Unknown")); // Alice
        System.out.println(empty.orElse("Unknown"));     // Unknown
    }
}
```

**Output:**
```
Alice
Unknown
```

---

## 7. New Date and Time API (java.time)

A modern, immutable, thread-safe date/time API (`LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime`, etc.).

**Example:**

```java
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class DateTimeExample {
    public static void main(String[] args) {
        LocalDate today = LocalDate.now();
        LocalDateTime now = LocalDateTime.now();

        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm");
        String formatted = now.format(formatter);

        System.out.println("Today: " + today);
        System.out.println("Now: " + formatted);
    }
}
```

**Sample Output (will vary by date/time):**
```
Today: 2026-05-20
Now: 2026-05-20 14:37
```

---

## 8. Streams Parallelism

Easy parallel processing using `parallelStream()`.

**Example:**

```java
import java.util.stream.IntStream;

public class ParallelStreamExample {
    public static void main(String[] args) {
        int sum = IntStream.rangeClosed(1, 10)
                .parallel()
                .sum();

        System.out.println(sum);
    }
}
```

**Output:**
```
55
```

---

**Main Idea Summary:**

Java 8 introduced functional programming features (lambdas, method references, functional interfaces), the Stream API for powerful collection processing, improved interfaces (default/static methods), safer null handling with `Optional`, and a modern date/time API, along with easier parallelism. These changes make Java code more concise, expressive, and less error-prone.
