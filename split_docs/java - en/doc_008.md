
Here are the key ideas about Java 8+ functional interfaces, with concise examples and sample inputs/outputs.

---

## 1. Functional Interfaces: Overview

- A **functional interface** has exactly **one abstract method**.
- Used heavily with **lambda expressions** and **method references**.
- Common built-in functional interfaces in `java.util.function`:
  - `Function<T, R>`
  - `Predicate<T>`
  - `Supplier<T>`
  - `Consumer<T>`

---

## 2. `Function<T, R>`

- Represents a function that takes an input of type `T` and returns a result of type `R`.
- Method: `R apply(T t)`

### Example

```java
import java.util.function.Function;

public class FunctionExample {
    public static void main(String[] args) {
        // Function: Integer -> String
        Function<Integer, String> intToString = i -> "Number: " + i;

        String result = intToString.apply(5);
        System.out.println(result);
    }
}
```

**Input:** `5`  
**Output:**
```text
Number: 5
```

---

## 3. `Predicate<T>`

- Represents a boolean-valued function of one argument.
- Method: `boolean test(T t)`

### Example

```java
import java.util.function.Predicate;

public class PredicateExample {
    public static void main(String[] args) {
        // Predicate: Integer -> boolean (is even?)
        Predicate<Integer> isEven = n -> n % 2 == 0;

        System.out.println(isEven.test(4)); // true
        System.out.println(isEven.test(7)); // false
    }
}
```

**Inputs:** `4`, `7`  
**Outputs:**
```text
true
false
```

---

## 4. `Supplier<T>`

- Represents a provider of results; no input, only output.
- Method: `T get()`

### Example

```java
import java.util.function.Supplier;
import java.time.LocalDateTime;

public class SupplierExample {
    public static void main(String[] args) {
        // Supplier: () -> current time
        Supplier<LocalDateTime> nowSupplier = () -> LocalDateTime.now();

        LocalDateTime now = nowSupplier.get();
        System.out.println(now);
    }
}
```

**Input:** none  
**Output:** current date-time, e.g.
```text
2026-05-20T14:32:10.123
```

---

## 5. `Consumer<T>`

- Represents an operation that takes an input and returns no result (side effects only).
- Method: `void accept(T t)`

### Example

```java
import java.util.function.Consumer;

public class ConsumerExample {
    public static void main(String[] args) {
        // Consumer: String -> prints it
        Consumer<String> printer = s -> System.out.println("Hello, " + s);

        printer.accept("Alice");
    }
}
```

**Input:** `"Alice"`  
**Output:**
```text
Hello, Alice
```

---

## Summary

- **Function**: transforms input to output (`T -> R`).
- **Predicate**: checks a condition, returns `boolean`.
- **Supplier**: supplies a value, no input.
- **Consumer**: consumes a value, no return.

These interfaces enable concise, functional-style programming in Java 8+ using lambdas and method references.
