
Here’s a concise overview of notable Java features from 18 to 21, with short code examples where useful.

---

## Java 18

### 1. Simple Web Server (JEP 408)
A minimal HTTP file server for testing and local development.

**Run from CLI:**
```bash
# Serves current directory on port 8000
java -m jdk.httpserver
```

**Programmatic use:**
```java
import com.sun.net.httpserver.HttpServer;
import java.net.InetSocketAddress;

public class SimpleServer {
    public static void main(String[] args) throws Exception {
        HttpServer server = HttpServer.create(new InetSocketAddress(8000), 0);
        server.createContext("/", exchange -> {
            String response = "Hello, Java 18!";
            exchange.sendResponseHeaders(200, response.getBytes().length);
            exchange.getResponseBody().write(response.getBytes());
            exchange.close();
        });
        server.start();
    }
}
```

**Input:** HTTP GET `http://localhost:8000/`  
**Output:** `Hello, Java 18!`

## Java 19

Most big features are **preview** or **incubating** here.

### 1. Virtual Threads (Preview, JEP 425)
Lightweight threads managed by the JVM, making high-concurrency code easier.

```java
public class VirtualThreadsExample {
    public static void main(String[] args) throws Exception {
        try (var executor = java.util.concurrent.Executors.newVirtualThreadPerTaskExecutor()) {
            for (int i = 0; i < 3; i++) {
                int id = i;
                executor.submit(() -> {
                    System.out.println("Task " + id + " on " + Thread.currentThread());
                    Thread.sleep(100);
                    return null;
                });
            }
        }
    }
}
```

**Output (example):**
```
Task 0 on VirtualThread[#21]/runnable
Task 1 on VirtualThread[#22]/runnable
Task 2 on VirtualThread[#23]/runnable
```

### 2. Record Patterns (Preview, JEP 405)
Pattern matching for record deconstruction.

```java
record Point(int x, int y) {}

static void printPoint(Object obj) {
    if (obj instanceof Point(int x, int y)) {
        System.out.println("x = " + x + ", y = " + y);
    }
}

public static void main(String[] args) {
    printPoint(new Point(3, 4));
}
```

**Output:**
```
x = 3, y = 4
```

## Java 20

Mostly **second previews** and refinements.

### 1. Virtual Threads (2nd Preview, JEP 436)
Improved virtual threads; API similar to Java 19.

### 2. Record Patterns (2nd Preview, JEP 432)
More pattern matching capabilities, including nested patterns.

```java
record Point(int x, int y) {}
record Line(Point start, Point end) {}

static void printLine(Object obj) {
    if (obj instanceof Line(Point(int x1, int y1), Point(int x2, int y2))) {
        System.out.println("Line from (" + x1 + "," + y1 + ") to (" + x2 + "," + y2 + ")");
    }
}

public static void main(String[] args) {
    printLine(new Line(new Point(0, 0), new Point(5, 5)));
}
```

**Output:**
```
Line from (0,0) to (5,5)
```

## Java 21 (LTS – Long-Term Support)

Java 21 makes several features **final** and adds important new ones.

### 1. Virtual Threads (Final, JEP 444)
Now a standard feature.

```java
import java.util.concurrent.*;

public class VirtualThreadsFinal {
    public static void main(String[] args) throws Exception {
        try (ExecutorService executor = Executors.newVirtualThreadPerTaskExecutor()) {
            var futures = executor.invokeAll(
                java.util.List.of(
                    () -> "Task 1",
                    () -> "Task 2",
                    () -> "Task 3"
                )
            );
            for (var f : futures) {
                System.out.println(f.get());
            }
        }
    }
}
```

**Output:**
```
Task 1
Task 2
Task 3
```

### 2. Pattern Matching for switch (Final, JEP 441)
Use patterns directly in `switch`.

```java
static String format(Object obj) {
    return switch (obj) {
        case Integer i -> "int: " + i;
        case String s  -> "string: " + s.toUpperCase();
        case null      -> "null value";
        default        -> "unknown: " + obj;
    };
}

public static void main(String[] args) {
    System.out.println(format(10));
    System.out.println(format("hello"));
    System.out.println(format(null));
}
```

**Output:**
```
int: 10
string: HELLO
null value
```

### 3. Record Patterns (Final, JEP 440)
Record patterns are now standard.

```java
record Point(int x, int y) {}

static String describe(Object obj) {
    return switch (obj) {
        case Point(int x, int y) when x == y -> "Diagonal point " + x;
        case Point(int x, int y)            -> "Point(" + x + "," + y + ")";
        default                             -> "Not a point";
    };
}

public static void main(String[] args) {
    System.out.println(describe(new Point(3, 3)));
    System.out.println(describe(new Point(2, 5)));
}
```

**Output:**
```
Diagonal point 3
Point(2,5)
```

### 4. Sequenced Collections (JEP 431)
New interfaces that guarantee a well-defined encounter order: `SequencedCollection`, `SequencedSet`, `SequencedMap`.

```java
import java.util.*;

public class SequencedExample {
    public static void main(String[] args) {
        SequencedMap<Integer, String> map = new LinkedHashMap<>();
        map.putFirst(1, "first");
        map.putLast(2, "last");

        System.out.println(map.firstEntry()); // Input: map
        System.out.println(map.lastEntry());
    }
}
```

**Output:**
```
1=first
2=last
```

### 5. String Templates (Preview, JEP 430)
Safer, more powerful string interpolation.

```java
// Enable preview features when compiling/running
// javac --release 21 --enable-preview Main.java
// java --enable-preview Main

import static java.lang.StringTemplate.STR;

public class StringTemplatesExample {
    public static void main(String[] args) {
        int x = 10;
        int y = 20;
        String s = STR."x = \{x}, y = \{y}, sum = \{x + y}";
        System.out.println(s);
    }
}
```

**Output:**
```
x = 10, y = 20, sum = 30
```

### 6. Scoped Values (Preview, JEP 446)
A safer alternative to thread-local for sharing data across threads.

```java
// Preview feature; simplified example
import jdk.incubator.concurrent.ScopedValue;

public class ScopedValuesExample {
    static final ScopedValue<String> USER = ScopedValue.newInstance();

    public static void main(String[] args) throws Exception {
        ScopedValue.where(USER, "Alice").run(() -> {
            System.out.println("User = " + USER.get());
        });
    }
}
```

**Output:**
```
User = Alice
```

---

### Summary of Main Ideas

- **Java 18**: Simple web server for quick HTTP serving.
- **Java 19–20**: Introduce and refine **virtual threads** and **record patterns** as preview features.
- **Java 21 (LTS)**:
  - Virtual threads are **final** → easy, scalable concurrency.
  - Pattern matching for `switch` and record patterns are **final** → more expressive, safer type handling.
  - Sequenced collections → consistent ordering APIs.
  - String templates, scoped values → safer string building and context passing (still preview).
