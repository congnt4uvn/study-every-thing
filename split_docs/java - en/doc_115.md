
Here’s a concise summary and some Java-focused examples around cloud storage basics with S3 (AWS) and GCS (Google Cloud Storage).

---

## Key Ideas

### 1. What S3 and GCS Are
- **Amazon S3** and **Google Cloud Storage (GCS)** are object storage services.
- You store data as **objects** inside **buckets**.
- Common use cases:
  - File backups
  - Serving static content (images, videos, documents)
  - Storing logs and data for analytics
  - Application file uploads

### 2. Core Concepts (Both S3 and GCS)
- **Bucket**: Top-level container (like a folder at the root).
- **Object**: The actual file/data (e.g., `image.png`, `data.json`).
- **Key / Object name**: The path/name of the object in the bucket.
- **Region**: Physical location of the storage.
- **Permissions**: IAM roles, bucket policies, ACLs to control access.

### 3. Typical Operations
- Create a bucket
- Upload an object
- Download an object
- List objects in a bucket
- Delete objects/buckets
- Set access control (public/private, roles)

## Java with Amazon S3 (AWS SDK v2)

### Maven Dependency

```xml
<dependency>
    <groupId>software.amazon.awssdk</groupId>
    <artifactId>s3</artifactId>
    <version>2.25.0</version> <!-- example version -->
</dependency>
```

### 1) Create S3 Client

```java
import software.amazon.awssdk.regions.Region;
import software.amazon.awssdk.services.s3.S3Client;

S3Client s3 = S3Client.builder()
        .region(Region.US_EAST_1)
        .build();
```

### 2) Create a Bucket

```java
import software.amazon.awssdk.services.s3.model.CreateBucketRequest;

String bucketName = "my-example-bucket-12345";

CreateBucketRequest createBucketRequest = CreateBucketRequest.builder()
        .bucket(bucketName)
        .build();

s3.createBucket(createBucketRequest);
System.out.println("Bucket created: " + bucketName);
```

**Output (console):**
```text
Bucket created: my-example-bucket-12345
```

### 3) Upload a File (Put Object)

```java
import software.amazon.awssdk.services.s3.model.PutObjectRequest;
import java.nio.file.Paths;

String key = "folder/hello.txt";

PutObjectRequest putObjectRequest = PutObjectRequest.builder()
        .bucket(bucketName)
        .key(key)
        .build();

s3.putObject(putObjectRequest, Paths.get("local-hello.txt"));
System.out.println("Uploaded object: " + key);
```

**Input file:** `local-hello.txt`  
**Example content:**
```text
Hello from local file!
```

**Output (console):**
```text
Uploaded object: folder/hello.txt
```

### 4) Download a File (Get Object)

```java
import software.amazon.awssdk.services.s3.model.GetObjectRequest;
import java.nio.file.Path;
import java.nio.file.Paths;

GetObjectRequest getObjectRequest = GetObjectRequest.builder()
        .bucket(bucketName)
        .key(key)
        .build();

Path destination = Paths.get("downloaded-hello.txt");
s3.getObject(getObjectRequest, destination);
System.out.println("Downloaded to: " + destination.toAbsolutePath());
```

**Output (console):**
```text
Downloaded to: /path/to/project/downloaded-hello.txt
```

**Downloaded file content:**
```text
Hello from local file!
```

### 5) List Objects in a Bucket

```java
import software.amazon.awssdk.services.s3.model.ListObjectsV2Request;
import software.amazon.awssdk.services.s3.model.ListObjectsV2Response;
import software.amazon.awssdk.services.s3.model.S3Object;

ListObjectsV2Request listReq = ListObjectsV2Request.builder()
        .bucket(bucketName)
        .build();

ListObjectsV2Response listRes = s3.listObjectsV2(listReq);

for (S3Object obj : listRes.contents()) {
    System.out.println("Object key: " + obj.key() + ", size: " + obj.size());
}
```

**Example output:**
```text
Object key: folder/hello.txt, size: 23
```

## Java with Google Cloud Storage (GCS)

### Maven Dependency

```xml
<dependency>
    <groupId>com.google.cloud</groupId>
    <artifactId>google-cloud-storage</artifactId>
    <version>2.40.0</version> <!-- example version -->
</dependency>
```

### 1) Create Storage Client

```java
import com.google.cloud.storage.Storage;
import com.google.cloud.storage.StorageOptions;

Storage storage = StorageOptions.getDefaultInstance().getService();
```

### 2) Create a Bucket

```java
import com.google.cloud.storage.Bucket;

String bucketName = "my-gcs-bucket-12345";

Bucket bucket = storage.create(Bucket.newBuilder(bucketName).build());
System.out.println("Bucket created: " + bucket.getName());
```

**Output:**
```text
Bucket created: my-gcs-bucket-12345
```

### 3) Upload an Object

```java
import com.google.cloud.storage.Blob;
import com.google.cloud.storage.BlobId;
import com.google.cloud.storage.BlobInfo;
import java.nio.file.Files;
import java.nio.file.Paths;

String objectName = "folder/hello-gcs.txt";
byte[] content = Files.readAllBytes(Paths.get("local-hello-gcs.txt"));

BlobId blobId = BlobId.of(bucketName, objectName);
BlobInfo blobInfo = BlobInfo.newBuilder(blobId).build();
Blob blob = storage.create(blobInfo, content);

System.out.println("Uploaded object: " + blob.getName());
```

**Input file:** `local-hello-gcs.txt`  
**Example content:**
```text
Hello from GCS local file!
```

**Output:**
```text
Uploaded object: folder/hello-gcs.txt
```

### 4) Download an Object

```java
import java.nio.file.Path;
import java.nio.file.StandardOpenOption;

Blob downloaded = storage.get(BlobId.of(bucketName, objectName));
byte[] bytes = downloaded.getContent();

Path dest = Paths.get("downloaded-hello-gcs.txt");
Files.write(dest, bytes, StandardOpenOption.CREATE, StandardOpenOption.TRUNCATE_EXISTING);

System.out.println("Downloaded to: " + dest.toAbsolutePath());
```

**Output:**
```text
Downloaded to: /path/to/project/downloaded-hello-gcs.txt
```

**Downloaded file content:**
```text
Hello from GCS local file!
```

### 5) List Objects in a Bucket

```java
import com.google.cloud.storage.Blob;

for (Blob b : storage.list(bucketName).iterateAll()) {
    System.out.println("Object name: " + b.getName() + ", size: " + b.getSize());
}
```

**Example output:**
```text
Object name: folder/hello-gcs.txt, size: 29
```

## Summary

- S3 and GCS are **object storage** services used heavily in Cloud & DevOps.
- Main operations: create bucket, upload/download objects, list, delete, and manage permissions.
- Java provides official SDKs for both AWS and GCP to interact with these services programmatically.
