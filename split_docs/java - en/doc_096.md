
Here’s a concise summary and example of a **layered architecture** in Java with **presentation**, **service**, and **repository** layers.

---

## Key Ideas

- **Layered architecture** separates an application into distinct layers:
  1. **Presentation layer**: Handles user interaction (e.g., controllers, UI, REST endpoints).
  2. **Service layer**: Contains business logic and application rules.
  3. **Repository layer**: Handles data access (e.g., database operations).

- **Benefits**:
  - Clear separation of concerns
  - Easier testing and maintenance
  - Layers can be changed or replaced with minimal impact on others

---

## Simple Java Example (Spring-style)

### 1. Repository Layer (Data Access)

```java
// Input: userId (Long)
// Output: User object from "database" or null
public interface UserRepository {
    User findById(Long id);
}

public class InMemoryUserRepository implements UserRepository {
    private Map<Long, User> users = new HashMap<>();

    public InMemoryUserRepository() {
        users.put(1L, new User(1L, "Alice"));
        users.put(2L, new User(2L, "Bob"));
    }

    @Override
    public User findById(Long id) {
        return users.get(id);
    }
}
```

### 2. Service Layer (Business Logic)

```java
// Input: userId (Long)
// Output: String greeting message
public class UserService {
    private final UserRepository userRepository;

    public UserService(UserRepository userRepository) {
        this.userRepository = userRepository;
    }

    public String getGreetingForUser(Long id) {
        User user = userRepository.findById(id);
        if (user == null) {
            return "User not found";
        }
        return "Hello, " + user.getName() + "!";
    }
}
```

### 3. Presentation Layer (Controller / UI)

```java
// Simulated "controller" using main method
public class UserController {
    public static void main(String[] args) {
        UserRepository repo = new InMemoryUserRepository();
        UserService service = new UserService(repo);

        // Input: userId = 1
        Long inputId = 1L;
        String output = service.getGreetingForUser(inputId);

        System.out.println("Input: " + inputId);
        System.out.println("Output: " + output);
    }
}
```

### Example Run (Console)

**Input:**
```text
userId = 1
```

**Output:**
```text
Input: 1
Output: Hello, Alice!
```

**Input:**
```text
userId = 99
```

**Output:**
```text
Input: 99
Output: User not found
```

---

This demonstrates how each layer has a clear responsibility:
- Presentation: receives input, shows output.
- Service: decides what to do (business rules).
- Repository: fetches data.
