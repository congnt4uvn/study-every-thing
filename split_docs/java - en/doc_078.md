## Key Ideas

- **SQL & RDBMS**: You should be comfortable working with relational databases (e.g., MySQL, PostgreSQL, Oracle).
- **Strong SQL skills**:
  - Writing and understanding **JOINs** (INNER, LEFT, RIGHT, FULL).
  - Using **subqueries** (in `SELECT`, `FROM`, `WHERE`).
  - Aggregating data with **GROUP BY** and filtering groups with **HAVING`.
  - Using **window functions** (if the database supports them), such as `ROW_NUMBER`, `RANK`, `SUM() OVER(...)`.

---

## Code Examples

Assume a simple schema:

```sql
-- Tables
Employees(id, name, department_id, salary)
Departments(id, name)
```

### 1. JOIN Example

**Task**: List employees with their department names.

```sql
SELECT e.name AS employee_name,
       d.name AS department_name
FROM Employees e
JOIN Departments d ON e.department_id = d.id;
```

**Sample Output**

| employee_name | department_name |
|---------------|-----------------|
| Alice         | HR              |
| Bob           | IT              |

---

### 2. Subquery Example

**Task**: Find employees who earn more than the average salary.

```sql
SELECT name, salary
FROM Employees
WHERE salary > (
    SELECT AVG(salary)
    FROM Employees
);
```

**Sample Output**

| name  | salary |
|-------|--------|
| Alice | 90000  |
| Bob   | 95000  |

---

### 3. GROUP BY and HAVING Example

**Task**: Show departments with more than 3 employees and their average salary.

```sql
SELECT d.name AS department_name,
       COUNT(e.id) AS employee_count,
       AVG(e.salary) AS avg_salary
FROM Departments d
JOIN Employees e ON e.department_id = d.id
GROUP BY d.name
HAVING COUNT(e.id) > 3;
```

**Sample Output**

| department_name | employee_count | avg_salary |
|-----------------|----------------|-----------|
| IT              | 5              | 88000     |

---

### 4. Window Function Example

**Task**: Rank employees by salary within each department.

```sql
SELECT e.name,
       d.name AS department_name,
       e.salary,
       RANK() OVER (
           PARTITION BY d.id
           ORDER BY e.salary DESC
       ) AS salary_rank_in_dept
FROM Employees e
JOIN Departments d ON e.department_id = d.id;
```

**Sample Output**

| name  | department_name | salary | salary_rank_in_dept |
|-------|-----------------|--------|---------------------|
| Bob   | IT              | 95000  | 1                   |
| Carol | IT              | 90000  | 2                   |
| Alice | HR              | 90000  | 1                   |
