## 1. JVM (Java Virtual Machine)

**What it is:**  
- A virtual machine that runs Java bytecode.  
- It is platform-dependent (different JVMs for Windows, Linux, etc.), but it makes Java programs *platform-independent* at the source/bytecode level.

**Main roles:**
- Loads `.class` files (bytecode).
- Verifies and executes bytecode.
- Manages memory (heap, stack, garbage collection).

**Key point:**  
You cannot ‚Äúsee‚Äù the JVM as a separate install usually; it‚Äôs part of a JRE or JDK.

---

## 2. JRE (Java Runtime Environment)

**What it is:**  
- The environment needed to *run* Java applications.  
- Includes:
  - JVM
  - Core libraries (e.g., `java.lang`, `java.util`)
  - Supporting files

**Does NOT include:**  
- Development tools like `javac` (the Java compiler).

**Use case:**  
- For users who only need to run Java programs, not develop them.

---

## 3. JDK (Java Development Kit)

**What it is:**  
- A full toolkit for *developing* and running Java applications.  
- Includes:
  - JRE (and thus the JVM)
  - Compiler (`javac`)
  - Tools like `javadoc`, `jar`, `jshell`, etc.

**Use case:**  
- For developers writing, compiling, debugging, and packaging Java code.

---

## Relationship Summary

- **JDK** ‚äÉ **JRE** ‚äÉ **JVM**

In words:
- The JDK contains the JRE.
- The JRE contains the JVM and core libraries.

## Simple Code Example

### Java Source Code (HelloWorld.java)

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, JVM!");
    }
}
```

### Steps and Tools

1. **Write code** (requires JDK, but conceptually this is just editing a file):
   - File: `HelloWorld.java`

2. **Compile** (JDK: `javac`):
   ```bash
   javac HelloWorld.java
   ```
   - Input: `HelloWorld.java` (source code)
   - Output: `HelloWorld.class` (bytecode)

3. **Run** (JRE/JDK: `java` which launches the JVM):
   ```bash
   java HelloWorld
   ```
   - Input: `HelloWorld.class` (bytecode)
   - The JVM loads and executes the bytecode.

### Program Output

```text
Hello, JVM!
```

| Component | Contains                | Used For                 | Includes Compiler? |
|----------|-------------------------|--------------------------|--------------------|
| JVM      | Execution engine        | Running bytecode         | No                 |

This captures the core differences and how they work together in the Java ecosystem.
