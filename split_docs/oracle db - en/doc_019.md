
Here’s a concise summary and examples for converting rows to columns and columns to rows in Oracle Database.

---

## 1. Converting Rows to Columns (Pivot)

Use `PIVOT` to turn row values into columns.

### Example Table

```sql
CREATE TABLE sales (
    sale_year  NUMBER,
    region     VARCHAR2(20),
    amount     NUMBER
);

INSERT INTO sales VALUES (2023, 'NORTH', 100);
INSERT INTO sales VALUES (2023, 'SOUTH', 150);
INSERT INTO sales VALUES (2023, 'EAST',  200);
INSERT INTO sales VALUES (2024, 'NORTH', 120);
INSERT INTO sales VALUES (2024, 'SOUTH', 180);
INSERT INTO sales VALUES (2024, 'EAST',  220);
COMMIT;
```

### Pivot Query (Rows → Columns)

```sql
SELECT *
FROM sales
PIVOT (
    SUM(amount) AS amt
    FOR region IN ('NORTH' AS north, 'SOUTH' AS south, 'EAST' AS east)
)
ORDER BY sale_year;
```

**Output:**

| SALE_YEAR | NORTH_AMT | SOUTH_AMT | EAST_AMT |
|----------:|----------:|----------:|---------:|
| 2023      | 100       | 150       | 200      |
| 2024      | 120       | 180       | 220      |

Key ideas:
- `PIVOT` aggregates data and turns distinct values in one column (`region`) into separate columns.
- You must list the values to pivot (`IN ('NORTH' AS north, ...)`).

---

## 2. Converting Columns to Rows (Unpivot)

Use `UNPIVOT` to turn columns into rows.

### Example Table

```sql
CREATE TABLE sales_pivoted (
    sale_year  NUMBER,
    north_amt  NUMBER,
    south_amt  NUMBER,
    east_amt   NUMBER
);

INSERT INTO sales_pivoted VALUES (2023, 100, 150, 200);
INSERT INTO sales_pivoted VALUES (2024, 120, 180, 220);
COMMIT;
```

### Unpivot Query (Columns → Rows)

```sql
SELECT sale_year,
       region,
       amount
FROM sales_pivoted
UNPIVOT (
    amount FOR region IN (
        north_amt AS 'NORTH',
        south_amt AS 'SOUTH',
        east_amt  AS 'EAST'
    )
)
ORDER BY sale_year, region;
```

**Output:**

| SALE_YEAR | REGION | AMOUNT |
|----------:|--------|--------|
| 2023      | EAST   | 200    |
| 2023      | NORTH  | 100    |
| 2023      | SOUTH  | 150    |
| 2024      | EAST   | 220    |
| 2024      | NORTH  | 120    |
| 2024      | SOUTH  | 180    |

Key ideas:
- `UNPIVOT` takes multiple columns and converts them into rows.
- `FOR region IN (...)` maps each column to a row label.

---

## 3. Alternative: Conditional Aggregation (Manual Pivot)

If you can’t use `PIVOT`, you can simulate it with `CASE`:

```sql
SELECT
    sale_year,
    SUM(CASE WHEN region = 'NORTH' THEN amount END) AS north_amt,
    SUM(CASE WHEN region = 'SOUTH' THEN amount END) AS south_amt,
    SUM(CASE WHEN region = 'EAST'  THEN amount END) AS east_amt
FROM sales
GROUP BY sale_year
ORDER BY sale_year;
```

**Output:** same as the `PIVOT` example.

---

**Main ideas:**
- Use `PIVOT` to convert rows to columns (row values become column names).
- Use `UNPIVOT` to convert columns to rows (columns become row values).
- You can also pivot manually with `CASE` + `GROUP BY` when needed.
