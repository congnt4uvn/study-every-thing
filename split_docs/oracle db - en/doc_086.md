
Here’s a concise summary and examples for **Oracle Object Types in PL/SQL & SQL**.

---

## Key Ideas

1. **Object Types**  
   - User-defined composite types that bundle **attributes** (data) and **methods** (procedures/functions).  
   - Defined at the **schema level** and usable in both **SQL** and **PL/SQL**.

2. **Structure**
   - `CREATE TYPE` defines the **specification** (attributes + method signatures).
   - `CREATE TYPE BODY` defines the **implementation** of methods.

3. **Usage**
   - Can be used as:
     - Column types in tables.
     - Variables in PL/SQL blocks.
     - Parameters and return types of procedures/functions.
   - Support **object tables**, **nested tables**, and **VARRAYs**.

4. **Benefits**
   - Encapsulation of data + behavior.
   - Reusability across SQL and PL/SQL.
   - More “object-oriented” modeling of business entities.

---

## Basic Example: Define and Use an Object Type

### 1) Create an Object Type

```sql
CREATE TYPE person_t AS OBJECT (
    person_id   NUMBER,
    first_name  VARCHAR2(50),
    last_name   VARCHAR2(50),
    MEMBER FUNCTION full_name RETURN VARCHAR2
);
/
```

### 2) Implement the Methods

```sql
CREATE TYPE BODY person_t AS
    MEMBER FUNCTION full_name RETURN VARCHAR2 IS
    BEGIN
        RETURN first_name || ' ' || last_name;
    END;
END;
/
```

---

## Using Object Types in SQL

### 3) Object Type as a Column in a Table

```sql
CREATE TABLE employees_obj (
    emp_no   NUMBER PRIMARY KEY,
    person   person_t
);
```

### 4) Insert Data

**Input:**

```sql
INSERT INTO employees_obj (emp_no, person)
VALUES (
    1001,
    person_t(1, 'John', 'Doe')
);

INSERT INTO employees_obj (emp_no, person)
VALUES (
    1002,
    person_t(2, 'Jane', 'Smith')
);
COMMIT;
```

### 5) Query Data and Call Methods

**Input:**

```sql
SELECT e.emp_no,
       e.person.person_id      AS person_id,
       e.person.full_name()    AS full_name
FROM   employees_obj e
ORDER BY e.emp_no;
```

**Output (example):**

| EMP_NO | PERSON_ID | FULL_NAME   |
|--------|-----------|-------------|
| 1001   | 1         | John Doe    |
| 1002   | 2         | Jane Smith  |

---

## Using Object Types in PL/SQL

### 6) Declare and Use Object Variables

**Input:**

```plsql
DECLARE
    l_person person_t;
BEGIN
    -- Construct an object instance
    l_person := person_t(10, 'Alice', 'Brown');

    DBMS_OUTPUT.PUT_LINE('ID: ' || l_person.person_id);
    DBMS_OUTPUT.PUT_LINE('Name: ' || l_person.full_name());
END;
/
```

**Output (DBMS_OUTPUT):**

```
ID: 10
Name: Alice Brown
```

## Object Tables

You can create a table directly of an object type:

```sql
CREATE TABLE person_tab OF person_t;

INSERT INTO person_tab
VALUES (person_t(1, 'Tom', 'Harris'));

SELECT p.person_id, p.full_name()
FROM   person_tab p;
```
