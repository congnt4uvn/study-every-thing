
Here’s a concise summary and some practical examples based on your content.

---

## Key Points

- **Common Oracle Database Tools**  
  These are popular GUI tools used to connect to and work with Oracle databases:
  1. **SQL Developer** (free, from Oracle)
  2. **TOAD** (commercial, rich features for developers/DBAs)
  3. **PL/SQL Developer** (focused on PL/SQL development)

- **Main Uses of These Tools**
  - Connect to Oracle databases
  - Run SQL queries
  - Develop and debug PL/SQL (procedures, functions, packages)
  - Browse tables, views, indexes, and other schema objects
  - Export/import data

---

## Typical Workflow Example (applies to all three tools)

### 1. Connect to the Database

You usually provide:
- Host: `dbserver.company.com`
- Port: `1521`
- Service name / SID: `ORCL`
- Username: `hr`
- Password: `hr_password`

In SQL Developer, for example:
- New Connection → enter above details → Test → Connect

---

### 2. Run a Simple Query

**Input (SQL):**
```sql
SELECT employee_id,
       first_name,
       last_name,
       salary
FROM   employees
WHERE  department_id = 10;
```

**Output (Result Grid):**

| EMPLOYEE_ID | FIRST_NAME | LAST_NAME | SALARY |
|-------------|------------|-----------|--------|
| 200         | Jennifer   | Whalen    | 4400   |

---

### 3. Create a Table

**Input (SQL):**
```sql
CREATE TABLE employees_test (
    employee_id   NUMBER(6)    PRIMARY KEY,
    first_name    VARCHAR2(20),
    last_name     VARCHAR2(25) NOT NULL,
    hire_date     DATE         NOT NULL,
    salary        NUMBER(8,2)
);
```

**Output:**
- Message pane: `Table EMPLOYEES_TEST created.`

---

### 4. Insert and Query Data

**Input (SQL):**
```sql
INSERT INTO employees_test (employee_id, first_name, last_name, hire_date, salary)
VALUES (1, 'John', 'Smith', DATE '2024-01-01', 5000);

COMMIT;

SELECT * FROM employees_test;
```

**Output (Result Grid):**

| EMPLOYEE_ID | FIRST_NAME | LAST_NAME | HIRE_DATE   | SALARY |
|-------------|------------|-----------|-------------|--------|
| 1           | John       | Smith     | 2024-01-01  | 5000   |

---

### 5. Simple PL/SQL Block

**Input (PL/SQL):**
```sql
SET SERVEROUTPUT ON;

DECLARE
    v_count NUMBER;
BEGIN
    SELECT COUNT(*)
    INTO   v_count
    FROM   employees_test;

    DBMS_OUTPUT.PUT_LINE('Number of rows: ' || v_count);
END;
/
```

**Output (DBMS Output panel):**
```
Number of rows: 1
```

---

These tools mainly differ in interface and extra features, but the SQL and PL/SQL you write in them is the same Oracle language.
