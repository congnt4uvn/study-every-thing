


Nội dung trên nói về tính năng mới của AWS Lambda: hỗ trợ triển khai hàm dưới dạng **container image** (tối đa 10GB) từ Amazon ECR.

Các ý chính:

1. **Lambda hỗ trợ container image**  
   - Có thể đóng gói mã, dependency phức tạp và lớn vào một container.  
   - Image này được lưu trên ECR và Lambda sẽ kéo về để chạy.

2. **Yêu cầu đối với container image**  
   - Không phải bất kỳ Docker image nào cũng chạy được trên Lambda.  
   - Image phải dựa trên **base image** có triển khai **Lambda Runtime API**.  
   - AWS cung cấp sẵn base image cho nhiều ngôn ngữ: Python, Node.js, Java, .NET, Go, Ruby.  
   - Bạn cũng có thể tự tạo base image miễn là triển khai đúng Lambda Runtime API (xem tài liệu).

3. **Quy trình xây dựng container cho Lambda (ví dụ Node.js)**  
   - Chọn base image: `amazon/aws-lambda-nodejs:12`.  
   - Copy mã nguồn và file (app.js, package.json, dữ liệu…) vào image.  
   - Cài đặt dependency (npm install).  
   - Chỉ định handler sẽ chạy khi Lambda được gọi, ví dụ: `CMD ["app.lambdaHandler"]`.  
   - Vì dùng đúng base image của Lambda, image này có thể chạy trên Lambda.

4. **Lợi ích**  
   - Dễ đóng gói dependency phức tạp, lớn.  
   - Có thể test cục bộ bằng **Lambda Runtime Interface Emulator**.  
   - Có thể dùng chung workflow container cho ECS và Lambda: build → push ECR → deploy.

5. **Best practices khi dùng container image cho Lambda**  
   - **Dùng base image AWS** (Amazon Linux 2) vì Lambda đã cache, giúp khởi chạy nhanh hơn.  
   - **Multi-stage builds**:  
     - Dùng image lớn để build, sau đó chỉ copy artifact cần thiết sang image cuối nhỏ gọn.  
   - **Tổ chức layer hợp lý**:  
     - Những phần ít thay đổi (cài package cơ bản) đặt ở các layer đầu.  
     - Những phần thay đổi thường xuyên (mã ứng dụng) đặt ở layer cuối để tận dụng cache.  
   - **Dùng một ECR repository cho các hàm có layer lớn giống nhau** để ECR tái sử dụng layer, tránh upload/lưu trữ trùng lặp.

6. **Trường hợp sử dụng nổi bật**  
   - Khi cần triển khai Lambda rất lớn (tới 10GB), dùng container image là giải pháp phù hợp hơn so với cách upload mã truyền thống.
