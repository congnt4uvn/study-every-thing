


Dưới đây là tóm tắt nội dung:

- Chủ đề: Cách sử dụng Xác thực đa yếu tố (MFA) với AWS CLI/SDK bằng cách tạo phiên tạm thời thông qua AWS STS.

- Ý chính cần nhớ:  
  Để lấy thông tin đăng nhập tạm thời khi dùng MFA, phải gọi API `STS GetSessionToken`.  
  Kết quả trả về: `AccessKeyId`, `SecretAccessKey`, `SessionToken` và `Expiration`.

- Các bước thực hành:

  1. **Gắn thiết bị MFA cho user IAM**  
     - Vào IAM → Users → chọn user (ví dụ: Stephane).  
     - Vào Security credentials → gán thiết bị MFA (chọn Virtual MFA device).  
     - Dùng ứng dụng như Authy quét mã QR.  
     - Nhập liên tiếp 2 mã MFA (ví dụ: 902495 và 865515).  
     - Sau khi thành công, lấy ARN của thiết bị MFA (Assign MFA device ARN).

  2. **Gọi lệnh STS GetSessionToken trên CLI**  
     - Dùng lệnh:  
       `aws sts get-session-token --serial-number <MFA_ARN> --token-code <MFA_code>`  
     - `--serial-number`: ARN của thiết bị MFA.  
     - `--token-code`: mã OTP hiện tại từ ứng dụng MFA.  
     - Kết quả trả về là bộ thông tin đăng nhập tạm thời (AccessKeyId, SecretAccessKey, SessionToken, Expiration ~ 1 giờ).

  3. **Cấu hình profile CLI dùng thông tin tạm thời**  
     - Chạy: `aws configure --profile mfa`  
       - Nhập Access Key ID tạm thời.  
       - Nhập Secret Access Key tạm thời.  
       - Region và output format có thể giữ mặc định.  
     - Mở file `~/.aws/credentials` (bằng editor bất kỳ, ví dụ VS Code).  
     - Trong profile `[mfa]`, thêm dòng:  
       `aws_session_token = <SessionToken>`  
     - Lưu file.

  4. **Sử dụng profile MFA**  
     - Chạy lệnh với profile:  
       `aws s3 ls --profile mfa`  
     - Lệnh này sẽ dùng thông tin đăng nhập tạm thời (có MFA) để gọi API S3.

- Kết luận:  
  - MFA được tích hợp vào CLI bằng cách dùng `aws sts get-session-token` với `--serial-number` (ARN thiết bị MFA) và `--token-code` (mã OTP).  
  - Thông tin trả về là bộ credentials tạm thời (AccessKey, SecretKey, SessionToken) được cấu hình vào một profile CLI.  
  - Điều quan trọng cần nhớ cho kỳ thi: **API dùng để tạo session tạm thời với MFA là `STS GetSessionToken`**.
