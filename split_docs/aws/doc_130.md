


Nội dung trên nói về tổng quan AWS SDK:

- SDK (Software Development Kit) là bộ công cụ giúp bạn tương tác với các dịch vụ AWS trực tiếp từ mã nguồn ứng dụng, thay vì phải dùng AWS CLI.  
- AWS cung cấp nhiều SDK chính thức cho các ngôn ngữ như: Java, .NET, Node.js, PHP, Python, Go, Ruby, C++… và danh sách này còn có thể mở rộng.  
- AWS CLI thực chất được viết bằng Python và sử dụng SDK Python là Boto3.  
- SDK được dùng khi bạn muốn gửi các yêu cầu API đến các dịch vụ AWS như DynamoDB, S3… từ trong code.  
- Trong kỳ thi (và trong thực tế), bạn cần biết khi nào nên dùng SDK.  
- Trong khóa học, phần Lambda sẽ minh họa cách dùng SDK trong mã nguồn, nên sau này sẽ có phần thực hành cụ thể.  
- Phần này chỉ là giới thiệu tổng quan, giúp có cái nhìn chung trước khi đi sâu hơn.
