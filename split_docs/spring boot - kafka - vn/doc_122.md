
Dưới đây là tóm tắt ngắn gọn về các operation chính trong Kafka Streams, kèm ví dụ Spring Boot (Java) và input/output minh họa.

---

## 1. `map` – Biến đổi từng record

- Dùng để chuyển đổi **key/value** của từng record.
- Không thay đổi số lượng record (1 vào → 1 ra).

**Ví dụ:**

```java
KStream<String, String> inputStream = builder.stream("input-topic");

KStream<String, String> mappedStream = inputStream.map(
    (key, value) -> KeyValue.pair(key, value.toUpperCase())
);

mappedStream.to("mapped-topic");
```

**Input (input-topic):**
- key: "user1", value: "hello"
- key: "user2", value: "kafka"

**Output (mapped-topic):**
- key: "user1", value: "HELLO"
- key: "user2", value: "KAFKA"

---

## 2. `filter` – Lọc record theo điều kiện

- Giữ lại những record thỏa điều kiện.
- Có thể làm giảm số lượng record.

**Ví dụ:**

```java
KStream<String, String> inputStream = builder.stream("input-topic");

KStream<String, String> filteredStream = inputStream.filter(
    (key, value) -> value.length() > 5
);

filteredStream.to("filtered-topic");
```

**Input (input-topic):**
- ("user1", "hello")      → length = 5
- ("user2", "kafka-streams") → length = 13

**Output (filtered-topic):**
- Chỉ có: ("user2", "kafka-streams")

---

## 3. `groupByKey` – Nhóm theo key

- Dùng cho `KStream` để chuyển thành `KGroupedStream`.
- Chuẩn bị cho các operation như `count`, `aggregate`, `reduce`.

**Ví dụ: đếm số message theo user**

```java
KStream<String, String> inputStream = builder.stream("input-topic");

KGroupedStream<String, String> grouped = inputStream.groupByKey();

KTable<String, Long> counts = grouped.count();

counts.toStream().to("user-counts", Produced.with(Serdes.String(), Serdes.Long()));
```

**Input (input-topic):**
- ("user1", "msg1")
- ("user1", "msg2")
- ("user2", "msg3")

**Output (user-counts):**
- ("user1", 1)
- ("user1", 2)
- ("user2", 1)

---

## 4. `aggregate` – Tự định nghĩa cách gom dữ liệu

- Cho phép tạo ra state phức tạp (List, Map, Object…).
- Chạy trên `KGroupedStream` hoặc `TimeWindowedKStream`.

**Ví dụ: gom tất cả message của mỗi user vào List**

```java
KStream<String, String> inputStream = builder.stream("input-topic");

KTable<String, List<String>> aggregated = inputStream
    .groupByKey()
    .aggregate(
        () -> new ArrayList<String>(), // giá trị khởi tạo
        (key, value, aggregate) -> {
            aggregate.add(value);
            return aggregate;
        },
        Materialized.with(Serdes.String(), new ListSerde())
    );
```

**Input:**
- ("user1", "a")
- ("user1", "b")
- ("user2", "c")

**State (KTable):**
- key "user1" → ["a", "b"]
- key "user2" → ["c"]

---

## 5. `join` – Kết hợp dữ liệu từ nhiều stream/table

Có nhiều loại join: `KStream-KStream`, `KStream-KTable`, `KTable-KTable`.

### Ví dụ: KStream-KTable join

- Stream: sự kiện mua hàng
- Table: thông tin user

```java
KStream<String, String> purchases = builder.stream("purchases");
KTable<String, String> users = builder.table("users");

KStream<String, String> enriched = purchases.join(
    users,
    (purchaseValue, userInfo) -> purchaseValue + " by " + userInfo
);

enriched.to("enriched-purchases");
```

**Input (users – KTable):**
- key: "user1", value: "Alice"
- key: "user2", value: "Bob"

**Input (purchases – KStream):**
- ("user1", "Order#100")
- ("user2", "Order#200")

**Output (enriched-purchases):**
- ("user1", "Order#100 by Alice")
- ("user2", "Order#200 by Bob")

---

## 6. Windowed operations – Xử lý theo cửa sổ thời gian

Dùng để tính toán theo khoảng thời gian (ví dụ: mỗi 1 phút, 5 phút…).

### Ví dụ: đếm số message theo user trong cửa sổ 1 phút

```java
KStream<String, String> inputStream = builder.stream("input-topic");

TimeWindowedKStream<String, String> windowed = inputStream
    .groupByKey()
    .windowedBy(TimeWindows.ofSizeWithNoGrace(Duration.ofMinutes(1)));

KTable<Windowed<String>, Long> windowedCounts = windowed.count();

windowedCounts
    .toStream()
    .map((windowedKey, count) -> {
        String newKey = windowedKey.key() + "@" +
                windowedKey.window().startTime() + "-" +
                windowedKey.window().endTime();
        return KeyValue.pair(newKey, count);
    })
    .to("windowed-counts", Produced.with(Serdes.String(), Serdes.Long()));
```

**Input (trong 1 phút):**
- 10 message của "user1"
- 3 message của "user2"

**Output (windowed-counts):**
- key: "user1@2026-05-20T10:00-2026-05-20T10:01", value: 10  
- key: "user2@2026-05-20T10:00-2026-05-20T10:01", value: 3  

---

## Tóm tắt nhanh

- `map`: biến đổi từng record (key/value).
- `filter`: loại bỏ record không thỏa điều kiện.
- `groupByKey`: nhóm theo key để chuẩn bị aggregate.
- `aggregate`: tự định nghĩa cách gom dữ liệu (stateful).
- `join`: kết hợp dữ liệu từ nhiều stream/table.
- Windowed operations: tính toán theo khoảng thời gian (time window).
