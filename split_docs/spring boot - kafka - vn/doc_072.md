
Dưới đây là tóm tắt ngắn gọn về **Spring Kafka basics: Auto-configuration trong Spring Boot (`KafkaAutoConfiguration`)** bằng tiếng Việt, kèm ví dụ code.

---

## 1. `KafkaAutoConfiguration` là gì?

- Là cơ chế **tự động cấu hình** Kafka trong Spring Boot.
- Khi bạn thêm dependency `spring-kafka` và cấu hình tối thiểu trong `application.properties` / `application.yml`, Spring Boot sẽ:
  - Tự tạo **ProducerFactory**, **ConsumerFactory**.
  - Tự tạo **KafkaTemplate** (gửi message).
  - Tự cấu hình **KafkaListenerContainerFactory** (nhận message).
- Giúp bạn không phải tự khai báo các bean Kafka cơ bản.

---

## 2. Cấu hình tối thiểu trong `application.yml`

```yaml
spring:
  kafka:
    bootstrap-servers: localhost:9092

    producer:
      key-serializer: org.apache.kafka.common.serialization.StringSerializer
      value-serializer: org.apache.kafka.common.serialization.StringSerializer

    consumer:
      group-id: my-group
      key-deserializer: org.apache.kafka.common.serialization.StringDeserializer
      value-deserializer: org.apache.kafka.common.serialization.StringDeserializer
```

Chỉ với cấu hình này, `KafkaAutoConfiguration` sẽ tự tạo các bean cần thiết.

---

## 3. Gửi message với `KafkaTemplate`

### Code ví dụ

```java
import org.springframework.kafka.core.KafkaTemplate;
import org.springframework.stereotype.Service;

@Service
public class KafkaProducerService {

    private final KafkaTemplate<String, String> kafkaTemplate;

    public KafkaProducerService(KafkaTemplate<String, String> kafkaTemplate) {
        this.kafkaTemplate = kafkaTemplate;
    }

    public void sendMessage(String topic, String key, String message) {
        kafkaTemplate.send(topic, key, message);
    }
}
```

### Input / Output ví dụ

- Input (gọi service):

```java
producerService.sendMessage("demo-topic", "user-1", "Xin chào Kafka!");
```

- Output (trên Kafka topic `demo-topic`):

```text
Key: "user-1"
Value: "Xin chào Kafka!"
```

---

## 4. Nhận message với `@KafkaListener`

```java
import org.springframework.kafka.annotation.KafkaListener;
import org.springframework.stereotype.Service;

@Service
public class KafkaConsumerService {

    @KafkaListener(topics = "demo-topic", groupId = "my-group")
    public void listen(String message) {
        System.out.println("Nhận message: " + message);
    }
}
```

### Input / Output ví dụ

- Input: message được gửi lên topic `demo-topic` (như ví dụ ở trên).
- Output (log):

```text
Nhận message: Xin chào Kafka!
```

## 5. Tùy biến auto-configuration (override)

Nếu bạn muốn tự cấu hình thay vì dùng mặc định:

```java
import org.apache.kafka.clients.producer.ProducerConfig;
import org.apache.kafka.common.serialization.StringSerializer;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.kafka.core.*;

import java.util.HashMap;
import java.util.Map;

@Configuration
public class CustomKafkaConfig {

    @Bean
    public ProducerFactory<String, String> producerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, String> kafkaTemplate() {
        return new KafkaTemplate<>(producerFactory());
    }
}
```

Khi bạn định nghĩa các bean này, Spring Boot sẽ dùng chúng thay cho cấu hình auto (`KafkaAutoConfiguration`).

---

## 6. Ý chính cần nhớ

- `KafkaAutoConfiguration` giúp:
  - Giảm cấu hình thủ công.
  - Nhanh chóng tích hợp Kafka với Spring Boot.
- Chỉ cần:
  - Thêm dependency `spring-kafka`.
  - Cấu hình `spring.kafka.*`.
- Sau đó:
  - Dùng `KafkaTemplate` để gửi.
  - Dùng `@KafkaListener` để nhận.
