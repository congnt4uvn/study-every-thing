
Dưới đây là tóm tắt ngắn gọn về việc dùng **Kafka Streams với Spring Boot (Spring Integration)**, kèm ví dụ code và input/output minh họa.

---

## 1. Ý tưởng chính

- **Kafka Streams**: thư viện Java để xử lý stream dữ liệu trực tiếp từ Kafka (filter, map, group, aggregate…).
- **Spring Boot + Spring Kafka**: giúp cấu hình và chạy Kafka Streams dễ dàng hơn (tự động cấu hình, quản lý lifecycle, dependency injection).
- Bạn định nghĩa **topology** (luồng xử lý) bằng code Java, Spring Boot sẽ:
  - Tạo `StreamsBuilder`
  - Build `Topology`
  - Tạo `KafkaStreams` và start app

## 2. Cấu hình cơ bản trong Spring Boot

### 2.1. Maven dependencies

```xml
<dependencies>
    <!-- Spring Boot Kafka -->
    <dependency>
        <groupId>org.springframework.kafka</groupId>
        <artifactId>spring-kafka</artifactId>
    </dependency>

    <!-- Kafka Streams -->
    <dependency>
        <groupId>org.apache.kafka</groupId>
        <artifactId>kafka-streams</artifactId>
    </dependency>
</dependencies>
```

### 2.2. application.yml

```yaml
spring:
  application:
    name: kafka-streams-demo

  kafka:
    bootstrap-servers: localhost:9092
    streams:
      application-id: kafka-streams-demo-app
      properties:
        default.key.serde: org.apache.kafka.common.serialization.Serdes$StringSerde
        default.value.serde: org.apache.kafka.common.serialization.Serdes$StringSerde
```

## 3. Định nghĩa topology với `StreamsBuilder`

Ví dụ: đọc từ topic `input-topic`, chuyển chữ thường thành chữ hoa, ghi ra `output-topic`.

```java
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.KStream;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class KafkaStreamsConfig {

    @Bean
    public KStream<String, String> kStream(StreamsBuilder builder) {
        // 1. Đọc stream từ topic input-topic
        KStream<String, String> inputStream = builder.stream("input-topic");

        // 2. Xử lý: chuyển value thành chữ hoa
        KStream<String, String> uppercasedStream = inputStream
                .mapValues(value -> value == null ? null : value.toUpperCase());

        // 3. Ghi kết quả ra topic output-topic
        uppercasedStream.to("output-topic");

        // Trả về stream để Spring quản lý bean
        return uppercasedStream;
    }
}
```

## 5. Ví dụ xử lý phức tạp hơn: đếm số lần xuất hiện từ

Đọc từ `words-input`, tách câu thành từ, đếm số lần xuất hiện, ghi ra `word-count-output`.

```java
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.*;
import org.apache.kafka.common.serialization.Serdes;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

import java.util.Arrays;

@Configuration
public class WordCountStreamConfig {

    @Bean
    public KStream<String, String> wordCountStream(StreamsBuilder builder) {
        KStream<String, String> textLines = builder.stream("words-input");

        KTable<String, Long> wordCounts = textLines
                // tách câu thành các từ
                .flatMapValues(value -> Arrays.asList(value.toLowerCase().split("\\W+")))
                // dùng từ làm key
                .selectKey((key, word) -> word)
                // group theo key (từ)
                .groupByKey(Grouped.with(Serdes.String(), Serdes.String()))
                // đếm số lần xuất hiện
                .count(Materialized.as("word-counts-store"));

        // ghi kết quả ra topic
        wordCounts
                .toStream()
                .to("word-count-output", Produced.with(Serdes.String(), Serdes.Long()));

        return textLines;
    }
}
```

### Input

Topic `words-input`:

```text
hello kafka
hello spring
kafka streams with spring
```

### Output

Topic `word-count-output` (key = từ, value = số lần):

```text
hello -> 1
kafka -> 1
hello -> 2
spring -> 1
kafka -> 2
streams -> 1
with -> 1
spring -> 2
```

## 6. Điểm chính cần nhớ

- Spring Boot + Spring Kafka giúp:
  - Cấu hình Kafka Streams qua `application.yml`.
  - Tự động tạo `KafkaStreams` từ các bean `KStream`, `KTable`.
- Bạn chỉ cần:
  1. Thêm dependency.
  2. Cấu hình `spring.kafka.streams`.
  3. Định nghĩa topology bằng `StreamsBuilder` trong các `@Configuration` class.
- Có thể dễ dàng:
  - Filter, map, group, aggregate.
  - Đọc/ghi từ nhiều topic.
  - Kết hợp với các service Spring khác (DI, logging, metrics…).
