
Dưới đây là tóm tắt ngắn gọn về cách dùng Spring Boot với Schema Registry, sử dụng `KafkaAvroSerializer` / `KafkaAvroDeserializer` (Confluent):

---

## 1. Mục tiêu chính

- Gửi/nhận message Kafka dưới dạng **Avro**.
- Tự động **đăng ký và lấy schema** từ **Confluent Schema Registry**.
- Dùng `KafkaAvroSerializer` để serialize object Avro khi gửi.
- Dùng `KafkaAvroDeserializer` để deserialize message Avro khi nhận.

---

## 2. Cấu hình Maven/Gradle

### Maven

```xml
<dependencies>
    <!-- Spring Kafka -->
    <dependency>
        <groupId>org.springframework.kafka</groupId>
        <artifactId>spring-kafka</artifactId>
    </dependency>

    <!-- Confluent Avro Serializer/Deserializer -->
    <dependency>
        <groupId>io.confluent</groupId>
        <artifactId>kafka-avro-serializer</artifactId>
        <version>7.6.0</version> <!-- chỉnh theo version bạn dùng -->
    </dependency>

    <!-- Avro (nếu cần generate class từ .avsc) -->
    <dependency>
        <groupId>org.apache.avro</groupId>
        <artifactId>avro</artifactId>
        <version>1.11.3</version>
    </dependency>
</dependencies>
```

---

## 3. Cấu hình application.yml

```yaml
spring:
  kafka:
    bootstrap-servers: localhost:9092
    properties:
      schema.registry.url: http://localhost:8081
      specific.avro.reader: true   # nếu dùng SpecificRecord
    producer:
      key-serializer: org.apache.kafka.common.serialization.StringSerializer
      value-serializer: io.confluent.kafka.serializers.KafkaAvroSerializer
    consumer:
      key-deserializer: org.apache.kafka.common.serialization.StringDeserializer
      value-deserializer: io.confluent.kafka.serializers.KafkaAvroDeserializer
      group-id: my-avro-group
      properties:
        specific.avro.reader: true
```

---

## 4. Định nghĩa schema Avro và generate class

Ví dụ file `user.avsc`:

```json
{
  "type": "record",
  "name": "User",
  "namespace": "com.example.avro",
  "fields": [
    { "name": "id", "type": "string" },
    { "name": "name", "type": "string" },
    { "name": "age", "type": ["null", "int"], "default": null }
  ]
}
```

Sau khi generate, bạn sẽ có class Java: `com.example.avro.User` (implements `SpecificRecord`).

## 5. Producer: gửi message Avro

### Cấu hình KafkaTemplate

```java
@Configuration
public class KafkaProducerConfig {

    @Bean
    public ProducerFactory<String, User> producerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG,
                  org.apache.kafka.common.serialization.StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG,
                  io.confluent.kafka.serializers.KafkaAvroSerializer.class);
        props.put("schema.registry.url", "http://localhost:8081");
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, User> kafkaTemplate() {
        return new KafkaTemplate<>(producerFactory());
    }
}
```

### Gửi message

```java
@Service
public class UserProducer {

    private final KafkaTemplate<String, User> kafkaTemplate;

    public UserProducer(KafkaTemplate<String, User> kafkaTemplate) {
        this.kafkaTemplate = kafkaTemplate;
    }

    public void sendUser(User user) {
        kafkaTemplate.send("user-topic", user.getId().toString(), user);
    }
}
```

**Input (object Java):**

```java
User user = User.newBuilder()
        .setId("u-1")
        .setName("Nguyen Van A")
        .setAge(30)
        .build();

userProducer.sendUser(user);
```

**Output (trên Kafka topic `user-topic`):**

- Message value: binary Avro (đã serialize).
- Schema được đăng ký trong Schema Registry với subject `user-topic-value` (hoặc theo config).

## 6. Consumer: nhận message Avro

### Cấu hình ConsumerFactory & ListenerContainer

```java
@Configuration
@EnableKafka
public class KafkaConsumerConfig {

    @Bean
    public ConsumerFactory<String, User> consumerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ConsumerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ConsumerConfig.KEY_DESERIALIZER_CLASS_CONFIG,
                  org.apache.kafka.common.serialization.StringDeserializer.class);
        props.put(ConsumerConfig.VALUE_DESERIALIZER_CLASS_CONFIG,
                  io.confluent.kafka.serializers.KafkaAvroDeserializer.class);
        props.put("schema.registry.url", "http://localhost:8081");
        props.put("specific.avro.reader", true);
        props.put(ConsumerConfig.GROUP_ID_CONFIG, "my-avro-group");
        return new DefaultKafkaConsumerFactory<>(props);
    }

    @Bean
    public ConcurrentKafkaListenerContainerFactory<String, User> kafkaListenerContainerFactory() {
        ConcurrentKafkaListenerContainerFactory<String, User> factory =
                new ConcurrentKafkaListenerContainerFactory<>();
        factory.setConsumerFactory(consumerFactory());
        return factory;
    }
}
```

### Lắng nghe message

```java
@Service
public class UserConsumer {

    @KafkaListener(topics = "user-topic", groupId = "my-avro-group")
    public void listen(User user) {
        System.out.println("Received user: " + user);
    }
}
```

**Input (trên topic):** message Avro đã serialize (từ Producer).

**Output (log Consumer):**

```text
Received user: {"id": "u-1", "name": "Nguyen Van A", "age": 30}
```

---

## 7. Ý chính cần nhớ

1. **Schema Registry** lưu trữ schema Avro, cho phép versioning và tương thích schema.
2. `KafkaAvroSerializer`:
   - Serialize object Avro.
   - Tự động đăng ký schema (nếu chưa có) lên Schema Registry.
3. `KafkaAvroDeserializer`:
   - Lấy schema từ Schema Registry.
   - Deserialize bytes thành `SpecificRecord` (hoặc `GenericRecord`).
4. Cần cấu hình:
   - `schema.registry.url`
   - `value-serializer` / `value-deserializer` là Avro của Confluent.
   - `specific.avro.reader=true` nếu dùng class Avro generate sẵn.
