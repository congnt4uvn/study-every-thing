
Dưới đây là tóm tắt ngắn gọn về **Spring Kafka Producer Serializers** (String, JSON, Avro, custom), kèm ví dụ code và input/output minh họa.

---

## 1. Khái niệm chung

- **Serializer**: chuyển đối tượng Java thành `byte[]` để gửi lên Kafka.
- Spring Kafka dùng các serializer của Kafka (hoặc custom) thông qua cấu hình `ProducerFactory` / `KafkaTemplate`.

---

## 2. String Serializer

### Cấu hình

```java
import org.apache.kafka.clients.producer.ProducerConfig;
import org.apache.kafka.common.serialization.StringSerializer;
import org.springframework.kafka.core.DefaultKafkaProducerFactory;
import org.springframework.kafka.core.KafkaTemplate;
import org.springframework.kafka.core.ProducerFactory;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

import java.util.HashMap;
import java.util.Map;

@Configuration
public class KafkaStringProducerConfig {

    @Bean
    public ProducerFactory<String, String> stringProducerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, String> stringKafkaTemplate() {
        return new KafkaTemplate<>(stringProducerFactory());
    }
}
```

### Gửi message

```java
@Autowired
private KafkaTemplate<String, String> stringKafkaTemplate;

public void sendStringMessage() {
    stringKafkaTemplate.send("string-topic", "key1", "Xin chào Kafka");
}
```

**Input (Java)**: `"Xin chào Kafka"`  
**Output (Kafka message)**: key = `"key1"`, value = `"Xin chào Kafka"` (dạng chuỗi, được serialize thành bytes).

## 3. JSON Serializer (Spring Kafka)

Dùng `JsonSerializer` của Spring Kafka để serialize object thành JSON.

### DTO

```java
public class User {
    private String id;
    private String name;
    // getters/setters, constructor
}
```

### Cấu hình

```java
import org.springframework.kafka.support.serializer.JsonSerializer;

@Configuration
public class KafkaJsonProducerConfig {

    @Bean
    public ProducerFactory<String, User> jsonProducerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG, JsonSerializer.class);
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, User> jsonKafkaTemplate() {
        return new KafkaTemplate<>(jsonProducerFactory());
    }
}
```

### Gửi message

```java
@Autowired
private KafkaTemplate<String, User> jsonKafkaTemplate;

public void sendUser() {
    User user = new User("1", "Nguyen Van A");
    jsonKafkaTemplate.send("user-topic", "user-key", user);
}
```

**Input (Java)**: `User{id="1", name="Nguyen Van A"}`  
**Output (Kafka message)**: value là JSON, ví dụ:
```json
{"id":"1","name":"Nguyen Van A"}
```

## 4. Avro Serializer

Dùng Avro để định nghĩa schema, thường kết hợp Schema Registry (Confluent).

### 4.1. Định nghĩa schema Avro (user.avsc)

```json
{
  "type": "record",
  "name": "UserAvro",
  "namespace": "com.example.avro",
  "fields": [
    {"name": "id", "type": "string"},
    {"name": "name", "type": "string"}
  ]
}
```

Sau đó dùng plugin Maven/Gradle sinh class Java `UserAvro`.

### 4.2. Cấu hình Producer với Avro

```xml
<!-- Maven dependencies (ví dụ Confluent) -->
<dependency>
    <groupId>io.confluent</groupId>
    <artifactId>kafka-avro-serializer</artifactId>
    <version>7.6.0</version>
</dependency>
```

```java
import io.confluent.kafka.serializers.KafkaAvroSerializer;

@Configuration
public class KafkaAvroProducerConfig {

    @Bean
    public ProducerFactory<String, UserAvro> avroProducerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG, KafkaAvroSerializer.class);
        props.put("schema.registry.url", "http://localhost:8081");
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, UserAvro> avroKafkaTemplate() {
        return new KafkaTemplate<>(avroProducerFactory());
    }
}
```

### Gửi message Avro

```java
@Autowired
private KafkaTemplate<String, UserAvro> avroKafkaTemplate;

public void sendAvroUser() {
    UserAvro user = UserAvro.newBuilder()
            .setId("1")
            .setName("Nguyen Van B")
            .build();
    avroKafkaTemplate.send("avro-user-topic", "user-key", user);
}
```

**Input (Java)**: `UserAvro{id="1", name="Nguyen Van B"}`  
**Output (Kafka message)**: dữ liệu Avro (binary) + schema id (tùy theo serializer), không phải JSON.

## 5. Custom Serializer

Khi cần format riêng (ví dụ: nén, mã hóa, hoặc format nhị phân đặc biệt).

### 5.1. Tạo Custom Serializer

```java
import org.apache.kafka.common.serialization.Serializer;
import com.fasterxml.jackson.databind.ObjectMapper;

public class CustomUserSerializer implements Serializer<User> {

    private final ObjectMapper objectMapper = new ObjectMapper();

    @Override
    public byte[] serialize(String topic, User data) {
        if (data == null) return null;
        try {
            // Ví dụ: JSON + thêm prefix
            byte[] jsonBytes = objectMapper.writeValueAsBytes(data);
            byte prefix = 0x01; // version
            byte[] result = new byte[jsonBytes.length + 1];
            result[0] = prefix;
            System.arraycopy(jsonBytes, 0, result, 1, jsonBytes.length);
            return result;
        } catch (Exception e) {
            throw new RuntimeException("Serialize error", e);
        }
    }
}
```

### 5.2. Cấu hình Producer với Custom Serializer

```java
@Configuration
public class KafkaCustomProducerConfig {

    @Bean
    public ProducerFactory<String, User> customProducerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ProducerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ProducerConfig.KEY_SERIALIZER_CLASS_CONFIG, StringSerializer.class);
        props.put(ProducerConfig.VALUE_SERIALIZER_CLASS_CONFIG, CustomUserSerializer.class);
        return new DefaultKafkaProducerFactory<>(props);
    }

    @Bean
    public KafkaTemplate<String, User> customKafkaTemplate() {
        return new KafkaTemplate<>(customProducerFactory());
    }
}
```

### Gửi message

```java
@Autowired
private KafkaTemplate<String, User> customKafkaTemplate;

public void sendCustomUser() {
    User user = new User("2", "Tran Thi C");
    customKafkaTemplate.send("custom-user-topic", "user-key", user);
}
```

**Input (Java)**: `User{id="2", name="Tran Thi C"}`  
**Output (Kafka message)**: bytes với byte đầu là prefix `0x01`, phần còn lại là JSON của User.

## 6. So sánh nhanh

- **StringSerializer**  
  - Dùng cho dữ liệu text đơn giản.  
  - Dễ debug, phù hợp log, test nhanh.

- **JsonSerializer**  
  - Dễ dùng với object Java.  
  - Không cần schema registry, nhưng schema không chặt chẽ.

- **AvroSerializer**  
  - Có schema rõ ràng, tương thích ngược/tiến tốt.  
  - Phù hợp hệ thống lớn, cần quản lý schema.

- **Custom Serializer**  
  - Linh hoạt tối đa (nén, mã hóa, format riêng).  
  - Cần tự quản lý logic serialize/deserialize.
