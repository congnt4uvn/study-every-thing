
Dưới đây là tóm tắt ngắn gọn về `@KafkaListener` trong Spring Kafka, kèm ví dụ code và input/output minh họa.

---

## 1. `@KafkaListener` là gì?

- Là annotation của Spring Kafka dùng để đánh dấu một method là **Kafka consumer**.
- Tự động:
  - Lắng nghe (subscribe) vào topic Kafka.
  - Nhận message và gọi method tương ứng.
  - Xử lý việc deserialization, groupId, concurrency… thông qua cấu hình.

---

## 2. Ví dụ cơ bản với `@KafkaListener`

### Cấu hình Maven (pom.xml)

```xml
<dependencies>
    <dependency>
        <groupId>org.springframework.kafka</groupId>
        <artifactId>spring-kafka</artifactId>
    </dependency>
</dependencies>
```

### Cấu hình Kafka Consumer (Java)

```java
import org.apache.kafka.clients.consumer.ConsumerConfig;
import org.apache.kafka.common.serialization.StringDeserializer;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.kafka.config.ConcurrentKafkaListenerContainerFactory;
import org.springframework.kafka.core.ConsumerFactory;
import org.springframework.kafka.core.DefaultKafkaConsumerFactory;

import java.util.HashMap;
import java.util.Map;

@Configuration
public class KafkaConsumerConfig {

    @Bean
    public ConsumerFactory<String, String> consumerFactory() {
        Map<String, Object> props = new HashMap<>();
        props.put(ConsumerConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
        props.put(ConsumerConfig.GROUP_ID_CONFIG, "my-group");
        props.put(ConsumerConfig.KEY_DESERIALIZER_CLASS_CONFIG, StringDeserializer.class);
        props.put(ConsumerConfig.VALUE_DESERIALIZER_CLASS_CONFIG, StringDeserializer.class);
        return new DefaultKafkaConsumerFactory<>(props);
    }

    @Bean
    public ConcurrentKafkaListenerContainerFactory<String, String>
    kafkaListenerContainerFactory() {
        ConcurrentKafkaListenerContainerFactory<String, String> factory =
                new ConcurrentKafkaListenerContainerFactory<>();
        factory.setConsumerFactory(consumerFactory());
        return factory;
    }
}
```

### Consumer sử dụng `@KafkaListener`

```java
import org.springframework.kafka.annotation.KafkaListener;
import org.springframework.stereotype.Service;

@Service
public class SimpleKafkaConsumer {

    @KafkaListener(topics = "my-topic", groupId = "my-group")
    public void listen(String message) {
        System.out.println("Received message: " + message);
    }
}
```

#### Input/Output minh họa

- **Input** (producer gửi):

```bash
# Ví dụ dùng kafka-console-producer
echo "Hello Kafka" | kafka-console-producer \
  --broker-list localhost:9092 \
  --topic my-topic
```

- **Output** (log của ứng dụng Spring Boot):

```text
Received message: Hello Kafka
```

## 4. Nhận thêm metadata (topic, partition, offset, key)

```java
import org.apache.kafka.clients.consumer.ConsumerRecord;
import org.springframework.kafka.support.Acknowledgment;
import org.springframework.kafka.annotation.KafkaListener;

@Service
public class AdvancedKafkaConsumer {

    @KafkaListener(topics = "my-topic", groupId = "my-group")
    public void listenWithMetadata(ConsumerRecord<String, String> record) {
        System.out.printf(
            "Topic: %s, Partition: %d, Offset: %d, Key: %s, Value: %s%n",
            record.topic(),
            record.partition(),
            record.offset(),
            record.key(),
            record.value()
        );
    }
}
```

#### Input/Output minh họa

- **Input**: gửi message với key:

```bash
kafka-console-producer \
  --broker-list localhost:9092 \
  --topic my-topic \
  --property "parse.key=true" \
  --property "key.separator=:"
# Gõ:
user1:Hello with key
```

- **Output**:

```text
Topic: my-topic, Partition: 0, Offset: 15, Key: user1, Value: Hello with key
```

## 5. Concurrency (nhiều consumer thread)

```java
@Bean
public ConcurrentKafkaListenerContainerFactory<String, String> kafkaListenerContainerFactory() {
    ConcurrentKafkaListenerContainerFactory<String, String> factory =
            new ConcurrentKafkaListenerContainerFactory<>();
    factory.setConsumerFactory(consumerFactory());
    factory.setConcurrency(3); // 3 thread consumer
    return factory;
}
```

```java
@KafkaListener(topics = "my-topic", groupId = "my-group")
public void listenConcurrent(String message) {
    System.out.println(Thread.currentThread().getName() + " received: " + message);
}
```

## 6. Manual Acknowledgment (commit offset thủ công)

```java
import org.springframework.kafka.support.Acknowledgment;

@KafkaListener(topics = "my-topic", groupId = "my-group")
public void listenManualAck(String message, Acknowledgment ack) {
    try {
        // Xử lý message
        System.out.println("Processing: " + message);
        // Commit offset sau khi xử lý thành công
        ack.acknowledge();
    } catch (Exception e) {
        // Xử lý lỗi, có thể không commit để message được đọc lại
    }
}
```

## 7. Tóm tắt ý chính

- `@KafkaListener` dùng để:
  - Định nghĩa method làm Kafka consumer.
  - Chỉ định topic, groupId, containerFactory, concurrency…
- Có thể:
  - Lắng nghe 1 hoặc nhiều topic.
  - Nhận message dạng đơn giản (`String`, DTO) hoặc `ConsumerRecord`.
  - Lấy metadata (topic, partition, offset, key).
  - Dùng manual ack để kiểm soát commit offset.
- Tích hợp chặt với Spring Boot, cấu hình chủ yếu qua `application.yml` hoặc bean config.
