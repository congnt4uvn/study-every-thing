# Java Roadmap (Zero → Hero)

This roadmap is a practical, hands-on path to learn **Java** from the basics to building and operating a small production-style backend service.

## How to use this roadmap

- Do lessons in order.
- For each lesson: read the concepts, run the code, then complete the checklist.
- Keep a small notes file with: what you tried, what broke, and what you fixed.

## Prerequisites

- No programming experience required (but it helps).
- A computer that can install a JDK and run a terminal.

## Environment setup (recommended)

- JDK 21 (or 17 if you need LTS compatibility)
- An IDE: IntelliJ IDEA Community or VS Code
- Git

Optional but helpful later:

- Docker Desktop
- PostgreSQL (local) or a managed DB

## Suggested weekly pacing (adjust as needed)

- Week 1: Lessons 01–03
- Week 2: Lessons 04–06
- Week 3: Lessons 07–10
- Week 4: Lessons 11–13 (capstone)

## Success criteria (what “hero” looks like)

- You can read and write idiomatic Java code (classes, generics, collections, streams).
- You can build a REST API with validation, tests, and database persistence.
- You understand the JVM at a practical level (memory, GC basics, how to profile).
- You can ship a small service with good defaults: logging, configuration, error handling, and a repeatable build.
