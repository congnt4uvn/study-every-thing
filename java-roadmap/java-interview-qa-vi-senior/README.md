## Gợi ý ôn tập

- Khi trả lời, luôn nêu **trade-off** (throughput vs latency, đơn giản vs tối ưu).
- Với vấn đề hiệu năng, ưu tiên: **đo đạc → giả thuyết → thử nghiệm → xác nhận**.
- Với concurrency, luôn nói rõ “**happens-before**” và “**safe publication**”.
